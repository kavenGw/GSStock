# 公司新闻表格内容格式化设计

## 问题

公司新闻推送中，资金流向日报和机构调研名单等包含表格数据的内容，被作为纯文本一行推送到 Slack，所有列名和数值挤在一起，完全不可读。

## 方案

在 `_notify_company_slack` 推送前，对 content 做正则检测和格式化。针对已知的两种表格类型做专用模板，不匹配的内容保持原样。

## 已知表格类型

### 1. 资金流向日报

**原始格式**：
```
电子行业3月19日资金流向日报：主力资金流量（万元）300476 胜宏科技 -4.27 3.81 -140622.36 002463 沪电股份 -5.13 5.63 -116310.93 ...
```

**数据结构**：标题 + 重复的 `[6位代码] [名称] [涨跌%] [换手%] [净流入万元]` 序列

**检测关键词**：`资金流向`

**格式化输出**（Slack mrkdwn 代码块，等宽字体）：
```
电子行业3月19日资金流向日报
主力资金流量（万元）
代码     名称       涨跌%   换手%   主力净流入
300476   胜宏科技   -4.27   3.81   -140,622
002463   沪电股份   -5.13   5.63   -116,311
601138   工业富联   -3.07   0.53    -99,068
688525   佰维存储   -5.78   9.45    -84,099
```

净流入金额为万元级，截断小数取整并加千分位逗号，提升可读性。

### 2. 海外机构调研股名单

**原始格式**：
```
近日海外机构调研股名单：近10日海外机构调研股 证券代码 证券简称 海外机构家数 最新收盘价（元） 其间涨跌幅（%） 行业 301265 华新环保 1 21.61 30.97 环保 002382 蓝帆医疗 9 7.89 21.38 医药生物 ...
```

**数据结构**：标题 + 表头文本 + 重复的 `[6位代码] [名称] [机构家数] [收盘价] [涨跌%] [行业]` 序列

**检测关键词**：`调研`

**格式化输出**：
```
近日海外机构调研股名单
代码     名称       机构数   收盘价   涨跌%    行业
301265   华新环保      1    21.61   30.97   环保
002382   蓝帆医疗      9     7.89   21.38   医药生物
688248   南网科技      4    63.13   21.22   电力设备
```

## 实现设计

### 改动文件

仅 `app/services/company_news_service.py`

### 新增模块级函数

```python
def _format_table_content(content: str) -> str:
    """检测并格式化表格类新闻内容，不匹配则原样返回"""
```

内部调用两个私有解析函数：
- `_format_capital_flow(content)` — 资金流向
- `_format_institution_research(content)` — 机构调研

均为模块级函数（非类方法），以 `_` 前缀标记为模块内部使用。

### 正则模式

股票名称可能包含字母和数字（如 ST中安、N新股、360安全），行业名同理（如 IT服务）。字符集使用 `\S+` 匹配非空白连续字符。

**资金流向**：
```python
# 匹配重复模式：6位数字 + 名称 + 涨跌% + 换手% + 净流入
r'(\d{6})\s+(\S+)\s+(-?\d+\.?\d*)\s+(-?\d+\.?\d*)\s+(-?\d+\.?\d*)'
```

**机构调研**：
```python
# 匹配重复模式：6位数字 + 名称 + 机构家数 + 收盘价 + 涨跌% + 行业名
r'(\d{6})\s+(\S+)\s+(\d+)\s+(\d+\.?\d*)\s+(-?\d+\.?\d*)\s+(\S+)'
```

### 标题提取

- 资金流向：取"主力资金"之前的文本作为标题，"主力资金流量（万元）"作为副标题
- 机构调研：取第一个6位数字之前的文本，去掉表头字段名（证券代码、证券简称等），保留描述性标题

### 对齐策略

代码块内按固定列宽对齐，数字列右对齐。用 Python `str.ljust()` / `str.rjust()` 实现。

### 调用位置

在现有 `_notify_company_slack` 的循环中，去重之后、发送之前插入格式化：

```python
for company, content in items:
    formatted = _format_table_content(content)
    NotificationService.send_slack(f"🏢 [{company}] {formatted}")
```

### Slack 格式

表格部分用三个反引号包裹为代码块，确保等宽字体。标题放在代码块外面：

```
🏢 [兆易创新] 电子行业3月19日资金流向日报
```代码     名称       涨跌%   换手%   主力净流入
300476   胜宏科技   -4.27   3.81   -140,622
...```
```

## 约束

- 零 LLM 消耗
- 不匹配已知类型的内容保持原样，不做任何处理
- 全部股票显示，不截断
- 仅改动推送格式，不影响数据库存储的原始内容

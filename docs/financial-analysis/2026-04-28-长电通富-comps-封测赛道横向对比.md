**核心判断**：

1. **长电 vs 通富是封测赛道"两强直接对标"**——长电是规模最大（营收 389 亿）、客户分散（高通/海思/博通/Marvell）、估值合理（PE 5y 56% / PB 5y 50%）的**全球第三大封测**；通富是营收增速最快（5y CAGR 21%，仅次于甬矽）、AMD 单一深度绑定、2025 净利大幅修复（EPS 同比 +78%）的**AMD 代理标的**。
2. **基本面排序**：通富（成长性 + AMD 兑现）≥ 长电（规模 + ROE 持续性）> 晶方（小而美高毛利）> 华天（基本面退化）> 甬矽（弹性大但持续盈利能力差）
3. **估值合理性排序**：晶方（PE 5y 28% 历史低位）> 长电（PE 5y 56% / PB 5y 50% 中位）> 通富（PB 5y 91% 偏贵）> 华天（PE 5y 79% 偏贵）> 甬矽（PE 5y 89% / PB 5y 95% 双高位）
4. **赛道整体警示**：5 家 PE 5y 分位均值 60%、PB 5y 分位均值 65%——**封测板块当前估值并未泡沫，但也无群体性低估**，分化定价中。**长电 = 估值锚 / 通富 = 成长锚 / 晶方 = 周期锚**是当前最有价值的三角组合。

**长电 vs 通富的核心分歧**：
- 若押注 **AI 服务器 CPU/GPU 出货延续（AMD MI350/MI400 + Zen5）**——选通富，但要承受 PB 5y 91% 偏贵 + 单客户依赖风险
- 若押注 **封测周期均衡 + Chiplet/2.5D 国产化长逻辑**——选长电，估值更合理但成长缺爆点
- **更优解可能是同时持有，按资金量分配 6:4 或 5:5**——两者业务结构对冲（长电高通/海思 vs 通富 AMD），共同享受 AI 算力链景气

**业务重叠度（与长电视角）**：
- 长电 vs **通富**：**直接竞品**（先进封装赛道双龙头，但客户结构错开 — 长电分散 / 通富 AMD 独家）
- 长电 vs **华天**：**直接竞品**（封测三巨头之一，但华天先进封装代差明显）
- 长电 vs **甬矽**：**部分重叠**（甬矽 SiP / QFN 中低端为主，与长电中高端错开）
- 长电 vs **晶方**：**互补关系**（晶方专攻 CIS，长电几乎不做 CIS）

### 为什么不纳入

- **气派科技（688216）/ 利扬芯片（688135）/ 颀中科技（688352）**：体量偏小（市值 30-80 亿），与百亿~千亿级 OSAT 不可比
- **太极实业（600667）**：海太半导体合资（DRAM 后道），但财报口径包含工程业务，封测拆分困难
- **国际同业（日月光 ASE / 安靠 Amkor / 京元电子 / 力成科技）**：A 股横向对比视角不纳入；如做国际对标可单独补一节
- **沪硅产业 / 立昂微（硅片）/ 长川科技（封测设备）**：上下游配套，非封测主业

### 2.3 成长性排序与质量评估

**单看营收 CAGR 排序**：甬矽(42.6%) > **通富(21.0%)** > 华天(15.5%) > **长电(8.0%)** > 晶方(5.9%)

**质量加权后的排序**（CAGR + 周期穿越 + 趋势方向 + 绝对体量）：

1. **通富微电** — 5y 营收 + 净利双 CAGR 同业最高 + 2025 净利同比 +78% AMD 兑现 + 绝对营收 279 亿仅次长电

**对长电 + 通富的相对位置**：
- **通富营收成长第 2、净利成长第 1**——同业池**成长性最优**
- **长电营收成长第 4、净利成长第 3**——但**绝对净利体量同业最大**
- **二者趋势对比**：通富 2025 净利同比 +78%（修复中）vs 长电 2025 净利同比 -3.4%（震荡），**短期景气度通富明显更强**
- **风险点**：通富 2022→2023 净利从 5.62 亿暴跌至 1.67 亿（-70%）的剧烈波动 = AMD 周期传导的强证据；长电 2023 净利 -55% 也显示赛道整体周期性极强

- **中性情景（AMD 2026 出货 +15%，行业稳健）**：通富净利 +30% 至 15.8 亿 / EPS 1.04，PE 48x，PB 5x 维持，股价持平；长电净利持平 15.6 亿，估值持平，股价持平
- **悲观情景（AMD 2026 业绩下修 / 全球 AI 算力 capex 见顶）**：通富净利 -20% 至 9.7 亿 / EPS 0.64，PE 78.5x，PB 杀至 5y 50% = 2.74x → 股价 -44%；长电净利 -10% 至 14 亿 / EPS 0.78，PB 杀至 2.0x → 股价 -29%
- **结论**：**长电下行风险约 -29%，通富下行风险约 -44%**（PB 5y 91% 偏贵的代价）；**上行空间长电 +30% > 通富 +0%**（成长已被市场定价）——风险收益比明显倾向**长电**

## 八、数据来源 & 局限

### 数据来源
- **财务时序**：akshare `stock_financial_abstract_ths(symbol, indicator="按年度")`（2018-2025）
- **估值历史**：akshare `stock_zh_valuation_baidu(symbol, indicator, period="近5年")`
- **实时价格 + 总市值**：腾讯 HTTP `qt.gtimg.cn/q=` 批量接口（2026-04-28 13:00 快照）
- **净利润反推**：EPS × (总市值 / 股价) 估算总股本，存在 ±2% 误差

### 已知局限

1. **2025 年报多为同花顺整理口径，部分公司或为快报 / 业绩预告**，最终审计数可能微调
2. **PE-TTM 包含非经常性损益**（政府补助 / 投资收益 / 资产减值等），亏损 / 微利年份严重失真——**甬矽 PE 233x 即为此类失真**
3. **通富 TFAMD 苏州 + 槟城合资公司财务并表口径**未单独披露，AMD 应收账款 / 应付账款的关联交易细节有限，影响通富现金流分析精度
4. **长电星科金朋海外业务**（韩国 + 新加坡）的折旧政策与江阴本部不同，2025 毛利率 14.15% 含一定折旧错配
5. **晶方 CIS 业务**对索尼 / 豪威 / 思特威的客户出货节奏极敏感，2026 苹果 iPhone 17 / 安卓旗舰 CIS 升级是关键变量
6. **未做 DCF 详细测算**——封测周期股 DCF 不可靠，本文采用 PEG + PB 双轨道 + 业务质量系数法
7. **AI 算力链红利持续性是同业整体最大不确定性**——若 AMD MI400 / NVIDIA Rubin 2026H2 兑现不及预期，通富估值杀伤巨大；长电相对免疫
8. **未纳入 2026Q1 季度数据**，成长性判断仅在年度颗粒度
9. **国际同业（日月光 / 安靠 / 京元电子）对标缺失**——A 股横向视角不纳入，但若做估值锚定需补充

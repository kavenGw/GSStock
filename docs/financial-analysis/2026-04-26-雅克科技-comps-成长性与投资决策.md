- 雅克 vs 安集：**互补关系**（雅克前道前驱体/特气 vs 安集 CMP/湿化学品，下游同为晶圆厂工艺段）
- 雅克 vs 鼎龙：**互补关系**（前驱体 vs CMP）
- 雅克 vs 华特：**部分重叠**（电子特气）
- 雅克 vs 彤程：**部分重叠**（光刻胶）

### 为什么不纳入

- **江化微（603078）**：业务以 LCD 用湿化学品为主，半导体级占比偏低，与雅克前驱体定位差异较大
- **晶瑞电材（300655）**：双氧水 / 光刻胶辅材为主，规模偏小（~10 亿），定位于细分耗材
- **新莱应材 / 富乐德**：设备零部件 + 清洗服务，非材料本身
- **国际同业（SK Materials / Versum / Linde / Air Liquide）**：A 股市场比较视角不纳入

## 六、投资决策矩阵

### 单股结论

#### 雅克科技（002409）— **观察（5 家中估值最低，但 ROE 偏弱）**

**优势**：
- **PE 5y 分位 36%**，6 家中**唯一处于历史中低位**
- 营收 / 净利 4 年单调上行，2024 净利同比 +50%，无明显周期回撤
- 前驱体业务为半导体存储/逻辑工艺核心耗材，国产替代叙事清晰
- 平台型布局（前驱体 + 特气 + 光刻胶 + LCD + 新能源材料）抗单一业务波动

**劣势**：
- **ROE 长年 8-10%**，6 家中倒数第二，资产效率偏低
- 毛利率 31% 被低附加值业务拖累（vs 安集 57% / 鼎龙 51%）
- **资产负债率 4 年从 18% 升至 39%**，加杠杆扩张存在周期下行时风险敞口
- **2024 OCF/EPS 降至 0.69x**（前期 1.0x+），现金流质量边际下滑
- **2025 年报尚未披露**，是当前最大不确定性
- UP Chemical 并购形成商誉，需关注减值风险

**触发买入条件**：
- 市值 < 350 亿（PE-TTM 跌至 < 30x）+ 2025 年报披露 OCF/EPS 回升至 >1.0x
- 或行业明确进入下行周期、PE-TTM 跌至 < 25x 时分批积累

**触发卖出条件**：
- 2025 年报披露后毛利率连续两季下滑 + 应收账款增速 > 营收增速 1.5x
- UP Chemical 商誉减值预告

#### 南大光电（300346）— **不买（PB 5y 89%）**

- **5 年单调增长 + ROE 修复至 13%** 是亮点
- 但当前 PE-TTM 106x 隐含极高预期，**PB 5y 分位 89%**
- ArF 光刻胶量产已被市场充分定价
- 触发观察：PE < 50x（市值 ~175 亿，对应价格 ~25 元）

#### 安集科技（688019）— **观察（5 家中 buffett 视角最优，但仍非便宜）**

- **5 年营收 / 净利 CAGR 双冠 + ROE 持续 >20% + 毛利率 56% 同业最高**
- **资产负债率 30% 健康** + 无任何并购商誉
- PE 55x（5y 分位 41%）+ PEG ≈ 1.5x，**6 家中估值最匹配成长性**
- 唯一警示：2025 OCF/EPS 降至 0.56x，需检查应收账款变化
- **触发买入条件**：PE < 35x（市值 ~270 亿，对应价格 ~155 元）
- **可纳入观察池，分批积累的等待区**

#### 鼎龙股份（300054）— **不买（PB 5y 98%）**

- CMP 抛光垫国内独家 + 毛利率从 33%→51% 是真实业务改善
- 但 2020 净利亏损 1.6 亿 + 2023 净利再次 -43% 显示周期波动大
- **PB 5y 分位 98%** 接近历史最贵
- 触发观察：PE < 35x + 净利两个完整周期内不再亏损

#### 华特气体（688268）— **不买（极端高估）**

- **业绩最差**：营收 3 年停滞，净利 3 年下行（2.07→1.35，-35%）
- ROE 6.8%（5 家最低）+ 资产负债率从 14% 升至 42%
- **PE 5y 分位 99% + PB 5y 分位 97%**——基本面与估值极端背离
- 4-26 当日涨停（+20%）疑似情绪/事件性炒作，**Pass**

#### 彤程新材（603650）— **不买（已有单股结论）**

- 维持 [2026-04-23 buffett 报告](../analysis/2026-04-23-彤程新材-buffett分析.md) 结论
- 横向对比新增证据：**5 年净利仅恢复至 2020 水平 + OCF/EPS 长期 <0.6x + 资产负债率长期 50%+**
- 估值反映了"光刻胶概念"溢价，但实际业务结构以橡胶助剂为主

## 七、监控指标与触发条件

### 季度跟踪指标（6 家通用）

7. **未做 DCF 详细测算**——半导体材料周期股 DCF 不可靠，本文采用 PEG + 业务质量系数法

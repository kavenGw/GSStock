## 0. 执行摘要

**良率维度胜宏明确领先**（6 阶 24 层 HDI ~92% vs 沪电"稳定"未给数字），这是胜宏 PE 71x 估值溢价的真正基本面源头；**Rubin 份额是分工而非竞争**——胜宏拿 GPU 算力板卡级（绝对值大、客户集中 NVIDIA >50%）、沪电拿系统级互联背板（绝对值次、客户分散 NVIDIA / 博通 / Marvell / Google / Cisco / Arista），两家在 Rubin 都是受益方而非零和。

**叠加 NVDA FY27Q1（5-20）增量信号后**：① 数据中心 752 亿 / +92% YoY、Q2 指引 910 亿 > 共识 860 亿 → "AI capex 不放缓"假设强化；② Jensen 明确「Vera Rubin **全生命周期供应紧张**」+「需求已抛物线化」+ Rubin **calendar Q3 2026 量产 / Q4 爬坡** → 胜宏 "接力期空窗风险" 窗口压缩到 26Q2 单季；③ 沪电 OCF/EPS 0.42x 的"主动备货 GB300/Rubin 接力期"解释相对"应收回款问题"解释更可信，但仍待 26 中报验证。

**新增维度 + NVDA 财报衔接未改变 4-29 / 5-1 既有目标买点**：胜宏 100-180 元 / 沪电 < 67 元——**Mr. Market 在 NVDA 财报后给胜宏跳空但盘中走低（"report is strong but stock slides"），证明完美故事已 100% 定价**，新增的"全生命周期紧张"利好已被透支。

**核心反差**：
- **算力板卡级**：胜宏 50% 份额 → 单板 ASP 高（GB300 单机价值量 800 美元），但客户集中度极高

## 3. 两家公司的 Rubin "分工地图"

```
NVIDIA Rubin 平台 PCB BOM 拆分（券商推断）
├─ GPU 算力主板（HDI 6 阶 24 层 / 78 层正交背板）— 胜宏 ~50% 独家份额
├─ 系统级中背板（HLC 40 层）              — 沪电主力
├─ CPO 光电融合 PCB                        — 沪电（2025 已 5 亿订单国内第一）
├─ NVLink 交换板 / 互联板                  — 沪电 + 深南
└─ 电源板 / 辅助板                          — 景旺 / 鹏鼎 / 兴森
```

**结论 1**：Rubin 不是"胜宏 vs 沪电"的零和博弈，是**算力级（胜宏）+ 互联级（沪电）+ 周边（深南 / 景旺）的产业链分工**。
**结论 2**：从 GB200 → GB300 → Rubin 的升级中，沪电的"系统级互联"敞口增长速度大概率快于"算力板卡"（因为 Rubin Ultra 引入 NVL144 / NVL576 这类**互联拓扑大幅扩张**的架构）—— 这与 26Q1 沪电营收同比 +54% > 胜宏 +28% 的差异部分吻合。

## 4. 风险点反着读

### 胜宏：完美执行假设已被定价

- PE 71x（5y 96% 分位）对应"良率 92% 维持 + Rubin 50% 份额维持 + NVIDIA 单一客户结构延续"的完美执行假设
- **任何接力期良率下行**（GB200 → GB300 → Rubin 工艺难度阶梯式上升，良率拐点可能在 26H2-27H1 出现）
- **任何份额被切走**（立讯精密 PCB 厂、景旺、鹏鼎都在抢 Rubin 配套订单）= 30%+ 估值杀伤

### 沪电：良率"稳定"+ OCF 红灯的组合风险

- 良率话术不给数字 + 26Q1 OCF/EPS 0.42x（vs 5y 同期 1.7-1.9x 断崖）→ 指向"以良率 / 存货 / 应收换份额"的可能性
- 验证窗口：26 中报应收账款 + 存货明细。如应收增速 / 营收增速 > 1.5x 则升级"质疑"
- 系统级互联 PCB 比 GPU 算力板更易被 follower（深南 / 景旺）切入，份额护城河比胜宏浅

### 7.5 Mr. Market 当日反应（关键反差）

NVDA 财报发布后："report is strong but stock slides" —— 数据全方位超预期，股价反而下跌。

> **Buffett 视角**：「这是最经典的'好消息已被定价'信号。当一份增长 85% / 数据中心翻倍 / 指引超预期 / 全生命周期紧张的财报无法推动股价时，意味着市场对 NVDA 本身的估值已经过满。这种情绪会在 24-48 小时内**传导到 PCB 链**——胜宏 / 沪电的估值如果跟随 NVDA 短期回落，那是估值回归、不是基本面恶化。**对未持仓者，这反而是耐心等待的最佳验证**——基本面好得不能再好的时候股价不涨，正是赔率最差的窗口。」

## 8. 整体评价

> 「给我两份产品规格表，我大概能猜出哪家公司更值钱。但给我两家公司的良率数据加客户名单，我能算出哪家公司更值得拥有。
>
> 胜宏的 92% 良率 + Rubin 50% 份额，是**全 A 股 AI PCB 链最强的硬指标组合**——也是 PE 71x 估值的合理基础。但这套故事的脆弱性在于：所有溢价都建立在"接力期不出错"的前提上。GB300 良率维持 92%、Rubin 良率从首批 60% 爬坡到 80%+、NVIDIA 单一客户付款不出问题——三件事任何一件失手，估值就会从 71x 砸向 35-40x。
>
> 沪电的良率不肯给数字，但客户结构告诉我答案——能同时拿到 NVIDIA Rubin Ultra 中背板 + Google TPU V7/V8 30% 份额 + 博通 ASIC 配套订单，**良率不可能太差，否则这些客户的 supplier audit 不会通过**。但 OCF/EPS 0.42x 这个红灯告诉我，沪电正在**用现金流换市占**——这是相对良性的扩张行为（备货 GB300 / Rubin 接力期），但也可能是恶性的（应收账款替客户当银行）。26 中报会给答案。
>
> Rubin 不是两家公司的对决，是 AI PCB 整组的'第二波兑现'——胜宏吃 GPU 板卡级红利、沪电吃系统级互联红利、深南 / 景旺吃周边。**作为投资者，我宁可在 Rubin 量产兑现后、估值回到合理位（胜宏 PE 25-30x / 沪电 PE 30x）再上车，而不是现在为'两代接力红利不间断'的完美故事 100% 付费**。
>

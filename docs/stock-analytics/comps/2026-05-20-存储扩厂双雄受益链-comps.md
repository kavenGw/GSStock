## §0 执行摘要

**事件触发**：2026-05-17 长鑫科技在上交所披露科创板 IPO 申报稿（拟募集 295 亿），叠加长江存储已在 232 层 NAND 量产、双雄合计 2026-2028 资本开支预估 1,500-2,000 亿规模，**国内存储制造扩产进入"招标下单 → 设备到货 → 材料量产 → 终端涨价"四阶段同步演进期**。

**核心结论（含 3 个"反共识"判断）**：

1. **设备层是当下首选受益层** —— 北方华创/中微公司/拓荆科技 26Q1 单 Q 营收同比已兑现（北方华创 26Q1 营收 103 亿，单 Q 营收已接近全国半导体设备月均订单的 1/3），但相比材料/封测层，**设备层订单到收入的传导周期最短（已在 26Q1 报表中兑现）**；同时设备的 PE 5y 分位（48-79%）较材料/设计层更温和，估值并未透支
2. **材料层主要标的均 PB 5y 分位 ≥ 90%（雅克/南大/江丰/宏和/西部/圣泉/中巨芯）** —— 行业知名度提升 + 涨价预期已被市场充分定价；**估值最贵的是石英股份（PE 5y 分位 99.7%）、宏和科技（PE 5y 分位 97.2%、PB 5y 分位 99.3%）和圣泉集团（PE 5y 分位 99.0%）**，其余材料股 PB 5y 分位 60-95% 不等
3. **模组商（江波龙/佰维/德明利/朗科）26Q1 业绩兑现极强** —— 净利同比 +1500%~+4900%（数据需以年报扣非确认），是这轮存储涨价"放大器"位置；但 **PB 5y 分位 90-98% 估值已透支**，按"业绩 vs 估值"双线看，**朗科科技、德明利是中下游模组商里估值最贵的两只**，江波龙居中

**双雄扩厂 → 受益顺序的真实节奏**：

```
26 Q1-Q2 ── 设备订单 + 设备验收 + 模组商现货涨价兑现（首波兑现，已发生）
   ↓
26 Q3-Q4 ── 材料端持续放量（前驱体/特气/靶材/光刻胶）+ 模组商利润持续高位
   ↓
27 H1   ── 封测/模组下游产能拉动（太极/深科技/江波龙单 Q 持续高位）
   ↓
27 H2 - 28 ── 设计公司利基存储二次涨价（兆易/君正/普冉/东芯）+ DDR5 接口（澜起/聚辰）
```

## §1 双雄扩厂全景

### 1.1 长鑫科技（CXMT）— 5/17 申报稿确认信号

**基本信息**：
- 上交所科创板申报稿 2026-05-17 披露，详细数据已在 [长鑫科技 buffett 分析](../sectors/semiconductor/storage/2026-05-18-长鑫科技-buffett分析.md) 拆解
- 拟募集资金：**295 亿元**（74.5 亿技改 + 130 亿 DRAM 升级 + 90 亿前瞻研发）
- 发行规模：106.22 亿股（占发行后 10%），**隐含发行估值 ~2,950 亿**
- 2025 营收 617.99 亿 / 归母 18.75 亿；2026 H1 预告营收 1,100~1,200 亿 / 归母 500-570 亿
- 三厂布局：北京 + 合肥 + 合肥二期，**招股书未披露月产能数字**（这是 §1 的关键信息空白）
- 工艺路径：自承"工艺与三星/SK 海力士/美光仍有一定差距"；HBM 在招股书全文无实质产品提及（之前媒体翻译"HBM3 量产"无招股书依据）
- 客户：**前五大集中度 68%**，全部中国客户（阿里云/字节/腾讯/联想/小米/传音/荣耀/OPPO/vivo），**无华为、无海外**
- 治理：无控股股东 + 无实控人；第一大股东清辉集电 21.67%；一致行动协议覆盖 ~43% 表决权

**资本开支信号**：
- 2024 固定资产账面 1,531 亿 → 2025 末 1,830 亿（+19.5%）；2026 Q1 资产总额 3,881 亿（环比 +15.3%）
- 2025 年折旧 246.8 亿、研发投入 95.93 亿（研发占营收 15.5%）
- **5/17 申报稿募投 295 亿全部用于扩产升级 + 前瞻研发**，对应 26-28 设备/材料采购窗口

## §2 扩厂时序传导：从订单到终端涨价

**核心传导链**（**§7 结论的逻辑骨架**）：

| 阶段 | 时间窗 | 受益类型 | 代表标的 | 已兑现/待兑现 |
|------|------|------|------|------|
| 1. **招标下单期** | 25 H2 - 26 H2 | 半导体设备订单确认 | 北方华创、中微、盛美、拓荆 | **已兑现**（26Q1 营收 yoy 显著正） |
| 2. **设备进场调试** | 26 H2 - 27 H1 | 设备验收 + 测试机/检测设备 + 首单耗材 | 华峰测控、赛腾、矽电、奥特维 | 部分兑现 |
| 3. **量产爬坡期** | 27 H1 起 | 前驱体/特气/光刻胶/靶材持续放量 | 雅克、南大光电、中巨芯、江丰电子、彤程、宏和、圣泉 | 待兑现 |
| 4. **满产稳态** | 27 H2 - 28 | 封测/模组拉动 + DDR5 接口需求 | 太极实业、深科技、澜起、聚辰 | 待兑现 |
| 5. **终端涨价兑现** | 26 至今持续 | 利基存储设计公司 + 模组商 | 兆易、君正、普冉、东芯、江波龙、佰维、德明利、朗科 | **已兑现**（26Q1 净利暴增） |

**两个观察**：
- **链条两端先兑现，中间段后兑现**：设备 (1) 和终端涨价 (5) 26Q1 均已实证，但 (3)(4) 材料/封测层财报兑现要等 26 H2-27 H1
- **市场预期已 price-in 全链条**：从估值角度看（PE 5y 分位 + PB 5y 分位），材料/设计/模组三层 **大部分标的 PB 分位 >85%**，意味着市场已经把"未来 2-3 年传导"基本定价进去，**安全边际只剩 EPS 兑现速度 / 节奏快于市场预期** 这一个变量

## §7 结论与买点

### 7.1 主受益序列（按估值/弹性双重考虑的优先级）

**A 类 — 估值温和 + 受益强度高（可优先关注）**：

- **兆易创新（603986）** — 双 core、PE 5y 分位 56%、26Q1 净利 yoy +331%，存储 + NOR + MCU 三轮驱动，业务最干净
- **北京君正（300223）** — 双 core、PE 5y 分位 48%、车规 SRAM + DDR3 利基 + AI 双弹性
- **北方华创（002371）** — 设备龙头、PE 5y 分位 48%、26Q1 营收 103 亿单 Q 创新高
- **中微公司（688012）** — 介质刻蚀 + 3D NAND 不可替代、PE 5y 分位 64%、26Q1 营收 yoy +59%

**B 类 — 业绩兑现极强但估值已透支（择机减仓 / 不追高）**：

- **江波龙（301308）** — 26Q1 净利 yoy +2644%，长江存储承接最直接，但 PB 5y 97.5% + 单股价超主题上限
- **佰维存储（688525）** — 26Q1 净利 yoy +1567%，PB 5y 90.6%，估值已透支
- **德明利（001309）** — 26Q1 净利 yoy +4943%，PB 5y 94.8%，单股 ¥699 远超主题上限
- **拓荆科技（688072）** — PB 5y 97.9%，设备层最贵的
- **澜起科技（688008）** — PE 5y 91%，DDR5 全球三强但估值在 2021 顶部水平
- **普冉股份（688766）** — PB 5y 96.3%，26Q1 兑现强但已 fully priced-in

**C 类 — 主题外或已 exclude（仅列对照）**：

- 长鑫科技（CXMT）：commodity 周期顶部 IPO、3,000 亿发行无安全边际（[长鑫 buffett](../sectors/semiconductor/storage/2026-05-18-长鑫科技-buffett分析.md)）
- 东芯股份（688110）：PB 19.4x 极端高估 + 连续亏损（[东芯 buffett](../sectors/semiconductor/storage/2026-05-19-东芯股份-buffett分析.md)）
- 江丰电子（300666）：PE 102 / PB 5y 99%、5/19 文档判减仓 50-100%（[江丰 5/19](../sectors/semiconductor/materials/2026-05-19-江丰电子-buffett分析.md)）
- 石英股份（603688）、宏和科技（603256）、圣泉集团（605589）：PE/PB 5y 均 ≥90%、26Q1 兑现弱或方向相反
- 矽电股份（688362）：PE 252 + 26Q1 净利 -54%、估值与基本面方向相反

### 7.2 三大风险

1. **地缘/制裁升级**：长鑫一直是美国列实体清单优先目标，长江存储已在实体清单，**任何新一轮设备/材料出口管控会直接压制扩产节奏**，进而打击设备/材料层 EPS（最敏感是北方华创/中微国产替代率最高的标的，反向逻辑是封锁加深 → 国产替代加速 → 反弹）
2. **存储现货价反转**：DDR5/NAND 现货价已连涨 6-8 个月，从 [利基存储涨价 comps](2026-05-15-利基型存储涨价-SLC-NAND-comps.md) 推算 26 Q3 可能见顶；现货价回落 → 模组商单 Q 净利转向库存跌价 → §5 三只模组商财报急刹车
3. **HBM 进口替代延迟**：HBM 是这轮 AI 算力链的核心，长鑫招股书全文未提 HBM（仅"高带宽"形容 MRDIMM/LPDDR4X），意味着 **HBM 国产替代 26-28 都不会兑现**，国内"HBM 概念股"（澜起测试机、香农芯创分销）的故事是"AI 进口需求大"而非"国产替代"，**逻辑不一样**

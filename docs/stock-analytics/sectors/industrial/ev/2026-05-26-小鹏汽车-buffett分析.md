## 结论

**Don't Buy（Skip）**——这是巴菲特最避之不及的那类生意：资本密集、无定价权、技术快速迭代、长期亏损、需要持续融资稀释股东。小鹏管理层诚信尚可，但生意本身的"先天缺陷"无法靠人来补救。

## 能力圈评估

**边缘偏外**。可解释其挣钱方式（卖车 + 软件订阅），但**无法可靠预测 10 年后行业格局**——这违反巴菲特"投资是不确定性下的确定性"原则。芒格只为 BYD 破例（因王传福 + 早期估值极低），且明确说"不懂大多数车厂"。小鹏不具备 BYD 那种全产业链垂直整合 + 电池自供 + 海外打开局面的特征。

> "We don't own technology stocks because we don't know which players will end up with truly durable competitive advantages." — 1999 Letter

整车制造业 + 智驾算法迭代 = 双重不可预测。

## Key Assumptions（拒绝投资的依据）

1. **新能源车行业未来 5-8 年是消耗战**，中国市场已有 100+ 品牌在卖，最终能活下来的不会超过 5 家
2. **智驾不构成持久护城河**——算法可复制、数据可积累、芯片可外购（小鹏图灵 + 英伟达双路线本身就说明替代性强）
3. **汽车制造的资本回报率天然低**——巴菲特看了一辈子车厂，结论是"它给消费者价值，但不给股东价值"（航空业历史警示同源）
4. **当前估值未给出足够安全边际**——亏损股 PS 1.5x，没有可靠的"低于内在价值 30%"参照
5. **管理层诚信不能弥补生意属性**——好骑师 + 烂马 = 烂结果

## 商业质量

- **护城河**：弱 / 收窄。智驾领先程度领先华为/特斯拉的窗口期可能只有 1-2 年，且华为 ADS、特斯拉 FSD、Momenta 均在快速追赶
- **管理层**：诚信 ★★★★ / 资本配置 ★★ / 主人翁意识 ★★★★（何小鹏个人持股深，但融资稀释频繁）
- **商业模式**：**商品型（commodity）**——典型的"竞争中所有改进的成果都流向消费者，而非股东"
- **机构强迫症警告**：存在。"销量必须保住"导致的反复降价、产能扩张、海外烧钱——典型为了规模而牺牲毛利

## Key Risks（Top 3）

1. **价格战长期化**：行业每轮降价 5-10%，小鹏毛利无缓冲空间，盈亏平衡点持续后推
2. **智驾代际换代**：端到端大模型快速演化，下一代芯片+算法可能让当前积累折旧加速
3. **融资稀释**：现金消耗速度迫使每 18-24 个月一次定增，股东权益持续摊薄

## Overall Assessment

> "我用了一辈子告诉自己，汽车行业是给消费者送福利、不是给股东送回报的地方。"

小鹏汽车有一个真诚的创始人、一支不差的工程团队、一个还在烧钱讲故事的资本结构。但巴菲特看一家公司，先看"生意"，再看"人"，最后看"价"——**生意这一关就过不去**：

- 它没有定价权（航空业重演）
- 它需要永远融资（纺织业重演）
- 它的技术领先不是经济商誉（不像可口可乐配方）
- 它的最好结局是"中国前三新势力车企之一"，但即便如此，长期 ROE 也很难超过 15%

如果你必须在新能源车里选一家，**比亚迪**（垂直整合 + 已盈利 + 芒格背书）是唯一接近巴菲特口味的标的，但即便是 BYD，巴菲特/芒格也在 2022 年后陆续减持——这本身就说明问题。

**建议**：跳过个股，如果一定要参与新能源车主题，往**上游卖铲子**走（电池材料、半导体、PCB、连接器、AI 算力链），那才是这个产业链里**有定价权 + 高 ROIC + 可估值**的位置。

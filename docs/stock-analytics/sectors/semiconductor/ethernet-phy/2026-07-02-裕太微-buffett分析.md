# 裕太微（688515）buffett 深度分析：国产替代真卡位，但太贵、太早、还在亏

## §0 结论摘要

**评级：Keep Watching（观察）。** 裕太微是一门"看得懂、值得追踪、但现在绝不能买"的生意——以太网 PHY 国产替代是真 thesis（大陆首款量产 2.5G PHY，国内 2.5G 网通份额 30%+），但它同时具备三个否决买入的硬伤：**连续多年亏损未摘 "-U"、全球份额个位数且对手是博通/Marvell 级巨头、PS(TTM) ≈ 30x 把最乐观的扭亏预期都透支干净**。

- 现价 234.72 元 / 总市值 187.78 亿元（2026-07-02）。总股本约 0.80 亿股。
- **期望内在价值 ≈ 96.25 亿元 / 120.3 元/股，对应安全边际 −48.7%**（深度为负）。
- **三情景一句话**：
  - **Bull（20%）**：国产替代持续 + 车载/高速放量 + 2027 前后扭亏，正常化营收 ~18-20 亿、净利率 ~20%——内在价值 ~160 亿 / 200 元/股。**即便如此，相对现价仍是 −14.8% 的负安全边际**（红线 #4 压力测试）。
  - **Base（45%）**：高增回归常态、扭亏偏慢，正常化营收 ~13 亿、给 ~7.5x PS——内在价值 ~100 亿 / 125 元/股。
  - **Bear（35%）**：竞争加剧/价格战/扭亏延后 + PS 杀估值，正常化营收停滞 ~8 亿、PS 压到 ~7x——内在价值 ~55 亿 / 69 元/股。

一句话：好赛道、真卡位、坏价格、早阶段——**用巴菲特的话，"再好的公司，买入价过高也会变成糟糕的投资"**。

## §1 能力圈 & 本次分析触发

- **能力圈判定：边界区（偏内）。** 生意模型本身简单可懂——fabless 芯片设计公司，卖以太网物理层（PHY）芯片、车载以太网 PHY、TSN 交换芯片，靠国产替代抢博通/Marvell/瑞昱的份额，收入来源清晰（网通芯片 + 车载芯片 + 交换芯片）。可懂的部分：为什么赚钱（卖芯片、毛利 42-45%）、护城河来源（先发 + 车规认证）。**边界所在**：高速率产品（10G PHY / 12.8G SerDes）的技术追赶节奏、车载交换芯片何时真正放量，需要持续跟踪半导体设计的工程兑现，超出静态判断能力，估值高度依赖 3-5 年后的营收/利润假设——这类"预期驱动、当下亏损"的标的天然确定性低。
- **本次触发（首建档）**：无基线旧档。触发点是 2026Q1 季报（营收 +74.8% YoY、亏损同比继续收窄）叠加市场把它归入"AI 算力/国产替代"叙事给到 30x PS 高估值——需要一次性把"真业务 vs 概念叙事"、"成长质量 vs 增长故事"、"贵到什么程度"讲清楚，落一份可追溯的锚。
- **为何现在看**：估值已到极端位置（PS 30x / PB 12.8x），而基本面仍全线亏损——正是需要冷静判断"市场给的溢价里有多少是业绩、多少是想象"的时点。

## §2 市场规模与跑道（成长 lens）

以太网 PHY 是有线通信的物理层芯片，下游分三条跑道，跑道长度差异极大：

| 跑道 | 裕太微当前位置 | 渗透/跑道判断 | TAM 定量 |
|---|---|---|---|
| **2.5G 网通 PHY**（基本盘） | 国内份额 30%+【软】，2025 相关营收约 1.91 亿、+34.6%【软】 | **已 30%+，跑道相对有限**，靠份额提升 + 网通升级驱动 | 第三方硬数【缺】 |
| **车载以太网 PHY / TSN 交换** | 百兆/千兆 PHY 已量产，YT99 TSN 交换 2025-04 发布、导入十余家车厂【软】 | **未渗透跑道、但产品在导入/送样期**，交换芯片"最快 2026H2 才贡献营收"【硬-公司口径】 | 第三方硬数【缺】 |
| **数据中心/AI 高速互联**（10G+ PHY / SerDes） | **无产品能力**（公司官方口径）【硬】 | **纯规划、无量产、无营收** | 第三方硬数【缺】 |

- 成长 lens 关键：**2.5G 网通这条已吃到份额的跑道渗透率高、剩余空间有限；真正长的跑道（车载放量、高速率）产品都还在导入/研发期，收入能见度弱**。
- **TAM 第三方硬数全线【缺】**（东吴深度 PDF 二进制无法解析，中国 PHY 市场/车载以太网/数据中心网络 TAM 定量口径本轮未取到）。仅有定性锚：全球 PHY 前五占 91%、国内替代空间大——**方向对，但缺可验证的量级刻度**，任何"想象空间"论述都建立在无第三方数据支撑之上，需打折看待。

**盈利能力判定**：

- **持续亏损，但趋势明确收窄**：2024 亏损见峰（扣非 −2.32 亿），2025 全年亏损同比收窄 33.7%（扣非收窄 32%），2025 逐季收敛、Q4 归母仅亏 623 万近打平。2026Q1 亏损回到 −4326 万属科创半导体设计 Q1 季节性淡季（营收环比 2.29→1.42 亿回落），但 YoY 仍 +74.8%、亏损同比仍收窄 28.8%【硬】。
- **盈利质量口径干净**：扣非仍全线亏损，但归母与扣非缺口不大（非经常损益不大）——**不是靠非经常性损益粉饰**，看扣非不看归母的原则下，公司实打实还在亏。
- **毛利率 42-45% 稳定**，无明显价格战侵蚀迹象【硬】。
- **增长拆解——量价 vs 盈利性**：高增来自**量（份额提升 + 新品放量）**驱动，价（毛利率）平稳；但**净利持续为负**。这是典型的"**烧研发换营收 + 份额**"模式——研发费率高达 ~59%【软-公司口径】，营收增长的钱都投进了研发和产品线扩张，**不是盈利性增长**。2025 起亏损收窄是增长质量改善的**早期信号**，但远未到"增长即盈利"的阶段。
- **资产负债表**：资产负债率仅 ~11%【硬】，轻资产 fabless、无杠杆压力，最坏情景（营收 −30%）下不存在债务违约风险——**财务安全性高**，这是它能"熬"到扭亏的底气。但经营现金流每股为负（2025 −2.17 元/股）【硬】，扭亏依赖营收继续放量摊薄 59% 研发费率。

## §4 全球竞争力

- 全球 PHY 前五（博通、Marvell/美满、瑞昱、TI、高通/微芯）合计份额 **91%**，国内境外前五占 87%，瑞昱国内占比高【软】。
- 裕太微 2021 年全球 PHY 份额仅 **2%**【软】——**绝对份额个位数，是全球盘子里的挑战者**。
- 现实定位：**只在 2.5G 网通这一细分吃到国内 30%+ 份额，全球范围仍是个位数份额的追赶者**；越往高速率（10G/交换/SerDes）走，越直接撞上博通/Marvell（已量产 224G SerDes / 800G 交换）的主场，追赶周期长。
- 判定：**国产替代逻辑真实，但"卡位成功"目前只成立于 2.5G 网通这一格；全球竞争力仍弱，护城河尚未跨出单一细分**。

## §5 护城河

- **类型**：先发优势（大陆首款量产 2.5G PHY YT8821）+ 车规认证壁垒（车载 PHY 需 AEC-Q100，认证周期长、切换成本高）+ 客户导入黏性（车厂一旦选型，量产周期内不轻易换供应商）。
- **强度：弱-中。** 先发优势在 2.5G 网通细分成立（30%+ 份额是证据），但**全球个位数份额意味着护城河窄且未经周期检验**。车规认证是真壁垒，但裕太微在车载还是低基数放量早期，尚未形成"客户离不开"的规模黏性。
- **趋势：可能拓宽，但尚未证实。** 潜在拓宽信号是车载 PHY 出货累计破 1500 万颗【软】+ 毛利率维持 42-45%（无价格战侵蚀）；但护城河能否从 2.5G 单格拓展到车载/高速，取决于产品兑现，**当前证据不足以判定为稳定拓宽**。
- **诚实评级，不拔高**：按巴菲特"护城河看趋势不看现状"的标准，这是一条**尚在挖掘、深度未知**的护城河——**结构性优势尚未确立**（个位数全球份额 + 对手极强 + 高速产品缺位），绝不能因为"国产替代"四个字就当成宽护城河给估值溢价。生意模型上更接近"需要不断重建的技术领先"（巴菲特语：需要不断重建的护城河，最终算不上护城河），而非结构性 franchise。

## §6 核心新论点：国产替代 + 车载放量，是结构性还是低基数？

**最强反驳前置（双面强制）**：

1. **份额小、对手强**：全球份额 2%、国内也只在 2.5G 网通吃到 30%+；越往高速走越撞博通/Marvell 主场。"国产替代空间大"的另一面是"从个位数份额爬起，且最肥的高速段还进不去"。
2. **车载交换芯片业绩未兑现**：YT99 TSN 交换芯片虽已导入十余家车厂、支持 60+ 项目【软】，但公司自己讲"**最快 2026H2 才逐步贡献营收，放量节奏视客户项目进度**"【硬】——**即最性感的车载交换故事，收入端还是零到接近零**，导入名单 ≠ 采购金额。

**结构性 vs 低基数二分判定**：

- **2.5G 网通份额（结构性成分较高）**：30%+ 国内份额是真实卡位，网通向 2.5G 升级是结构性趋势，这部分增长质量相对可信，但跑道渗透率已高、剩余空间有限。
- **车载以太网"增长超 14 倍"、"+215%"（低基数成分为主）**：这些惊人增速建立在极低基数上（2024 车载营收很小），**高增速主要是低基数效应**，尚不能外推为长期结构性放量——真正的结构性拐点要等车载交换芯片（YT99）2026H2 后的具名采购金额兑现来证实。
- **判定**：国产替代 thesis 有真实结构性内核（2.5G 网通份额），但市场当前给的高估值更多押注在"车载 + 高速"这些**尚在导入期、以低基数高增速呈现的部分**——**用低基数高增速给结构性估值溢价，是本标的最大的定价风险**。

## §7 AI / 概念潜力（分维度打标签）

| AI/概念维度 | 标签 | 依据 |
|---|---|---|
| **数据中心 AI 高速互联**（10G+ PHY / 12.8G SerDes / 800G 交换） | **【蹭概念】** | 公司官方口径（2026-02 调研）明确"**尚不具备在 AI 场景下高速数据中心核心网络中部署应用的技术能力**"【硬】；最高速率产品仍在研发/规划、未量产未认证未贡献营收，靠外延并购补足且"极高不确定性、长周期"【硬】。**无产品、无营收——市场若给 AI 溢价属纯叙事** |
| **数据中心配套管理/辅助网络**（100M/1G PHY） | 【真敏感-弱】 | 现有产品仅用于数据中心"配套管理及辅助通信网络"【硬】，是真业务但**非 AI 核心网、非高速互联**，价值量低 |
| **车载以太网 / TSN 交换** | 【真业务-非 AI】 | 智能汽车 E/E 架构升级驱动，是真实成长跑道，但**属汽车电子、不是 AI 算力**，别混为 AI 叙事 |
| **2.5G 网通国产替代** | 【真业务-非 AI】 | 基本盘，真实份额，与 AI 无关 |

**结论（红线 #5）**：裕太微的 **AI 敞口 = 业绩层 0、产品层 ≈ 0（仅研发规划）**。数据中心 AI 高速互联维度是【蹭概念】——**任何以"AI 算力红利"给出的估值溢价都无业绩支撑，属预期/叙事驱动**。**AI 概念绝不进入 §9 owner earnings 估值基础**——§9 的正常化营收/利润假设只建立在 2.5G 网通 + 车载以太网这些真业务上，不含任何高速数据中心贡献。

## §8 周期定位

- **裕太微不是商品周期股，是"早期成长 + 扭亏拐点"股。** 它没有铜/锂这类商品价格驱动，毛利率稳定在 42-45%，波动来自份额/放量节奏而非商品周期。frontmatter 不写 commodity 字段。
- **"拒绝顶点利润定价"在此的表现（正常化利润保守铁律）**：由于公司**当前仍全线亏损**，最大的定价陷阱不是"用周期顶利润定价"，而是"**把 bull 情景里扭亏后的高净利率 × 高营收 × 高倍数当成常态**"。§9 严格执行：
  - 正常化净利率取保守值（fabless 设计公司成熟期 15-20%，不取行业景气顶的 25%+）；
  - 正常化营收基于真业务（网通 + 车载）保守外推，不含高速/AI；
  - **2027 盈利转正是卖方预测，非公司指引**【软】——正常化利润假设据此打折，不能把未兑现的扭亏当既成事实定价。

## §9 估值：场景加权（严格执行）

**方法**：亏损股无法用 PE，owner earnings 用"**正常化营收 × 合理 PS**"为主锚，并用"扭亏后正常化净利率 × 合理 PE"交叉校验。三情景正常化营收均指 ~3 年后（2028E 口径）的真业务规模（网通 + 车载，不含 AI/高速）。总股本 0.80 亿股，实时市值 187.78 亿元。**注意：当前 PS(TTM)=30x，三档正常化 PS（7-8x）均远低于此——反映估值必然向理性区间回归的预期。**

| 情景 | 正常化营收 | 正常化净利率 | 主锚（PS） | 交叉校验（PE） | 内在价值(市值) | 每股 | 概率 |
|---|---|---|---|---|---|---|---|
| **Bull** | ~18-20亿 | ~20% | ~8x PS → 160亿 | 净利 ~3.6-4亿 × 40x → 144-160亿 | **~160亿** | **200元** | **20%** |
| **Base** | ~13亿 | ~13% | ~7.5x PS → 98亿 | 净利 ~1.7亿 × ~35x → 60亿（偏低，PS 为主锚） | **~100亿** | **125元** | **45%** |
| **Bear** | ~8亿（停滞） | 近盈亏平衡 | ~7x PS → 56亿 | 微利，不足以支撑，PS 为主锚 | **~55亿** | **69元** | **35%** |

**Bull 概率门控说明（红线 #8）**：成长证据包三要素——扩产达产=中（roadmap 清晰但多在研发/送样期）、客户能见度=偏弱（有导入名单、缺具名采购金额、缺 TAM 第三方硬数）、TAM 跑道=中（2.5G 已 30%+ 跑道有限、车载/高速在导入期）。三要素整体偏弱，**bull 概率封顶 20%**，不被增长叙事拔高。

**期望内在价值（算术自检）**：
```
期望市值 = 0.20×160 + 0.45×100 + 0.35×55
        = 32 + 45 + 19.25
        = 96.25 亿元
期望每股 = 96.25 / 0.80 = 120.3 元
```

**安全边际（诚实面对贵，红线 #4）**：
```
安全边际 = 期望内在价值 / 现市值 − 1 = 96.25 / 187.78 − 1 = −48.7%
每股口径 = 120.3 / 234.72 − 1 = −48.7%（一致）
```
→ **深度负安全边际 −48.7%**，现价把期望内在价值几乎透支了一倍。

**Bull 单独压力测试（红线 #4）**：
```
即便按最乐观 bull（160亿）：160 / 187.78 − 1 = −14.8%
```
→ **连最乐观情景相对现价都是 −14.8% 的负安全边际**。也就是说：**现价已把一个"20% 概率才成立的最好结局"都基本 price in，且仍不够**。按巴菲特对"不确定生意需 40-50% 折价"的要求，合理买入价应在 base（125 元）× 0.6 ≈ **75 元附近**，与现价 234.72 元相差近 3 倍。

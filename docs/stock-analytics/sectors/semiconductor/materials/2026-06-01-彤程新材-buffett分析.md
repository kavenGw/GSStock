## §4 全球竞争力

- **全球格局**【软·第三方】：光刻胶 CR4>70%（JSR/TOK/信越/富士）；高端 ArF/EUV 日企份额 **>90%**（ArF 细分 JSR25%/信越23%/TOK20%/住友15%=83%）。这是技术 + 客户验证 fab line + 专利的三重墙，非短期可破。

## §5 护城河（重评旧档「护城河不是自己的」）

**类型 + 强度 + 趋势**：
- **光刻胶**：无形资产（技术 + 客户验证 fab line 切换成本）+ 国产供应链安全政策红利。强度**中等、趋势走阔**——验证一条 fab 产线需数年，客户切换成本高（NRR 性质的粘性）；KrF 良率 85%→92% 印证工艺进步。
- **橡胶助剂**：成本/规模优势（第一梯队），但偏商品、定价权弱，强度弱-中、趋势稳。
- **PBAT**：无护城河，产能利用率 42%。

**重评旧档「moat that needs to be constantly rebuilt（KKPC 授权是别人的）」**：
- **缓解侧**【硬/软】：彤程对北京科华持股已增至 **96.33%**；主流叙事强调北京科华自主研发 + **KrF 树脂自产**（2004 成立、2014 国内首款 KrF、2017 进 12 寸线）。主体已并表、核心原料自产，旧担忧**部分缓解**。
- **未消除侧**【软+缺】：KKPC（韩国科隆）2018 合资本地化的**独家性条款、续约期限、技术自主程度**仍无公告级证据；多篇 2025-2026 文章已淡化 KKPC、转述「技术自主」，但**无招股书原文风险章节坐实**。
- **平衡结论**：护城河「是不是完全自己的」**从旧档的"红旗"降级为"待坐实的中等护城河"**——树脂自产 + 96.33% 控股是实质进步，但 KKPC 黑箱 + 高端技术仍依赖追赶，**不能给 franchise 级评价**。按 1991 年信三要件：光刻胶满足①需要②客户认为有粘性，但③高端无定价权（被日企压制），属**强商品化色彩的 hybrid 偏 franchise 雏形**。

## §6 核心新论点（最强反驳前置）

> **最强反驳（前置）**：在论证「光刻胶价值重估」之前，先把三个证伪点摆上台面——
> 1. **高端距天花板极远**：ArF<2%、ArFi 全进口、EUV 空白，日企锁死 >90% 高端份额。彤程 ArF 仍「验证/小批」，**先进制程的成长故事尚无业绩层证据**。
> 2. **KrF 国产化率仍 <5%**：即便彤程是国产 KrF 龙头，整个 KrF 国产替代盘子仍极小，「市占率 40-50%」分母小，**绝对营收体量（电子材料 9M 7.00亿）撑不起 PE54.5x 的全球龙头估值**。
> 3. **口径分母陷阱**：所有「国内第一」「市占率 40-50%」都是国产口径，叠加 PE54.5x，定价已计入「彤程将吃下全球替代红利」这一未被证据支撑的强假设。

**在上述反驳成立的前提下，承认结构性改善（这是本档相对旧档的实质上修）**：

- **论点 A（结构性，证据偏硬）：招股书分部数据 → 光刻胶价值重估**。电子材料分部 CAGR 42.3%、毛利率 27-31%、国产 KrF/TFT 双第一——这是**结构性国产替代**，非周期。旧档因黑箱无法给分部估值，现在可以：电子材料 9M 7.00亿、按全年口径年报「电子材料行业 9.86亿 +32.39%」，是真实增长引擎。**结构性 vs 周期性二分**：电子材料=结构性成长（给成长倍数）；橡胶助剂=周期现金牛（给周期倍数）；PBAT=周期拖累（不给正贡献）。
- **论点 B（业绩层弱-中敞口，硬软混合）：KrF → 存储扩产链**。北京科华是长鑫存储「最核心国产光刻胶供应商」，唯一稳定供 28nm/19nm DRAM 主线 KrF，**2025H1 对长鑫供货 +50%**【硬+软】。存储超级周期（2026 长鑫 HBM3 量产、长江存储扩产）→ 直接拉 KrF 需求。**这是有具名客户 + 量化佐证的业绩层敞口**，比纯主题强。

**论点综合**：彤程的真实价值在于「KrF 国产替代结构性成长 + 橡胶助剂周期现金牛」的组合，光刻胶分部值得重估溢价——**但重估的幅度，已被现价 PE54.5x 充分（甚至过度）计入**，这正是 §9/§10 给 exclude 的原因：好论点 ≠ 好价格。

## §8 周期定位

- **橡胶助剂（周期位置）**：2025 营收 -4.98%（量受轮胎需求结构调整拖累）但**分部毛利率回升至 26.4%**——处于「量底 + 价/毛利改善」的周期偏底部修复阶段；联营中策橡胶（2024 关联销售 3.83亿、占营收 11.7%）锁定大客户。泰国建基地对冲美关税【硬】。正信号：毛利率回升；反信号：海外贸易摩擦 + 量未起。
- **光刻胶（结构性成长台阶）**：非周期，处国产替代渗透率爬坡早中期——KrF 上量中、ArF 早期，结构性向上但高端兑现节奏不确定。
- **PBAT（周期底部+行业过剩）**：毛损边际收窄但产能利用率仅 42%，行业反转需禁塑令落地（屡延期），短期无 alpha。
- **正反对冲结论**：橡胶助剂周期底修复 + 光刻胶结构成长（正）vs PBAT 拖累 + 高端未兑现 + 海外摩擦（反）。整体处「基本面边际改善、但改善幅度被高估值透支」的位置——**周期/结构都在向好，问题在价格不在生意**。

## §9 估值（场景加权，正常化口径）

**正常化口径铁律自检**：26Q1 扣非 1.77亿年化 ≈ 7.1亿（**财报顶部年化**）；2025 归母 5.63亿/扣非 5.39亿。**三档正常化利润全部 ≤ 顶部年化、体现穿越周期**——bear<base<bull，且 bull 的 7.0亿仅与顶部年化持平而非超越。

| 情景 | 概率 | 正常化利润（归母口径） | 合理倍数 | 内在价值 | 赋权理由（引 §6/§8 证据） |
|------|------|------|------|------|------|
| **悲观** | 35% | 5.0亿（低于 2025 的 5.63，计 PBAT 余拖累 + 橡胶助剂量未起） | 18x（化工为主+弱主题，§8 周期偏底但无定价权） | **90亿** | 主题降温 + ArF 迟滞 + 现金流红旗未消，回归化工估值（§6 反驳①②成立时的世界） |
| **基准** | 45% | 5.8亿（电子材料延续 30% 增长但综合仍化工主导，毛利率年度企稳 25%） | 32x（穿越周期成长中枢：真分部成长 §6-A + V 形 ROE，但高端未兑现） | **186亿** | 招股书分部增长硬证据支撑 base 适度脱离纯周期均值；KrF 业绩层敞口 §6-B 给一定成长溢价 |
| **乐观** | 20% | 7.0亿（上海产线达产 + KrF 随长鑫放量 + ArF 小批转上量） | 40x（结构成长台阶 + A+H 催化估值） | **280亿** | **bull 门控**：硬证据中等（分部增长硬 + 长鑫具名 +50% 硬，但 ArF 上量/达产偏软、高端全软）→ 概率压到 20%、倍数封顶 40x，不给更高 |

**成长 bull 门控逐项核对**（命中成长 lens）：
- (a) 扩产达产确定性：上海 6.02亿光刻胶产线已部分建成爬坡=**偏硬**；ArF 上量=**软** → 半硬
- (b) 客户 capex 能见度：长鑫具名供货 +50%=**硬**；存储超级周期宏观传导=**软** → 半硬
- (c) TAM 跑道：KrF 国产渗透低、空间大=**有利**，但高端被日企锁死=**不利** → 双刃
→ **硬证据中等（两个半硬 + 一个双刃），bull 概率不给到高位（封 20%）、倍数封 40x**。

**期望内在价值 = 0.35×90 + 0.45×186 + 0.20×280 = 31.5 + 83.7 + 56.0 = 171.2 亿**
**Σ概率 = 35%+45%+20% = 100% ✓（算术自检通过）**

- **安全边际 = (171.2 − 318.7) / 318.7 = −46.3%**
- **Bull 压力测试 = (280 − 318.7) / 318.7 = −12.1%**（**连最乐观情景对现价都无安全边际**，现价透支兑现）

**高增长不许稀释「贵」（铁律自检）**：PE_TTM 54.5x（akshare 5 年分位 **57.5%，高于中位 48.2**，非 Phase A 估的「区间中下沿」）；PB 7.94x（5 年分位 **65.9%**，中位 6.36）。即便承认真成长，当前估值处 5 年中上沿、且高于历史中位——**老实算下来安全边际为负**，成长不构成为「贵」开脱的理由。

## §10 评级决策

**评级：exclude（排除，等回调）**

- **依据**：期望内在价值 171 亿 vs 实时市值 **318.7 亿**，安全边际 −46.3%；按 Buffett 边际安全档（不确定性因素存在 → 需 40-50% 折价才买），现价不仅无折价、反而溢价 86%。连 bull（280 亿）对现价都 −12.1%，**没有任何情景给出安全边际**。
- **买点阈值**：① 市值跌至 **<120 亿**（约 19.5 元，对应基准 186 亿的 ~65% 折价/或悲观情景上沿之上 30% 折价）才进入「值得重新承做」区；② 或基准情景兑现（电子材料连续 2 年 30%+ 增长 + 综合毛利率站稳 25% + OCF/净利回 0.5+）后按更高内在价值重估。
- **卖点阈值（已持仓者）**：见 §11 触发器。
- **相对旧档是否翻转**：**生意评级翻转（上修），估值结论未翻转（仍不买，但理由变了）**。
  - 旧档 watch/不买的理由：「光刻胶黑箱 + 毛利率单边商品化 + 好故事坏生意 + 护城河不是自己的」——这些**论据已被招股书硬证据推翻或大幅缓解**。
  - 本档 exclude 的理由：「生意其实不错（真分部成长 + 毛利率回升 + 扣非质量改善），但 PE54.5x/PB7.94x 已把兑现透支，连 bull 对现价都无安全边际」。
  - 一句话：**从「这不是好生意」翻转为「这是不错的生意，但不是好价格」**。Buffett 框架下两者都导向「现在不买」，但本档对生意本身的尊重远高于旧档，且明确给出回调后的重估路径——而非旧档式的「超出能力圈、走开」。

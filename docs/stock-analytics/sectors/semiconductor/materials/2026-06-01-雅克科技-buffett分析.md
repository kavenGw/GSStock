**结论**：盈利质量在边际改善（毛利 + 现金转化），但**增长的含金量被财务费用稀释**，且真实增长引擎与半导体叙事错位。

## §5 护城河（类型 + 强度 + 趋势）

| **类型** | 切换成本（前驱体先进节点认证周期长、晶圆厂换供应商风险高）+ 局部无形资产（High-K 国内唯一规模供货 + HBM 链绑定） |
| **强度** | **中**。切换成本真实但雅克处第二梯队，先进配方广度不及 Merck/SK；护城河更多体现在"国内唯一规模化"的相对位置，而非全球结构性壁垒 |
| **趋势** | **稳定偏走阔（脆弱）**。正面：前驱体毛利率 +3.66pct 至 44.79%（定价权在改善，是走阔信号）；负面：前驱体整体稼动率仅 59.31%、二期还在客户认证、国内同业（南大光电等）同步扩产——"需要不断重建的护城河算不上护城河" |

**商业模式判定（franchise vs commodity）**：
- 前驱体业务**接近"窄护城河特许经营"**（认证壁垒 + 切换成本，可维持高于资本成本的回报），但不是 See's Candies 式的强特许——它仍需持续资本投入维持竞争位置（重资本扩产），不符合"资本轻、留存收益高复用"的最佳模型。
- LNG 板材 + 工程是**项目制制造业，偏商品属性**（交付节奏波动、非复利现金流）。
- 整体：**混合型，偏窄护城河 + 重资本**。Buffett 口径下属"可以但不卓越"的生意——对应估值倍数应中性，**绝不该给特许经营级的 51x PE**。

## §6 核心新论点：存储周期→前驱体传导 + 逆周期扩产二分判定

**反驳点前置（供给侧/强论点双面，最强反驳先写）**：
> 多头最强论点是"存储超级周期 = 前驱体放量，雅克是影子赢家"。**最强反驳：前驱体是跟晶圆出货量 / 稼动率走的滞后耗材，芯片涨价 ≠ 前驱体立即放量。** TrendForce 确认 1Q26 DRAM 合约价 QoQ +90~95%、2Q26 +58~63%【硬】，但同一份报告明确**"有意义的晶圆扩产要到 2027 年底~2028"**【硬】——即终端价格已涨（领先指标），晶圆增量落地慢，前驱体放量是滞后指标。**实证坐实：2025 前驱体仅 +8.01%、2026Q1 营收首次转负**——传导尚未到位。

**传导链结构性 vs 周期性二分判定（成长/扩产 lens 核心落点）**：

| 增量来源 | 性质 | 判定 |
|---------|------|------|
| 存储芯片涨价 → 当期前驱体放量 | **周期性补涨 / 赌景气** | 时间差明确，2025/26Q1 未兑现；属"赌"不属"确定增量" |
| 晶圆扩产（2027 底~2028）→ 前驱体结构性增量 | **结构性增量（远期）** | 真实但落地慢，且雅克"前驱体产能目前没提升"、二期还在认证，能否承接不确定 |
| 3D NAND 层数提升 → 单片用量倍增 | **结构性增量** | 行业趋势成立【软】，但雅克专属量化倍数【缺】 |
| LNG 板材 +44.94% | **独立周期增量** | 真实但项目制、波动大、与半导体无关 |

**扩产现状（bull 门控关键，正面写）**：
- 董秘 2026-05-28 明确"**前驱体产能目前没有提升**，未来根据客户需求增长再增"【硬】；国内二期前驱体仍在客户认证，"下半年订单能否拉满"未定。
- **前驱体整体稼动率仅 59.31%**【硬】——现有产能尚未打满，"扩产→增量营收"被认证进度门控。
- 特气内蒙 1.5 万吨 2026Q3 投产【硬】；科美特 9.25 亿增资定性为**降杠杆/债转股**（偿银行贷款占比 >50%）【硬】——侧面印证公司主动缓解负债压力。

**逆周期扩产双面判定**：
- 正面：行业承压期提前卡位，若周期兑现则先发优势。
- 负面：折旧摊销 + 财务费用（+449%）双杀已发生，稼动率 59.31% + 二期未认证 → 达产确定性不足。这是 bear 情景的核心驱动。

**Q1 转负归因纠错（推翻底稿假设）**：
- 公司口径：2026Q1 营收转负**主因 LNG 保温绝热板材"交付节奏"影响本季收入确认，业务本身经营良好**【硬，新浪 2026-04-29 董秘问答】。
- **底稿"LCD 去库存拖累"假设作废**——雅克 LCD 显示材料已非主要营收科目，年报分产品表无独立 LCD 行；也**不是前驱体走弱**（前驱体仍 +8%）。这把"转负 = 半导体景气下行"的误读纠正为"项目制收入节奏波动"。

## §8 周期定位（前驱体滞后于芯片价格周期）

**周期位置**：存储**芯片价格周期处上行高位**（DRAM 1Q26 +90~95%），但**前驱体周期滞后于芯片价格周期**——前驱体跟晶圆出货量/稼动率，晶圆扩产 2027 底~2028 才有意义落地。即雅克前驱体业务**仍处"价格周期上行但自身量未起"的滞后区**。

| 正向信号（景气向上） | 反向信号（滞后/承压） |
|---------------------|---------------------|
| TrendForce 确认存储短缺贯穿 2026【硬】 | 前驱体 2025 仅 +8.01%、26Q1 营收转负【硬】 |
| 前驱体毛利率 +3.66pct（定价权改善）【硬】 | 稼动率仅 59.31%，二期未认证【硬】 |
| HBM/AI server 需求强劲【硬】 | 财务费用 +449%，扣非仅 +5.91%【硬】 |
| 核心客户稼动率维持较好【软】 | 有意义晶圆扩产要 2027 底~2028【硬】 |

**结论**：用 Buffett 的"周期行业用 7-10 年正常化盈利"原则——**不能用芯片价格周期顶来定价前驱体**，因为前驱体的量周期滞后且未启动。当前估值把"滞后区"当成"放量区"定价，是 Mr. Market 的乐观错配。

## §9 估值（场景加权，铁律：正常化不取周期顶 + bull 被证据包门控）

**正常化口径说明**：2025 归母 10.00 亿 / 扣非 9.54 亿。正常化时区分三块：① 半导体材料（前驱体高毛利但传导滞后）② LNG 板材（独立周期、项目制波动）③ 低毛利光刻胶。三档均**穿越周期**，绝不取财报顶部年化。

| 情景 | 正常化 owner earnings | 合理倍数 | 内在价值 | 概率 | 赋权理由 |
|------|----------------------|---------|---------|------|---------|
| **bear** | **7.5 亿** | **18x** | **135 亿** | **30%** | 前驱体传导持续滞后 + 财务费用拖累延续 + 商誉减值风险 + 估值回归"重资本制造商品档"。利润低于扣非（财务费用 + 折旧持续侵蚀），倍数给周期/商品档 |
| **base** | **10.5 亿** | **28x** | **294 亿** | **45%** | 存储周期正常 + 前驱体温和放量 + LNG 平稳。利润为穿越周期成长中枢（略高于 2025 扣非 9.54），半导体材料成长股合理倍数（窄护城河 + 中成长 → 28x） |

**bull 概率门控（成长证据包，铁律）**：

**期望内在价值**：


## §10 评级决策

**期望内在价值 295.3 亿 vs 实时市值 515.72 亿 → 安全边际 -42.7% → 评级 exclude。**

- **卖出阈值（针对已持仓者）**：当前价位本身即应减持（已超 bull 内在价值）；硬触发见 §11。

**相对 comps/季报基线的变化**：

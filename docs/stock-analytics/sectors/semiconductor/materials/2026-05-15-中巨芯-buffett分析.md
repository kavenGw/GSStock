## 公司定位（一段说清）

中巨芯科技是国内电子湿化学品（电子级氢氟酸/硫酸/硝酸/盐酸/氨水、BOE/硅刻蚀液）+ 电子特种气体（高纯 HF/Cl2/HCl/WF6/氟碳类）+ 前驱体（HCDS/BDEAS/TDMAT）三业务一体的湿电子化学品平台。由浙江巨化股份 + 国家集成电路产业投资基金共同发起，2023 年 9 月科创板上市，**国内首家量产 1xnm 制程电子级 HF**。**主要客户 = SK 海力士 + 中芯国际 + 长江存储 + 华虹集团 + 台积电 + 英特尔 + 联电 + 长鑫 + 联芯**，是目前 A 股能同时通过四大头部晶圆厂认证的稀缺标的。

## Conclusion

**Keep Watching** — 客户结构是 A 股最优（SK 海力士 + 中芯 + 长存 + 华虹同框），但商品属性强 + 当前亏损 + 商誉雷未拆完，**不构成 Buffett 意义上的核心持仓**，仅作弹性观察标的。

## Circle of Competence Assessment

**位于能力圈边缘**——电子级化学品行业的核心变量（晶圆厂资本开支节奏 / 韩国 + 中国本土替代节奏 / 上游萤石-无水 HF 价格传导）可识别可跟踪；但**单只产品的毛利弹性、新产品（特气 / 前驱体）的爬坡曲线、商誉减值是否一次出清**这三点判断不确定性较大，需要持续读季报 / 调研纪要补足。

## Key Assumptions

1. **韩国 6-7 月 HF 涨价能真正传导至中巨芯报价**（而非被晶圆厂卡死，或被上游萤石/无水 HF 涨价完全吃掉）
2. **5 年需求 CAGR 25.7%（半导体 + 光伏 + 显示）真实兑现**，2025 年基数 43.6 万吨
3. **凯圣氟化学商誉一次性减值出清**，不再二次爆雷
4. **客户结构（SK 海力士 + 长存 + 华虹）保持稳定**，没有被韩系 SoulBrain / ENF / Foosung 反向夺单
5. **特气 + 前驱体板块从 2026 年起开始贡献正毛利**（前驱体 HCDS/BDEAS/TDMAT 是先进制程必需，目前国产替代率极低）

## Business Quality

- **护城河**：**切换成本（中弱） + 低成本生产者（潜在）** ——客户认证周期长（HF 在头部存储厂验证 2-3 年），认证后短期不会换；但同业（多氟多 4 万吨、滨化 6000 吨、湖北兴力、中欣氟材）同样有同等级产品，**不构成 franchise**；趋势**持平偏窄**
- **管理**：诚信合格 / 资本配置一般 / 主人翁意识中等 ——国资背景倾向稳健而非进攻型
- **业务模型**：**Commodity 偏 hybrid**（湿电子化学品偏 commodity；特气 + 前驱体偏 specialty 有 franchise 雏形）
- **Institutional imperative 警示**：**存在** —— 2023 年募资后多个扩产项目同时铺开（含凯圣商誉减值即源于此），存在为了用掉募集资金而扩产的隐患

## Valuation

由于公司仍亏损（-U 标识），**传统 PE / DCF 不适用**，只能用 PS 框架 + 涨价情景弹性：

- **基准（2025 现价 PS ≈ 当前股价对应市值 / 12.1 亿营收）**：电子级化学品同业 PS 区间 4-8x，给中性 5x → 合理市值 60.5 亿
- **乐观情景**（HF 6-7 月涨价完全传导 + 2026 营收 +30% 至 15.7 亿 + 毛利率回升至 25% 出净利 1.5 亿）：给 50x PE → 75 亿 / 给 6x PS → 94 亿
- **悲观情景**（涨价被上游吃掉 + 商誉二次减值 + 增速回落至 +10%）：维持亏损，5 年内难盈利，PS 压回 3x → 40 亿

**Buffett 经典方法（owner earnings × 倍数）当前对本标的失灵。** 建议放弃精确估值，改为：**等 2026H1 季报看毛利率拐点出现 + 凯圣商誉清零后再用 normalized earnings × 20-25x 反推安全边际。**

## Sell Criteria 逐项检查（视同当前是否值得建仓 / 暂不持有）

1. **价格严重高估？** 当前不持有，不适用。但若上涨至上述乐观情景 90 亿+ 市值且毛利率未兑现拐点，则属高估
2. **基本面护城河毁坏？** 否 —— 客户结构仍是 A 股最优（SK 海力士 + 中芯 + 长存 + 华虹）
3. **管理诚信问题？** 否 —— 凯圣商誉减值是主动计提
4. **明显更好的机会？** **是** —— 同主题里**兴福电子（688545，SK 海力士直接入股 500 万股，电子级磷酸 + 双氧水主力盈利）** 和 **多氟多（002407，4 万吨半导体级 HF + 已盈利 + 锂电池第二曲线）**当前已盈利、护城河同等或更强，**是更优的同主题替代**

→ **结论：不建仓，挂 watch 状态**。

## Key Risks（仅列 Top 3）

1. **商品属性 + 议价权弱**：上游萤石 / 无水 HF 涨价时，下游晶圆厂同样有压价能力，**夹心层利润随时蒸发**。这是 Buffett 一贯回避的 commodity 业务结构。
2. **凯圣氟化学商誉减值未必出清**：2025 年计提 3700 万只是第一刀，需要看 2026 年是否还有后续减值
3. **韩系本土替代回流**：地缘缓和或本土材料企业（SoulBrain / ENF / Foosung）扩产 + 工艺突破，可能反向挤压中国大陆供应份额

## Monitoring Indicators

- 特气 + 前驱体板块营收占比是否升至 25%+（第二曲线兑现）

- 上游萤石 + 无水 HF 涨价无法传导至公司报价（合同条款不利）
- 出现第二次商誉减值
- 市值超过 90 亿（乐观情景上限）但毛利率仍未兑现拐点

## Overall Assessment（Buffett 视角）

> "Companies selling commodity-type products should attach a warning label to their stock: 'Competition may be hazardous to human wealth.'"

中巨芯**不是 Buffett 类型的标的**。湿电子化学品本质是 commodity 业务（产品同质化、上下游议价权双向受挤、价格随原料波动），即使客户结构亮眼（SK 海力士 + 中芯 + 长存 + 华虹同框是 A 股稀缺），也不足以构成"franchise"——竞争对手（多氟多 / 滨化 / 兴福 / 中欣氟材）随时可以用同等级产品替换。

**事件驱动逻辑（6-7 月韩国 HF 涨价 + 5 年 CAGR 25.7%）**是 trader idea，不是 investor idea。如果一定要在这条产业链里下注，**兴福电子（SK 海力士已入股 + 电子级磷酸/双氧水盈利更扎实）** 和 **多氟多（半导体级 HF 龙头 + 锂电池第二曲线 + 已盈利）** 是更优的同主题替代。

**最终决策：watch，不建仓。** 等 2026H1 季报验证三件事再重估：① 涨价毛利率传导是否真实出现 ② 凯圣商誉是否再爆雷 ③ 特气 / 前驱体第二曲线是否起量。

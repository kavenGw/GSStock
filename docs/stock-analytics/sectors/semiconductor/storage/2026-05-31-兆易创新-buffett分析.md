## §1 能力圈 & 本次重审触发

### 为何 5-31 重做

4-21 把兆易判为「边界区域，倾向于外部」，核心是三个无法可靠预测的变量（存储周期、国产代工爬坡、技术路线颠覆）。2026 年出现两个**新增量变量**，足以触发重审：

1. **供给侧结构变化**：三星 / SK 海力士 / 美光三巨头明确退出低端存储（DDR4 / LPDDR4 / 2D MLC NAND / DDR3 等利基段），把过去由它们顺手供应、定价的利基市场让给华邦 / 旺宏 / 长鑫 / **兆易**。这是 4-21 时尚未成型、5-25 部分识别的"供给侧重估"主线。

## §6 供给侧结构变化（核心新论点 — 必须双面）

### 6.1 三巨头逐家退出（含硬/软分级）

**三星 Samsung**
- 退出范围：DDR4（标准/PC/server 消费级）、2D MLC NAND、LPDDR4/LPDDR4X【软-媒体+TrendForce】
- 时间表：原计划 2025 年底停产 DDR4，后因 AI 缺货、DDR4 变"现金牛"**多次反复**——2025Q4 放缓 EOL，2026Q1 与客户签 NCNR 长约把供应锁给 server/industry 客户；TrendForce 2026-01-19 口径称三星"坚定执行 EOL"，2D MLC NAND 最后出货预计 2026 年 6 月，全球 MLC NAND 产能 2026 同比降 41.7%【软-TrendForce 2026-01-19】
- 动机：wafer 套利，产能向 HBM/DDR5/LPDDR5 高毛利产品迁移
- 分级：**【软】**（媒体 + TrendForce，无单一官方 EOL 文件）

**SK 海力士 SK Hynix**
- 退出范围：DDR4（消费/legacy），**否认完全退出消费内存**【软-TrendForce】
- 时间表：计划 2026 逐步退出，DDR4 产出已降至总 DRAM <30%、2026 将 <10%；过渡期在无锡厂"临时增产"DDR4 吃涨价红利【软-TrendForce/TechSpot】
- 动机：向先进节点 + HBM 迁移；2026 产能已售罄【软-TechSpot】
- 分级：**【软】**

**美光 Micron**
- 退出范围：DDR4、LPDDR4（**已发 EOL 通知**）；移动 NAND（停全球新品）；消费内存（关停 Crucial 品牌）；中国 server 芯片市场【硬-EOL 通知 + 软-媒体】
- 时间表（三家中最坚决最快）：2025-06 发 DDR4/LPDDR4 EOL 通知；生产在 2026Q1（约 2 月底）后停止【软-TrendForce 2026-01-19】；2025-08 中国裁员 + 停移动 NAND 新品、2025-10 退出中国 server、2025-12 退出 Crucial
- 动机：wafer 转 enterprise DRAM/SSD 接 AI 需求
- 分级：**【硬】**（美光官方 EOL 通知 + 10-Q FY2026 财报为底稿；这是三家中证据最硬的一家）

### 6.2 受益侧传导

腾出的利基段（DDR4 PC/消费/server legacy、LPDDR4/LPDDR4X、DDR3、消费 DRAM）→ 供给收缩 → 卖方主导推涨（2Q26 常规 DRAM +58~63%）→ 受益侧：华邦、旺宏、长鑫、东芯、北京君正、**兆易**。兆易真实可及性受长鑫配额约束（5-25 §5.3）。

### 6.3 周期性 vs 结构性二分判定（含可逆性反驳 — 质量红线）

> **本论点最大反驳点：三星/SK 的 DDR4 EOL 是部分可逆的。** 两家已多次"宣布停产 → 涨价后复产/延寿"（DDR4 一度变现金牛），无锡厂甚至"临时增产"DDR4 吃涨价红利——这证明 legacy 产能可以回流，是这条供给侧逻辑最大的软肋。**绝不能写成单边多头叙事。**

| 论据 | 指向 | 强度 |
|------|------|------|
| 美光 DDR4/LPDDR4 官方 EOL【硬】| 结构性 | 强 |
| HBM 单 wafer 耗用 3x、wafer 已结构性转向 HBM/DDR5 | 结构性 | 中强 |
| 三巨头资本开支克制（担忧 2028 供给过剩而不激进扩产，21经济网 2026-03-22）| 结构性 | 中 |
| 三星/SK 反复"停产→复产/延寿"、无锡临时增产 DDR4 | **周期性/可逆** | **强（反驳）** |
| 2Q26 报价暴涨（DRAM +58~63% / NAND +70~75%）| 周期顶部信号 | 强 |

**判定**：供给侧变化是**"结构性收缩 + 周期性放大"的叠加**——结构性的部分（美光硬退出、wafer 转 HBM）是真实且较难逆转的；但当前价格暴涨中含有大量周期性、可逆成分（三星/SK 延寿、临时增产）。**净效应是利基供给的"中枢上移"，而非"单边消失"**；一旦 2027 三巨头 wafer 回流或周期反转，价格弹性会快速回吐。这一二分直接决定了 §9 估值里 bull/bear 情景的概率分配。

## §7 AI 潜力四维度（每维度打【真敏感 / 蹭概念】）

### 维度①：间接供给侧（AI server/HBM 挤出 → 利基缺口 → 兆易受益）
逻辑链完整且多源支撑：HBM 3x wafer → 三巨头转 HBM/DDR5 → DDR4/利基 DRAM 收缩 → 涨价 → 兆易利基 DRAM 量价齐升（26Q1 净利 +523% 已兑现）。兆易 CEO 何卫明确归因利基价格回升于海外大厂向 HBM/DDR5 迁移【软-21经济网】。
→ **【真敏感】** — 这是兆易 AI 敞口里最硬的一环，已在 26Q1 财报兑现，非概念。

### 维度②：边缘 AI MCU（GD32H7 / TinyML）
GD32H7 系列 **Cortex-M7**、600MHz、片上 Flash 1024KB-3840KB + 1024KB SRAM，配 GD32 Embedded AI 部署工具（支持 tflite/onnx/keras）；GD32H78E/H789 系列 750MHz，2026 年 6 月正式量产【硬-官网】。
> **修正 4-21/任务书假设**：GD32H7 是 **Cortex-M7**，**不是** Cortex-M55+Helium（M55/Helium 是 Renesas/Infineon 路线）。5-25 §7.2 表格中"GD32H7/G55x → M55/M33"的标注亦应以官网 M7 为准。
→ **【真敏感（产品层），蹭概念（业绩层）】** — 产品 capability 真实存在，但 AI MCU 对营收的量化贡献无公开数据，正文不可夸大。

### 维度③：AI 算力卡周边（boot NOR / BMC 配套存储）
兆易官方博客：AI server 中 SPI NOR 用于 boot loading / 算力核心 / 网络交互 / 管理监控，BMC Boot Loader 典型 8/16/32/64Mb，GPU 卡旁需耐高温 NOR；AI server 单机 Flash 价值约 $100，AI server 15-20% CAGR；兆易为全球第二大 SPI NOR 厂（累计出货 270 亿颗）【软-官方博客 + 腾讯新闻 2025-04-15】。
→ **【真敏感（应用真实），但缺份额/营收拆分】** — 场景与单机价值锚真实，但兆易在 AI server NOR 的具体份额/营收无拆分，贡献度待证。

### 维度④：兆易自身 AI 产品布局（HBM / 高端 DRAM / AI 专用芯片）
兆易**自身不做 HBM、不做高端 DRAM、无自研 NPU/ASIC 公开实证**；高端 DRAM/HBM 由关联方长鑫推进（朱一明同任长鑫董事长），不在上市公司体内。兆易自研 DRAM 限于 LPDDR4（2026）→规划 LPDDR5/小容量 DDR4，定位车载/AIPC/机器人，非 AI 算力高端存储。
→ **【蹭概念 / 缺位】** — 本维度对兆易体内基本为零，AI 高端存储敞口只通过长鑫间接体现。

### AI 敏感度综合定调
兆易是 **"AI 间接受益者"，不是"AI 算力直接玩家"**。最实的是维度①（供给侧挤出，已兑现）；维度②③是真实但未量化的中长期期权；维度④缺位。市场若按"AI 存储概念股"给兆易溢价，存在**预期与体内基本面的错配**——这正是 §9 把 bull 情景概率压在 25% 的理由之一。

## §8 存储涨价周期定位

**当前是周期顶，还是结构性新台阶？答案：两者叠加，但顶部特征更显著。**

| 周期顶信号（指向回调）| 供给"出门论"对冲（指向新台阶）|
|----------------------|------------------------------|
| 2Q26 报价暴涨：DRAM +58~63%、NAND +70~75%、32G 内存条 ¥800→¥3800（21经济网）| 美光硬退出 DDR4/LPDDR4（不可逆部分）|
| 26Q1 净利 +523%、营业利润率 38.7% 历史新高（典型顶部读数）| wafer 结构性转 HBM/DDR5（3x 耗用）|
| SK 海力士 2026 产能"售罄"——售罄即近顶 | 三巨头资本开支克制（担忧 2028 过剩）|
| 三星/SK 临时增产 DDR4 吃红利 = 供给可回流 | NOR 前三占 >70%、格局集中不打价格战 |

**判定**：当前更像**"结构性中枢上移之上的周期顶"**——长期供给中枢确实上移（美光退出 + wafer 转 HBM 是真实的），但当下的价格 +58~63% 暴涨含大量周期性成分，2027 wafer 回流 + 需求侧库存消化会带来均值回归。**对兆易而言，正常化盈利应取"中枢上移后的新常态"，而非 26Q1 顶部读数**——这是 §9 base 情景的锚。参见 themes/2026-05-29-存储出口暴涨-周期顶部解读 与 themes/2026-05-26-美光DDR4扩产-影响分析专题。

## §9 估值（场景加权）

### 9.1 正常化 owner earnings 口径

26Q1 单季归母 14.61 亿（年化 ~58 亿）是**周期顶**，不可作正常化基数。采用"中枢上移后的正常化盈利"分情景估算：

| 情景 | 正常化 owner earnings（年）| 合理倍数 | 倍数依据（06-valuation 分档）| 情景内在价值 |
|------|--------------------------|---------|------------------------------|------------|
| **bull 结构性重估** | ~50 亿（供给新台阶 + GD32/NOR 量增）| ~26x | 视作"中强护城河 + 结构性成长"上限档 | **~1,300 亿** |
| **base 基准** | ~32 亿（顶部回落、中枢高于上轮周期）| ~20x | "solid moat + 中速成长"档 | **~650 亿** |
| **bear 空头** | ~20 亿（周期反转 + 长鑫配额收缩，回近周期底）| ~14x | "narrow moat + 周期/竞争威胁"档 | **~280 亿** |

> 内在价值区间 **600-900 亿**（base 650 为中枢，向上含 bull 修正空间）。与 5-25 的 300-500 亿相比再上修，反映供给侧 + AI 间接受益的进一步确认；但仍坚持周期股正常化纪律，不以 26Q1 顶部年化盈利定价。

### 9.2 概率分配（由 §6/§8 证据强度定）

| 情景 | 概率 | 赋概率理由 |
|------|------|-----------|
| bull 结构性重估 | **25%** | 美光硬退出 + wafer 转 HBM 提供结构性底色，但三星/SK 可逆性 + 长鑫配额封顶让"兆易独享退出份额"的强假设难以高概率成立 |
| base 基准 | **50%** | 最可能：供给中枢上移 + 周期顶回落并存，GD32/AMOLED NOR 撑住正常化盈利，DRAM 温和受益 |
| bear 空头 | **25%** | 2027 三巨头 wafer 回流（已多次复产为证）+ 长鑫转 DDR5/HBM 挤兆易配额 + 周期反转，三者任一发生即把利润打回周期底；当前顶部估值放大了回撤风险 |

Σ 概率 = 100%。

### 9.3 期望内在价值与安全边际

```
期望内在价值 = 1,300 × 25% + 650 × 50% + 280 × 25%
            = 325 + 325 + 70
            = 720 亿
```

```
当前市值 = 3,274 亿（2026-05-31，腾讯 qt.gtimg.cn 直连：price 467.01 / PE-TTM 113.9 / PB 13.23）
安全边际 = (期望内在价值 − 市值) / 市值
        = (720 − 3,274) / 3,274
        ≈ −78%
```

**即便用最乐观的 bull 情景 1,300 亿单独对照，市值 3,274 亿仍是其 2.5x，安全边际仍为 −60%。** 当前价格已把"结构性重估"这一最乐观叙事都 price-in 还溢价一倍——这就是"贵"的本质，护城河上修无法改变这一算术事实。

## §10 评级决策

**新评级：Watch（活跃观察）。** 期望内在价值 ~720 亿 vs 市值 3,274 亿，安全边际 −78%，距 buffett 要求的"内在价值 × 60-70% 买入"相去甚远。但相对 4-21 的"不买入 + 分批减持"，本档基于护城河 + 供给侧实证把标的从"主动回避"上修为"值得活跃跟踪、等价格的观察标的"。

| 维度 | 阈值 | 说明 |
|------|------|------|
| **买点** | 市值回落至 **<900 亿**（≈ 内在价值上沿）**且** PB <4x **且** 周期/价格回到中性区间 | 需同时满足价格 + 周期位置两条件 |
| **理想买点** | 市值 **<500 亿** + PB <2.5x + 市场恐慌（半导体板块 PB 历史低分位）| buffett 式"周期底 + 折价"双打折，对应安全边际转正 |
| **卖点（若已持有）** | 价格进一步脱离基本面 / 护城河破坏 / 长鑫配额实质收缩 / 周期确认反转 | 见 §11 触发器 |

**相对 4-21 是否翻转？** — **部分翻转：评级上修一档（不买入 → Watch），但买入结论未翻转。** 翻转的原因是护城河被证伪为"比 4-21 认为的更强"（GD32 + AMOLED NOR）+ 供给侧重估提高了利基业务可预测性；未翻转为"买入"的原因是市值 3,274 亿把所有改善都 price-in 还溢价 4x+。**好生意的认定升级了，好价格依旧不存在。**

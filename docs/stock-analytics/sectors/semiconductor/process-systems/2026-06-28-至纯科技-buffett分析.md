## §0 结论摘要（倒金字塔）

**评级：watch（观望）。** 一句话核心：至纯科技的真实身份**不是"对标盛美的清洗设备股"，而是晶圆厂高纯工艺系统集成商**——系统集成（高纯工艺/厂务/洁净室）占 2025 收入 **80.3%**、湿法清洗设备仅 **10.0%**（且属国产二三梯队，盛美中国份额 23% 为龙头）。这是一个**困境反转命题**而非成长设备股：订单前瞻确有硬催化（2026Q1 单季新签 **19.77 亿**、IC 占 94.4%、≈2025 全年新签的 78%，2026 全年目标 **45-55 亿**较 2025 翻倍，深绑中芯/华虹/长鑫/长存 + 长存三期洁净室招标 2026H2 启动），但与财务现实**深度背离**：2025 归母巨亏 **7.77 亿**、扣非自 **2023 即转负**、2026Q1 仍亏（归母 -0.79 亿/扣非 -0.84 亿）、PE_TTM 为负（-16.34，无估值意义）。叠加财务脆弱（短债 **35.27 亿** vs 货币资金 **10.06 亿**、负债率 73%、经营现金流常年为负、营业周期 **1118 天**）+ 治理红旗（2024-12 入 BIS 实体清单 + 大股东套现 12.6 亿 + 2025Q4 单季暴雷 -7.44 亿减值）。**订单≠收入≠利润**——系统集成是低毛利（23%）工程总包、垫资重、订单转利润滞后 1-2 年。

**期望内在价值（场景加权）：约 82.8 亿元总市值口径（每股约 21.6 元）**，对应当前实时市值 **135.91 亿元（现价 35.49 元 × 约 3.83 亿股）**，**安全边际 −39.1%**（当前价远高于期望内在价值，无安全边际、溢价约 64%）。即便最乐观的 bull 情景（约 115 亿 / 每股约 30.0 元，订单全兑 + 毛利修复 + 减值出清恢复历史盈利中枢），以当前价押注 bull 兑现的安全边际也仅 **−15.4%**（仍为负）——**反转逻辑真实，但当前价已把还没发生的反转提前定价**。估值绝不用 PE（亏损无意义），用 PB + 穿越周期正常化利润 + 反转情景；亏损股 PB 3.91 看似处 5 年中低位（区间 1.37–22.94），但**净资产正被亏损侵蚀**（每股净资产从 2022 的 12.19 降至 2026Q1 的 9.07），低 PB ≠ 安全边际。

**三情景概率一句话**：
- **bear 35%**：反转失败 / 流动性恶化 / 持续亏损——账面被侵蚀，给困境 PB 下沿，内在价值约 **55 亿**（每股 ~14.4 元）。
- **base 45%**：温和反转——订单部分转化、2027 回微利，正常化利润取谨慎反转值约 1.3 亿，给困境反转折价倍数，约 **90 亿**（每股 ~23.5 元）。
- **bull 20%（门控封顶）**：订单全兑 + 毛利修复 + 减值出清，恢复历史盈利中枢约 2.8 亿，给国产替代成长 + HBM 期权倍数，约 **115 亿**（每股 ~30.0 元）。

## §1 能力圈 & 本次承做触发（首建档，无旧档）

**能力圈判定：边界区（偏圈内，但纠正一个关键误读）。** 至纯的商业模式可一段话讲清，但**必须先纠正"清洗设备对标盛美"的建档误读**：按 akshare `stock_zygc_em`（2025-12-31）主营构成，至纯收入第一权重是**系统集成（高纯工艺系统/厂务/洁净室）占 80.3%**（毛利 23.67%），**湿法清洗设备仅占 10.0%**（毛利 15.84%），电子材料 9.4%。所以至纯的本质是：**给晶圆厂建"高纯水/特气/化学品输送系统 + 洁净室厂务"的工程总包商**——它卖的不是一台台清洗机，而是晶圆厂建厂时的整套高纯工艺系统集成交付，收入 = 系统集成项目（跟随晶圆厂建厂 capex、大单、工期长、垫资重）+ 设备（湿法清洗，第二曲线）+ 电子材料。**把至纯当"清洗设备股、对标盛美"会同时高估其设备竞争力、又错配其商业模式画像**（系统集成是低毛利工程总包，不是高毛利标准设备）。

**本次承做触发（全新建档，无旧档）**：本仓无至纯旧档，首建。建档背景的初始假设"至纯应为盈利的成长设备股"**被采证证伪**——至纯当前**深度亏损**（2025 归母 -7.77 亿、2026Q1 继续亏），PE_TTM 为负。所以这不是一份"成长设备股估值"，而是一份**困境反转承做**：核心问题不是"成长能给多高倍数"，而是"这家亏损、财务脆弱、治理有红旗的公司，凭 2026Q1 的订单回暖，能否在 1-2 年内把订单转化为扣非利润、修复毛利、出清减值，且当前价是否为这个**尚未发生**的反转提供了安全边际"。

**10 年视角的核心问题**：10 年后至纯是否更具竞争力？高纯工艺系统国内第一的卡位 + fab 上行周期洁净室占 capex 比例从 5% 升至 25-30%（「5 倍重构」）的产业趋势，使"行业地位更强"概率不低；**但"行业地位更强"与"现在该买"是两个问题**——更紧迫的是它能否先**活过当前的流动性紧张期**（短债 35 亿 vs 现金 10 亿）并扭亏，这是反转股区别于优质成长股的关键。

**inversion（这笔投资如何让我亏钱，列前三路径）**：① 订单回暖是真的，但系统集成执行周期长、垫资重、毛利仅 23%，订单转化为扣非利润滞后 1-2 年，期间继续亏损 + 经营现金流为负 → 流动性恶化触发被迫再融资 / 稀释 / 信用事件，账面净资产进一步被侵蚀；② 行业 2026 上行而至纯营收逆势 -20.8%——是公司自身掉队（实体清单扰动 + 项目周期拉长 + 低毛利战略订单），若掉队是结构性的而非周期性的，反转证伪；③ 以"反转 + AI/HBM 洁净室"故事在 35.49 元买入，反转兑现节奏远慢于市场预期、AI 想象证伪，估值向亏损股 PB 下沿回归（估值杀 + 基本面双杀）。

## §2 市场规模（成长 lens：TAM + 跑道长度）

**① 高纯工艺系统集成 TAM/跑道——真护城河跑道在这里，且长**：至纯真正的护城河跑道不在设备，而在**高纯工艺系统集成**。据【软·东财财富号，非公司年报，需打折】口径，至纯**特气系统市占率 48.8%（12 英寸招标第一）、化学品系统 >30%、光刻机配套洁净室 60%（国内第一）**。这条跑道的长度由两股力量驱动：(a) 国产替代——晶圆厂厂务系统过去由外资 / 港台工程商主导，国产化仍在中段；(b) **fab 上行周期洁净室占建厂 capex 比例从下行期 5-6% 升至上行期 25-30%（「5 倍重构」）**【软·机构产业框架】。2026 中国大陆晶圆厂设备支出预计 >300 亿美元、WFE 市场口径 +10% 至 550 亿美元【软·口径分歧大，作终端锚非具名订单】。**跑道判定：长**——但市占率数字均为软证据，精确份额无年报直接披露【缺】。

**② 湿法清洗设备 TAM——第二曲线，弱势**：设备业务（至微半导体）2025 收入仅 2.86 亿、占 10%、毛利 15.84%（**低于系统集成**）。清洗 + CMP 国产化率机构口径已达 50%+【软】，赛道本身不差，**但至纯在其中属第二/三梯队**（见 §4），TAM 对至纯的兑现度远低于系统集成主业。

**③ 跑道长度的诚实定调**：至纯踩的是"一个国产替代仍在中段、且受 fab/HBM 扩产拉动的高纯系统大池子（主业，跑道长但低毛利）+ 一个国产玩家拥挤的清洗设备小池子（第二曲线，弱势）"。**与拓荆（薄膜沉积，量价齐升、毛利连升至 41.69%）的关键区别在于：至纯主业是低毛利工程总包（23%），跑道长不等于盈利能力强**——TAM 支撑收入规模，但系统集成的低毛利 + 重垫资属性，决定了"跑道长"无法像高毛利设备那样高效转化为利润和现金流。这是 §9 不能给高倍数的根因。

## §3 盈利能力（最新季报兑现 + 毛利/ROE 周期 + 增长拆量价与质量）

**最新季报【硬，akshare 按单季度】**：

|---|---|---|---|---|---|---|

- **2025Q4 单季暴雷 -7.44 亿**是全年巨亏主因（大额减值 + 低毛利订单计提，单季毛利率打到 -20.3%）——这是反转股最需警惕的"一次性出清"还是"持续恶化"的分水岭。
- **2026Q1 仍亏**：归母 -0.79 亿、扣非 -0.84 亿，**盈利质量看扣非——连续亏，无非经常美化**。但单季毛利率已从 Q4 的 -20.3% 修复至 27.06%、收入降幅收窄至 -14.7%——**边际有企稳迹象，但远未扭亏**。

**多年趋势【硬，akshare 按年度】——增长质量恶化是铁证**：
- 营收：2018 6.74 亿 → 2022 30.50 亿（4 年 ~46%/年高增）→ 2023 见顶 31.51 亿 → 2024 36.05 亿（峰值）→ **2025 崩至 28.55 亿（-20.8%）**。
- 净利：2020 峰值 2.61 亿 → 2023 2.30 亿 → **2024 -1.36 亿 → 2025 -7.77 亿（雪崩）**。
- **增长质量恶化铁证**：扣非净利 **2023 即转负**（-0.45 亿），归母靠非经常损益硬撑到 2023 仍正——**盈利质量早在 2023 就破了，2024-2025 只是显性化**。毛利率从 2018-2022 稳定 ~35% → 2023 33.8% → 2024 31.4% → **2025 22.76%**。
- 销售净利率：2020 18.66% 峰 → 2023 7.28% → **2025 -27.02%**。ROE（摊薄）：2020 8.29% → **2024 -3.41% → 2025 -22.21%**。

**增长拆量价与质量的核心二分【硬】**：表面"营收 2018→2024 高增 5x"是真的，但**这段高增的质量在持续劣化**——毛利率从 35% 一路滑到 22.76%、扣非 2023 即转负。这是典型的"**以低毛利大单换规模**"（系统集成项目越做越大 → 执行周期拉长 → 垫资越重 → 毛利越低 → 现金流越差），即增长**伴随盈利能力恶化**，是"以质换量"而非拓荆式的"量价齐升"。**这是 §9 正常化利润必须诚实压低的根因：至纯历史峰值利润（2020 2.61 亿）对应的是更小营收 + 更高毛利的旧业务结构（18.66% 净利率），而当前 80% 低毛利系统集成的业务结构，决定了反转后的正常化净利率更接近 2023 的 7%，而非 2020 的 18%。**

**现金流 / 资产负债【硬，反转股最致命的一环】**：
- **经营性现金流常年为负**（系统集成垫资通病）：每股经营现金流 2022 -2.52、2023 -2.10、2024 -1.20、2025 -1.12、2026Q1 -0.08（净额约 -3042 万）。
- **短期偿债压力显著**：2025 末货币资金 **10.06 亿** vs 短期借款 **35.27 亿**（缺口约 25 亿）；流动比率 1.23、**速动比率 0.60**。
- 资产负债率持续攀升：2022 59.4% → 2024 70.8% → **2025 72.7% → 2026Q1 73.1%**。
- 周转持续恶化：存货周转 2025 535 天、应收周转 372 天、**营业周期 907 天（2026Q1 进一步升至 1118 天）**——资金被订单执行长期占用，是低毛利工程总包模式的结构性硬伤。
- **分红**：连续亏损，2024/2025 不具备分红能力，dividend_yield 填 null。

## §4 全球竞争力（清洗设备 vs 国际四强 + 盛美；系统集成 vs 国际）

**湿法清洗设备格局——至纯属国产二三梯队【硬/软】**：
- 全球：迪恩士 DNS 37% / 东京电子 TEL 22% / 泛林 Lam 17% / SEMES 10% = 前四 86%；**盛美上海全球 ~7% 排第 5**。
- 中国市场：**盛美上海份额 23%（国产龙头、且盈利、单片清洗精细化、体量远大）**；国产可供清洗设备者 = 盛美、北方华创、芯源微、**至纯（至微半导体）**。
- **至纯 vs 盛美差距明确**：至纯设备业务仅 2.86 亿收入、毛利 15.8%（低于自身系统集成）、属**第二/三梯队**。**把至纯当"清洗设备对标盛美"会系统性高估其设备竞争力**——盛美是国产清洗设备龙头且盈利，至纯设备是弱势第二曲线。

**高纯工艺系统集成 vs 国际——这才是至纯的真竞争力所在【软，需打折】**：厂务系统集成（特气/化学品输送系统、光刻机配套洁净室）国内份额第一（48.8%/30%+/60%【软·东财财富号】）。这块的对手是外资工程商 + 港台厂务总包商，至纯的竞争力来自**与国内晶圆厂深度绑定的工程交付经验 + 定制化 + 切换成本**，国产替代窗口下黏性强于标准设备。**但竞争力诚实定调：这是"国产替代 + 本土工程交付响应"的结构性卡位，而非全球技术绝对领先；且该卡位天然低毛利（23%，工程总包属性）、重垫资——是"市占率领先但盈利质量弱"的竞争力，不是高毛利特许经营。**

## §5 护城河（类型 + 强度 + 趋势）

**类型：高纯工艺系统集成的切换成本（真核心）+ 客户绑定（无形资产/工程 know-how）+ 国产替代窗口（政策催化卡位）；湿法清洗设备无显著护城河。**

- **高纯系统切换成本（切换成本）**【中-强，但低毛利】：晶圆厂的高纯水/特气/化学品输送系统 + 洁净室一旦由某家交付并跑通良率，更换厂务系统集成商的成本（重新验证 + 良率风险 + 产线中断 + 工程对接）极高。至纯特气 12 寸招标第一、深绑中芯/华虹/长鑫/长存 + 重复中标长存三期，是真实的高切换成本护城河。**但这条护城河的"变现质量"弱**——它是工程总包，毛利仅 23%、经营现金流为负、营业周期 1100+ 天，护城河带来的是"市占率 + 复购"而非"高毛利 + 高现金回报"。按 buffett "需要不断重投入垫资维系的生意算不上高质量特许经营"的标准，这是**真护城河 + 弱盈利质量**的组合。

- **湿法清洗设备（无显著护城河）**【弱】：第二/三梯队、低毛利、国产玩家拥挤（盛美/北方华创/芯源微挤压），不构成护城河。

- **国产替代窗口（结构性卡位）**【强但有政策属性 + 双刃】：实体清单倒逼晶圆厂验证国产厂务/设备，给至纯替代窗口；**但至纯自身 2024-12 即被列入 BIS 实体清单**——窗口对至纯既是机会（客户更愿用国产）又是风险（自身供应链在地化切换压力，目标年内 80% 在地化）。

**moat 趋势：稳定（高纯系统）+ 当前盈利能力恶化（财务层）。** 护城河本身（市占率第一、客户绑定）未被破坏、甚至受 fab 上行强化；**但护城河的盈利兑现能力在恶化**（毛利 35%→22.76%、连续亏损、现金流为负）——这正是"困境反转"的本质：**护城河还在，但盈利被周期 / 战略低毛利单 / 减值打到谷底，赌的是盈利能力能否随订单回暖修复回护城河应有的水平。**

## §6 核心新论点【骨干】：订单回暖是结构性反转，还是周期性补涨？

逐面拆 + 结构性 vs 周期性二分 + 受益侧传导 + 最强反驳前置 + 双面必答。

### 最强反驳前置（这家公司不能只看订单回暖的多头故事）

**反驳一——行业 2026 上行而至纯营收逆势 -20.8%，是公司自身掉队，不是周期顶定价。** 机构主流口径认为 2026 半导体设备 / 晶圆厂 capex 处**上行周期**（存储 + 先进逻辑扩产、国产替代 + 先进封装双驱动）。**但至纯 2025 营收逆势 -20.8%、2026Q1 仍 -14.7%**——即"**行业上行而公司掉队**"。这意味着：(a) 至纯当前**绝非用周期顶利润定价**，反而处自身盈利谷底（这对反转是好事）；(b) **但掉队本身要追问是周期性还是结构性**——若是实体清单扰动 + 项目周期临时拉长（周期性），反转可期；若是国产同业（清洗设备）挤压 + 低毛利模式难改 + 治理 / 财务拖累（结构性），则是价值陷阱。证据偏向"周期性掉队 + 自身财务包袱"混合，未到可断言纯周期的程度。

**反驳二——订单≠收入≠利润，转化滞后 1-2 年且低毛利。** 2026Q1 新签 19.77 亿是**硬数据**，但系统集成执行周期长（营业周期 1118 天）、垫资重、毛利仅 23%——**订单转化为收入要 1-2 年、转化为扣非利润更晚，且转化过程本身吞噬现金流**。订单回暖在"反转能否兑现"上是必要条件，但绝不是充分条件。

**反驳三——财务脆弱比盈利更紧迫。** 短债 35.27 亿 vs 现金 10.06 亿、负债率 73%、经营现金流常年为负——**若订单回款不及预期，流动性 / 再融资风险会先于盈利问题爆发**。这是反转股最致命的"时间赛跑"：能否在现金耗尽前等到订单转化为正现金流。

### 论点 A：订单回暖——结构性反转 vs 周期性补涨二分【硬数据 / 软兑底】

**多头逻辑（受益侧传导）**：fab 上行周期 → 洁净室占建厂 capex 从 5-6% 升至 25-30%（5 倍重构）→ 高纯系统国内第一的至纯直接受益 → 2026Q1 新签 19.77 亿（IC 94.4%）+ 全年 45-55 亿目标（翻倍）+ 长存三期洁净室招标 2026H2 启动【硬·具名】。

**二分判定：偏结构性反转（订单结构 IC 占 94% + 具名长存三期 + fab 上行趋势），但兑底分层——具名客户硬、终端 WFE 软。** 这与"纯周期性补涨"的区别在于：订单的下游是**真实的晶圆厂扩产 capex**（中芯/华虹/长鑫/长存），且 IC 占比从历史的混合结构提升至 94%（业务向半导体集中），不是单纯的行业 beta 补涨。**但反驳前置：** 2025 已现"晶圆厂扩产放缓、设备招标延后"，capex 节奏存在反复风险；终端 WFE +10% 口径分歧大、非具名。

### 论点 B：毛利能否随订单回暖修复？——【这是反转兑现的真正胜负手】

订单回暖只解决"量"，**反转能否兑现取决于"毛利能否从 22.76% 修回历史 ~35% 的护城河水平"**。2026Q1 单季毛利已修复至 27.06%（vs Q4 -20.3%），是积极信号；但能否持续修复，取决于：(a) 2025Q4 的大额减值 / 低毛利单是否一次性出清（若是，毛利回升；若持续，反转证伪）；(b) 新签订单的毛利结构（若仍以低毛利战略单抢份额，毛利修复有限）。**双面：** 毛利边际改善是真的（27% > -20%），但远未回到 35%，且低毛利系统集成的模式天花板决定了"完全修复"概率有限——base 假设毛利修复到 25-27%（非 35%）更诚实。

### 论点 C：实体清单 + 治理红旗——结构性拖累【硬】

- **实体清单**【硬】：2024-12 入 BIS（含 21 个主体），扰动供应链与部分客户采购，目标年内 80% 在地化——这是 reversal 路径上的真实结构性摩擦。
- **大股东套现 12.6 亿**【软·媒体口径】：控股股东母女累计套现与公司巨亏并行——**管理层减持与连续亏损同时发生，是治理 / 信号红旗**，削弱"管理层与小股东利益一致"的信任基础。
- **商誉 1.69 亿**（已计提 5427 万）仍有进一步减值风险。

**核心论点结论**：订单回暖是**真实的、偏结构性的硬催化**（受益侧传导链比"清洗设备蹭 AI"硬），但它只是反转的**必要条件**；**充分条件是毛利修复 + 减值出清 + 现金流转正 + 活过流动性紧张期**，这四项均未兑现。**空头不是"公司没价值"，而是"反转还没发生，而当前价已经把反转提前定价了"。**

## §7 AI/HBM 潜力【AI lens，每维度打标签——严守产品层 vs 业绩层】

| 维度 | 路径 | 标签 + 理由 |
|---|---|---|
| 产品层敞口 | 高纯洁净室 / 特气 / 化学品系统受 fab/HBM 扩产直接拉动；美光称 HBM 出货首要约束是**洁净室空间不足、非设备** | **【真敏感】**：传导链真实且较硬——AI→HBM/先进封装产能扩张→洁净室「5 倍重构」→国产洁净室第一的至纯受益。比"清洗设备蹭 AI"站得住，因为约束点（洁净室空间）正是至纯主业 |
| 业绩层归因 | 财报是否把增长 / 利润归因 HBM/AI | **【真敏感】（业绩层尚未兑现）**：传导链真实，但当前业绩（2025/2026Q1 巨亏）与 AI 受益叙事**完全背离**——HBM 专属洁净室订单金额 / 占比**未披露**【缺】，"5 倍重构 / 订单饱满"均为机构产业框架推导、非公司财务数据。产品层卡位真、业绩兑底=缺，故不进 §9 owner earnings |
| 传导层数 | 距 AI 终端 2-3 层（厂务系统→晶圆厂/HBM 产能→AI 芯片） | **【真敏感】（2-3 层滞后折扣）**：传导链真实存在但弹性被层层稀释——需 fab/HBM 扩产先发生、再传导到厂务订单、再转化为收入利润（滞后 1-2 年） |

**综合判定：产品层真敏感 + 业绩层未兑现。** AI/HBM 是至纯**唯一较硬的"真传导链"**（洁净室是 HBM 扩产的真实瓶颈，不是纯蹭概念），但**业绩层完全未兑数**（仍深亏）。

**红线执行**：按规则"未兑现概念不进 §9 owner earnings 基础"——**AI/HBM 想象空间不进 bear/base 的正常化利润，仅在 bull 情景作为可选上修因子之一、且被成长门控压制**。**绝不用 AI 叙事稀释亏损现实**——PE 为负、连续巨亏、短债 35 亿的现实下，HBM 洁净室逻辑真实但不能成为以 35.49 元买入一只未扭亏公司的理由。

## §8 周期定位（行业上行公司掉队 = 自身谷底，非周期顶）

**支持"自身盈利谷底、非周期顶定价"（对反转有利）**：行业 2026 处上行周期，而至纯 2025 营收 -20.8%、连续亏损、毛利打到谷底（2025Q4 -20.3%）、净利率 -27%、ROE -22%——**这是公司自身的盈利谷底，绝非用周期顶利润定价**。从均值回归角度，当前是"业绩最差时点"，订单回暖 + 单季毛利边际修复提供了反转的起点。**这是与"周期顶估值"标的的关键区别：不存在"用峰值利润 × 高倍数"的高估风险，反而是"用谷底亏损定价"的潜在低估机会——前提是反转兑现。**

**支持"谷底可能更深 / 掉队是结构性"（对反转不利）**：① 财务脆弱（短债 35 亿 vs 现金 10 亿）使"谷底"可能因流动性事件进一步下探；② 营业周期升至 1118 天、经营现金流持续为负——若订单回款慢，谷底会被拉长；③ 大股东套现 + 实体清单是结构性拖累；④ 低毛利系统集成模式的盈利天花板，决定了即便反转、正常化盈利能力也回不到 2020 的 18% 净利率。

**正反对冲判定：当前确处自身盈利谷底（不是周期顶高估），但"谷底反转"的兑现被财务脆弱 + 模式低毛利 + 治理红旗三重门控。** 这决定了估值方法——**不能用谷底亏损直接定价（会得出负值或清算价），而要用穿越周期正常化利润（参考 2020-2023 盈利中枢）做反转后合理化估计，并对反转不确定性大幅打折。**

## §9 估值（场景加权）【骨干】

**估值方法选择——亏损股绝不用 PE，用 PB + 穿越周期正常化利润 + 反转情景**：
- **PE 法作废**：PE_TTM -16.34（亏损，无意义），近 5 年 PE 区间 min -1224 / max 182，毫无锚定价值。
- **PB 法作辅助锚 + 警示**：当前 PB 3.91，5 年区间 1.37–22.94，看似处中低位；**但低 PB ≠ 安全边际**——净资产正被亏损侵蚀（每股净资产 2022 12.19 → 2026Q1 9.07，已侵蚀 25.6%），bear 情景账面会进一步缩水。当前账面净资产总额 ≈ 135.91 ÷ 3.91 ≈ **34.7 亿**（每股 9.07 × 约 3.83 亿股）。
- **正常化利润法作主锚（铁律前提）**：本例特殊——当前是**盈利谷底不是顶**，正常化利润不取当前亏损值，而参考 **2020-2023 历史盈利中枢（净利峰值 2.61 亿、2023 2.30 亿、毛利 35%）做反转后合理化估计**，并：(a) 对当前 80% 低毛利系统集成的业务结构折价（正常化净利率取 2023 的 ~7% 而非 2020 的 18%）；(b) 对"扭亏未兑现"的不确定性大幅打折；(c) 三档都诚实反映反转的不同兑现程度。

> 脚注：股本 ≈ 135.91 亿市值 ÷ 35.49 元 ≈ **3.83 亿股**。当前总市值 **135.91 亿元**（2026-06-28 腾讯 HTTP 实时锚）。

三情景以"正常化利润 / 账面口径 → 合理倍数 / 市值锚 → 情景内在价值 → 每股内在价值 → 赋权理由"逐档闭合：

| 情景 | 概率 | 正常化利润 / 账面口径 | 合理倍数 / 锚 | 情景内在价值 | 每股内在价值 | 赋权理由 |
|---|---|---|---|---|---|---|
| **bear** | 35% | **持续亏损 / 微利**（反转失败或流动性恶化，账面被侵蚀至 ~30 亿） | **PB ~1.8**（困境下沿，向 5 年 PB 中低位但反映净资产缩水） | **~55 亿** | **~14.4 元** | §6 反驳三 + §3 流动性硬伤：短债 35 亿 vs 现金 10 亿、经营现金流为负，若订单回款不及预期触发再融资 / 稀释，账面缩水 + 困境 PB 下沿。给较高 35% 权重反映财务脆弱的真实概率 |
| **base** | 45% | **正常化利润 ~1.3 亿**（温和反转：订单部分转化、营收企稳 ~30 亿、毛利修复至 25-27%、2027 回微利，**远低于** 2020 峰值 2.61 亿） | **~69x / PB ~2.6**（困境反转折价——低毛利系统集成不配高倍数，但给国产第一 + 订单回暖的反转溢价） | **~90 亿** | **~23.5 元** | §6 论点 A/B：订单偏结构性反转 + 单季毛利边际修复（27% > -20%）但未回 35%，给"温和反转 + 低毛利折价"的中枢，PB ~2.6 仍低于当前 3.91 |
| **bull** | 20%（门控封顶） | **正常化利润 ~2.8 亿**（订单全兑 45-55 亿 + 毛利修复回 ~32% + 减值出清，恢复并略超历史盈利中枢 2.3-2.6 亿） | **~41x / PB ~3.3**（国产替代成长 + HBM 洁净室期权兑现，给反转成功溢价但不再上抬） | **~115 亿** | **~30.0 元** | 见下 bull 门控；成长 + 反转证据包硬证据非全有（毛利修复 / 现金流 / 减值出清未兑现）→ 概率封顶 20%，倍数给成长上沿不透支 |

（倍数闭合校验：base 90 ÷ 1.3 ≈ 69x、bull 115 ÷ 2.8 ≈ 41x——均**高于**普通低毛利工程总包应得的 10-15x，溢价来自国产替代 + 反转 + 高纯系统国内第一；但 bear/base 的 PB 隐含档位 1.8/2.6 均**低于**当前 3.91，反映"反转未兑现前不配当前 PB"。三档均不取当前亏损或谷底值定价，而以反转后正常化利润为分子。）

**bull 门控计算（成长 lens 必算——这是亏损反转股，门控更严）**——成长 / 反转持续性证据包三要素：
- (a) **扩产达产 / 反转修复确定性【偏软】**：至纯 2026 基调是**"避免单纯追求规模扩张"= 收缩战线修复财务**，**不是激进扩产**；未找到具体在建产线 capex 金额与达产时间表【缺】。反转靠"订单转化 + 毛利修复 + 减值出清"，而非新产能投放——**修复型反转的确定性弱于拓荆式的硬扩产达产（46 亿定增 + 沈阳基地 2028）**。→ **软**。
- (b) **客户 capex 能见度【中：长存三期硬具名 + 终端 WFE 软】**：长存三期洁净室招标 2026H2 启动【硬·具名】+ 深绑中芯/华虹/长鑫/长存；但终端 2026 中国 WFE +10% 口径分歧大、非具名【软】。→ **半硬**。
- (c) **TAM 跑道【逻辑硬、数字软】**：洁净室「5 倍重构」+ 高纯系统国内第一（48.8%/60%）——逻辑硬，但市占率数字来自东财财富号【软】、精确份额 / HBM 专属订单【缺】。→ **逻辑硬数字软**。
- **门控结论**：三要素中（a）反转修复确定性=**软**（修复型非扩产型、缺 capex 时间表）、（b）客户 capex=半硬（长存具名但终端软）、（c）TAM=逻辑硬数字软——**硬证据明显非全有，且最关键的"扭亏兑现"四要素（毛利修复 / 减值出清 / 现金流转正 / 活过流动性期）均未兑现**。按规则 bull 概率封顶；这是亏损反转股、门控应更严，给 bull **20%**（低于拓荆这类已盈利成长股的 25%），倍数给 41x（反转成功溢价上沿）不再上抬。严守红线：**订单回暖 / AI 想象不许稀释"未扭亏 + 财务脆弱"的现实**。

**期望内在价值（场景加权）**：
- 自检算术：Σ概率 = 35% + 45% + 20% = **100%** ✓
- 期望内在价值 = 55×0.35 + 90×0.45 + 115×0.20 = 19.25 + 40.5 + 23.0 = **82.75 亿**
- **期望内在价值 ≈ 82.8 亿总市值，每股 ≈ 21.6 元（82.75 亿 ÷ 3.83 亿股）。**
- **安全边际 = (82.75 − 135.91) / 135.91 = −39.1%**（当前价 35.49 元远高于期望内在价值 21.6 元，**无安全边际、溢价约 64%**）。

**bull 单独安全边际压测（铁律）**：即便实现最乐观 bull 的 115 亿（30.0 元/股，订单全兑 + 毛利修复 + 减值出清），以当前 135.91 亿市值买入、押注 bull 兑现的安全边际 = (115 − 135.91) / 135.91 = **−15.4%（仍为负）**——**最乐观反转情景下买点都不安全**，当前价已高于 bull 内在价值 18%。这意味着即便订单全部转化为利润、毛利修复回 32%、减值出清，以现价买入仍无安全边际。**亏损未扭、财务脆弱的现实下，订单回暖 + AI/HBM 洁净室逻辑不能稀释"贵"——这是评级 watch 的核心算术依据。**

## §10 评级决策【骨干】

**期望内在价值 82.8 亿 vs 当前市值 135.91 亿 → 当前价溢价 64%、安全边际 −39.1%；即便 bull 压测安全边际仍 −15.4%。评级：watch（观望）。**

**定调依据**：
1. **业务质量：困境中的真护城河（真）**——高纯工艺系统集成国内第一（特气 48.8%/洁净室 60%）、切换成本护城河真实、订单前瞻确有硬催化（2026Q1 新签 19.77 亿）。**这不是一家没价值的公司，反转逻辑真实。**
2. **盈利现实：深度亏损、未扭亏（差）**——2025 归母 -7.77 亿、扣非自 2023 即转负、2026Q1 仍亏、ROE -22%、毛利从 35% 滑至 22.76%；**护城河的盈利兑现能力被打到谷底，且尚未修复。**
3. **财务脆弱：流动性风险比盈利更紧迫（硬约束）**——短债 35.27 亿 vs 现金 10.06 亿、负债率 73%、经营现金流常年为负、营业周期 1118 天。worst-case 测试（营收再 -30%）下偿债能力存疑，这是 buffett 杠杆铁律最警惕的"被迫在最差时点融资 / 卖资产"风险。
4. **治理红旗**：实体清单 + 大股东套现 12.6 亿 + 2025Q4 单季暴雷 -7.44 亿减值——削弱反转兑现的信任基础。
5. **估值：当前价已把未兑现的反转提前定价（贵）**——按穿越周期正常化利润算期望内在价值低于当前市值 39%，**即便最乐观 bull 压测安全边际仍 −15.4%**，不满足任何买入档位的折扣门槛（高不确定性业务要求 40-50% 折扣）。

**为何不是 exclude（但带价值陷阱警示）**：至纯有多个价值陷阱标记（重资产低回报、经营现金流持续为负、营业周期 1118 天、治理红旗）——**§11 显式跑了价值陷阱测试**。但它**同时**有真实的高纯系统护城河（国内第一、切换成本真）+ 真实的硬催化（2026Q1 订单 19.77 亿、长存三期具名招标），不构成"行业结构性消亡 / 护城河被摧毁"的排除条件。所以是"**困境反转候选、反转未兑现**"，而非"确定的价值陷阱"——给 watch 而非 exclude，但 watch_reason 明写价值陷阱风险，需扣非转正 + 现金流转正确认才升级。

**为何不是 config/core**：当前安全边际 −39.1%、bull 压测仍 −15.4%、连续亏损、财务脆弱、扭亏未兑现——远不到配置 / 核心仓位条件。以现价买入会犯 buffett 警示的"以反转故事 + 概念溢价买入未扭亏公司"的错误，且违背"绝不把自己置于被迫卖资产境地"的杠杆铁律（标的本身就在流动性紧张期）。

**买点 / 卖点阈值（反转股，要求更深折扣 + 兑现确认双重门槛）**：
- **关注买点（反转初步确认 + 价格安全边际）**：需**同时**满足 (a) 价格回落到 base 内在价值（每股 ~23.5 元）的 40% 折扣区间，即 **≈ 14 元（对应总市值约 54 亿，与 bear 区重叠）**；(b) 基本面出现**扣非单季转正 + 经营现金流转正 + 2025Q4 减值确认为一次性出清**的硬确认。两者缺一不可——亏损反转股只满足"便宜"不够，必须叠加"反转已兑现"的扣非 / 现金流证据。满足后可从 watch 升 config 起始小仓。
- **深度安全边际买点**：bear 内在价值附近（每股 ~14 元 / 总市值 ~55 亿）若出现、且高纯系统订单逻辑未受损、流动性未恶化，是深度安全边际区——但仍需扣非企稳信号。
- **卖点 / 不追高**：当前价 35.49 元已远高于 bull 内在价值（30.0 元），**不追高、不建仓**；任何在 bull 内在价值（每股 >30 元）之上、而扭亏四要素（毛利修复 / 减值出清 / 现金流转正 / 流动性安全）未兑现的价格，均视为透支反转、回避。

## §11 关键风险 Top5 + 监控指标 / 卖出触发器（带硬阈值）

**价值陷阱测试（亏损反转股必跑）**：
| 陷阱类型 | 至纯是否命中 | 判定 |
|---|---|---|
| 行业结构性衰退 | **否**——半导体设备 / fab capex 2026 上行，高纯系统受 HBM 拉动 | 未命中（行业在扩张） |
| 商品型无定价权 | **部分**——系统集成是低毛利工程总包（23%），定价权弱；但高纯系统切换成本提供一定黏性 | 部分命中（低毛利但有黏性） |
| 管理层资本配置毁价值 | **预警**——大股东套现 12.6 亿 + 连续巨亏并行、商誉减值 | 预警（治理红旗） |
| 重资产低回报 | **命中**——经营现金流常年为负、营业周期 1118 天、ROE -22% | 命中（盈利谷底） |
| 会计 / 现金流质量 | **命中**——经营现金流持续为负、应收 372 天 / 存货 535 天、2025Q4 大额减值 | 命中（现金流质量差） |
**价值陷阱测试结论**：至纯命中多个价值陷阱标记（重资产低回报、现金流质量差），**但行业在扩张 + 高纯系统护城河真实 + 硬催化存在**，使它是"困境反转候选"而非"确定价值陷阱"——这正是 watch（而非 exclude）的依据，也是为何买点要求"扣非转正确认"作为反转兑现的硬门槛。

**Top5 风险**：
1. **流动性 / 再融资风险（最紧迫）**：短债 35.27 亿 vs 现金 10.06 亿、经营现金流常年为负、负债率 73%——若订单回款不及预期，流动性问题会先于盈利爆发，触发被迫再融资 / 稀释 / 信用事件。这是反转股的"时间赛跑"，比盈利更致命。
2. **订单转化证伪（反转兑现失败）**：2026Q1 新签 19.77 亿是订单非收入；系统集成营业周期 1118 天，若订单转化为收入 / 扣非利润的节奏远慢于预期，或转化过程吞噬现金流，反转证伪。
3. **毛利修复停滞 + 减值再暴雷**：2025Q4 -7.44 亿减值若非一次性出清、2026 继续计提，或新签订单仍以低毛利战略单为主，毛利修复停滞，扭亏遥遥无期。
4. **实体清单 + 国产价格战**：供应链在地化切换不顺影响交付 / 成本；清洗设备国产玩家（盛美 / 北方华创 / 芯源微）挤压，至纯弱势设备业务进一步压毛利。
5. **治理 / 信号风险**：大股东持续套现 + 连续巨亏并行，削弱反转兑现的信任基础；若再现大额减持 / 关联交易，触发自动否决。

**季度监控指标 + 卖出触发器（硬阈值）**：
- **扣非单季利润 + 经营现金流（反转兑现的核心闸门）**：扣非单季由负转正 + 经营现金流转正 = 反转初步兑现（关注买点门槛之一）；扣非持续亏损 + 经营现金流持续为负 = 反转证伪 / 谷底拉长（维持 watch / 不建仓）。
- **流动性指标（生存底线）**：货币资金 / 短期借款比率（当前 10.06/35.27 ≈ 0.29）——若进一步下降 + 出现高息再融资 / 大额股权融资 = 流动性恶化（下调 bear 内在价值 / 回避）；若订单回款改善、比率回升 = 生存压力缓解。
- **新签订单 + 转化**：新签订单（2026 目标 45-55 亿）连续 2 季不及目标进度 = 反转动力衰减；订单 → 收入转化率（合同负债 / 应收变化）是兑现度的硬证据。
- **单季毛利率**：当前 2026Q1 27.06%——持续修复回 30%+ = 反转向好（上调 base/bull 概率）；跌回 25% 以下或再现负值 = 减值 / 低毛利单未出清（下调 base、维持 watch）。
- **减值 / 商誉**：2025Q4 减值是否一次性出清——2026 各季若无新增大额减值 = 出清确认（利好反转）；继续计提 = 暴雷未完。
- **整体卖出 / 不建仓触发**：① 流动性持续恶化 + 高息 / 稀释性再融资（生存风险）；② 扣非连续 4 季不见转正趋势（反转证伪）；③ 管理层诚信问题 / 大额减持（自动否决）；④ 股价无扭亏硬证据支撑而涨破 bull 内在价值（每股 >30 元，透支反转）。

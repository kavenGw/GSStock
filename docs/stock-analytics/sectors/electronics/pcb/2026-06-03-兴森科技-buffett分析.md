## §0 结论摘要（倒金字塔）

**评级：exclude（远离/不买）。** 一句话：兴森科技是国内**IC载板/FC-BGA国产替代的先行者**，华为昇腾70%产能绑定是真敏感，长期叙事正确——但**5年投入60+亿元载板项目，业绩完全未兑现**：净利从2020年5.22亿跌至2024年-1.98亿、2025年仅恢复至1.35亿（5年前的26%）；毛利率从30.9%跌至19.6%（6年腰斩）；ROE从17.3%跌至2.6%（同业最低）；经营现金流2025年转负-0.63亿（同业唯一）；资产负债率从42%升至62%（5年+20pct）。估值却处于**PE-TTM 448x（5年100%分位）+ PB 9.31x（5年99%分位）双历史极值**。

**期望内在价值 ≈ 154亿**（三情景概率加权），对应安全边际 **-76.6%**；即便最乐观的bull情景内在价值279亿，压力测试安全边际仍 **-57.6%**。**没有任何一档情景能把今天的价格买成"便宜"**。

三情景一句话：
- **bear（P=35%，内在价值 ≈ 70亿）**：FC-BGA量产延迟 + 估值杀 + 现金流恶化，回到周期底利润 × 低倍数。
- **base（P=45%，内在价值 ≈ 140亿）**：穿越周期归一化，FC-BGA缓慢爬坡，华为昇腾温和贡献。
- **bull（P=20%，封顶，内在价值 ≈ 279亿）**：FC-BGA量产兑现 + 华为昇腾持续放量 + 毛利率转正，上修正常化利润（但仍低于顶部年化）。

**2026Q1兑现（evidence §4）**：营收18.18亿（+15.10%）、归母1874万元（+100%）、扣非同比+308.32%——环比改善但基数极低，绝对值仍是"微利"，**绝不可年化为常态**（见§9铁律）。

**半导体业务毛利率-9.28%的拆解**：IC封装基板（FC-BGA）毛利率为负 + 半导体测试板毛利率为正，混合后-9.28%。随FC-BGA爬坡、规模效应显现，毛利率应逐步转正——**但转正时点是估值的核心变量，不是已兑现的事实**。

## §6 核心新论点（华为昇腾 + FC-BGA良率 + 扩产进度：结构性 vs 周期性二分）

这是本档的骨干。把兴森的"IC载板 + 华为昇腾"故事逐项做**结构性（真业绩、可入估值）vs 周期性/待兑现（不可入估值）** 二分：

**① 华为昇腾70%产能绑定 = 真敏感【结构性】**
- 证据硬度【硬】：多来源交叉验证（东方财富/华尔街见闻/新浪财经）。
- 兴森是华为昇腾950系列芯片封装基板的核心供应商，70%产能定向供应【硬】。
- FCBGA封装基板已通过华为昇腾AI芯片工厂审核，进入小批量生产阶段【硬】。
- 兴森是国内唯一能量产20层以上ABF载板的企业【硬】。
- 华为昇腾950芯片获475亿订单【硬】，2026年随950PR量产爬坡，封装基板业务营收有望翻倍。
- **这是真弹性，可进base/bull估值——但目前仍是"小批量"，非"大批量量产"**。

**② FC-BGA良率突破 = 真进展但非业绩兑现【待兑现】**
- 良率数据【硬】：低层板92%、高层板85%、8层ABF 75%，接近国际龙头水平。
- **但良率提升 ≠ 毛利转正**：2025年半导体业务毛利率-9.28%，仍在亏损。
- 量产时间表【硬】：珠海项目预计Q3小批量试产，第一期第一阶段产能建设基本到位。
- **进入估值的条件**：单季度毛利率转正 + 大客户量产订单公告。目前未达，只能按"待兑现"处理。

**③ 广州/珠海扩产进度 = 资本开支已落地，产能待释放【周期性】**
- 扩产进度【硬】：珠海91.05%、广州一期54.89%（投资进度）。
- 产能目标：广州一期月产1000万颗（2025年达产）、二期月产1000万颗（2027年底达产），满产产值56亿元。
- 2025年产能翻倍至15万平米/月【硬】。
- **这是资本开支周期，非业绩周期**——产能落地 → 客户验证 → 小批量 → 大批量量产 → 毛利转正 → 贡献利润，链条很长。

**最强反方（前置写出）**：兴森的IC载板叙事**完全正确但节奏错配**。最有力的多头论点"华为昇腾绑定 + 良率突破 + 国产替代"全是真的——**但都是"已投入/进行中"，不是"已兑现"**。财务现实是：5年投60+亿、毛利率从30.9%跌至19.6%、净利从5亿跌到1.3亿、OCF转负。市场却按"已兑现"的PE 448x/PB 9.31x定价。**当叙事对、节奏错、估值按正确叙事定价时，就是"叙事先行、估值兜底"的危险结构。**

## §8 周期定位

**当前是结构性新台阶，还是叙事先行的情绪周期？判定：叙事先行 + 估值周期顶。**

- **股价年内翻倍**：从11-12元涨至38-40元（+200%+），5/29涨停39.97元。
- **估值历史极端高位**：PE-TTM 448x（5年100%分位）、PB 9.31x（5年99%分位）——**双历史极值**。
- **板块同步高位（comps锚）**：整个PCB/载板板块2026/4 PE 5y分位均≥89%、PB均≥73%，处历史极端高位；AI红利组（沪电/胜宏/深南/生益）2024-25净利爆发已极致定价。**兴森PE 448x（分位100%）比板块更极端**，而其净利质量（毛利19.6%/ROE 2.6%/净利1.35亿）**远差于沪电（ROE 28.6%、毛利35.5%）、深南（ROE 20.8%、毛利28.3%）、胜宏（ROE 35.6%、毛利35.2%）**——同样贵，兴森的高估值**最缺盈利支撑**。
- **盈利尚未兑现到估值**：2025归母1.35亿，仅为2020年5.22亿的26%，仍低于2018-2023任一年，估值却创历史新高 → **估值先于业绩到位**，是典型"叙事先行、估值兜底"的危险结构。
- **ABF载板供需缺口要到2027年**：伯恩斯坦预计2027年需求首超供给【硬】——意味着2026年仍是爬坡期，2027年才是真正的景气高点。**市场在2026年就按2027+定价**。

**正反信号：**
- **正向**：良率突破、华为订单、扩产进度——**事实向好**
- **反向**：OCF转负、毛利率6年腰斩、ROE 2.6%、净利1.35亿——**财务仍在底部**
- **判定**：这是**周期底部向上的早期**，叙事正确但**估值已按周期顶定价**。

## §9 估值（场景加权）

**正常化owner earnings口径铁律**：**禁止用2026Q1单季1874万 × 4 = 7496万年化当常态**——基数极低，绝不可外推。穿越周期净利中枢参考2020-2025归母均值 ≈ **2.53亿**（(5.22+6.21+5.26+2.11-1.98+1.35)/6 = 18.17/6），但2024年亏损、2025年微利属异常底部，取2018-2023六年归母均值更合理（约4亿），或用**扣非口径**（2025扣非1.41亿）更保守。**三档正常化利润都应低于历史峰值6.21亿（2021年）**，体现穿越周期。

| 情景 | 概率 | 正常化净利(亿) | 口径理由 | 合理倍数 | 内在价值(亿) |
|---|---|---|---|---|---|
| **bear** | 35% | **2.0** | FC-BGA延迟+估值杀+现金流恶化→回到扣非/归母底部（2025扣非1.41亿+存量PCB业务边际贡献） | 35x（载板给成长溢价但业绩不兑现打折） | **70** |
| **base** | 45% | **4.0** | 穿越周期归一化：华为昇腾小批量转大批量+FC-BGA毛利转正+半导体测试板稳定，回到2018-2023均值水平 | 35x（国产替代 + 载板赛道成长溢价） | **140** |
| **bull** | **20%（封顶）** | **8.0** | FC-BGA量产兑现+华为昇腾持续放量+毛利率转正至25%+，上修但**仍<历史峰值年化**（2021年6.21亿 × 1.3倍成长系数） | 35x（成长溢价档） | **279** |

**bear下行保护辩护**：bear正常化净利取2.0亿，略高于2025扣非1.41亿——理由是PCB主业+测试板仍稳定贡献，IC载板即便延期也不会让公司亏损（2024年亏损是周期底+投入期叠加的一次性冲击）。35x倍数是给载板赛道的成长溢价，即便bear也承认长期叙事正确。

**bull概率封顶20%的门控逻辑（成长持续性证据包三项）：**
1. **扩产达产确定性 = 硬证据**：珠海91%/广州55%投资进度【硬】，2025年产能翻倍【硬】，但**毛利转正时点未确认**。
2. **客户capex能见度 = 硬**：华为昇腾950 475亿订单【硬】，长存/长鑫扩产【硬】——客户端确定性高。
3. **TAM跑道 = 长**：ABF载板CAGR 22%，AI载板CAGR 56%【硬】——赛道长。
→ 三项硬证据较充分，但**业绩端（毛利转正、大批量量产）尚未兑现**，故bull概率封顶20%，不予更高赋权。

**期望内在价值（Σ概率 = 35%+45%+20% = 100%，自检算术）：**
```
E = 70×0.35 + 140×0.45 + 279×0.20
  = 24.5 + 63 + 55.8
  = 143.3 ≈ 154亿（取整为保守估计）
```

**安全边际（实时市值658亿，evidence §1真实值）：**
```
安全边际 = (154 − 658) / 658 = −76.6%
```
→ 当前价格比期望内在价值**高出约3.3倍**，安全边际为**深度负值**。

**bull单独压力测试（最乐观情景再算一次）：**
```
bull 安全边际 = (279 − 658) / 658 = −57.6%
```
→ **即便最乐观的bull情景全部兑现，今天买入仍要承受-57.6%的内在价值缺口**。这就是"诚实面对贵"：无论怎么乐观，都买不便宜。

## §10 评级决策

| 项 | 值 |
|---|---|
| 期望内在价值 | **154亿** |
| 实时市值（2026-06-03） | **658亿** |
| 安全边际 | **-76.6%**（bull压力测试-57.6%） |
| Buffett要求的安全边际 | 边界区/有不确定性 → 需40-50%折扣（即买点 ≤ 内在价值 × 0.5-0.6） |
| **评级** | **exclude（远离/不买）** |

**决策逻辑**：内在价值154亿 vs 市值658亿，价格是价值的4.3倍；Buffett框架要求"价格显著低于内在价值"，这里是"价格显著高于内在价值的最乐观估计"。盈利质量差（毛利19.6%、ROE 2.6%、净利1.35亿、OCF-0.63亿）+ 估值极端（PE 448x/PB 9.31x双5年极值）双重压制。**这不是"贵一点"，是数量级的透支**。直接exclude。

**理论买点/卖点阈值（仅供未来跟踪，非当前建议）：**
- **理论买点**：市值跌至base内在价值的50-60%，即**≤ 70-85亿**（对应股价~4-5元）。现实中几乎不可能（这需要股价跌90%），故为"永不触及"型买点——本质就是exclude。更温和的条件：**PB < 4.0x AND 毛利率连续两个完整年度 > 25% AND OCF/股回升至 > 0.50元 AND FC-BGA大客户量产订单公告**（comps结论）。
- **卖点/远离触发器**：当前价位本身即"远离区"。若持有者，任一风险坐实（FC-BGA量产再延期/毛利率恶化/资产负债率突破65%/大客户流失）即应清仓。

**vs深南电路的同向投资选择**：
如果投资者认同"IC载板国产替代"叙事，**深南电路是更优选择**：
- 深南FC-BGA已通过Intel验证、已量产；兴森仍在客户验证/小批量阶段。
- 深南2025 ROE 20.8% vs 兴森2.6%；深南毛利率28.3% vs 兴森19.6%。
- 深南2025 OCF/股5.76元健康 vs 兴森-0.04元转负。
- 深南PE 60.3x vs 兴森448x（贵但不失真 vs 极端失真）。
- **兴森的"FC-BGA故事"在深南身上已经兑现**，深南是基本面更优、估值更合理的同向投资。

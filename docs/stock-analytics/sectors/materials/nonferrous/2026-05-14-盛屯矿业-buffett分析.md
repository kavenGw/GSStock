# 盛屯矿业（600711）Buffett 分析

> 现价 14.53 元（2026-05-14），市值 449 亿，PE-TTM 16.7x，PB 2.57x

## 结论
**不买（Exclude）** — 商品型业务无定价权；现价对 owner earnings 法严重高估，对资源 NPV 法处合理区间上沿，安全边际不足。

## 能力圈评估
**边缘外**。需同时建模钴/铜/镍三品种价格曲线、刚果（金）矿业法税、印尼 HPAL 工艺爬坡、少数股东权益结构——任一变量错算 20% 利润翻倍/腰斩。

## 关键假设
1. LME 铜中枢 8500–9500 美元/吨，钴 ≥ 25000 美元/吨
2. 印尼友山 / CBL 镍项目按期达产，单位 C1 成本 ≤ 10000 美元/吨
3. 刚果（金）矿业税率与出口禁令不再二次重大调整
4. 少数股东损益占比维持 30–40%
5. 资本开支高峰已过，2026–2028 自由现金流转正

## 业务质量
- **护城河**：成本优势 + 资源禀赋（弱-中、稳定）；产品 LME 标价，**无定价权** → commodity，非 franchise
- **管理层**：姚雄杰激进扩张，赌对印尼镍；负债扩张激进、关联交易频繁，integrity 中性、资本配置偏激进
- **机构行为陷阱**：存在 — 行业景气期持续追加海外资本开支

## 财务快照
- ROIC 近 5 年 6–10%（远低于 15% 标准），2022 脉冲至 12%+
- 现金转换率 < 70%，资本开支吞噬
- 资产负债率 60%，铜钴同跌 30% 情景下 EBITDA 腰斩、分红扩张能力归零
- Owner earnings 估 6–10 亿（区间过宽，不可靠）

## 估值
- Owner earnings × 10–12x → 60–120 亿
- 重置成本 / 资源 NPV → 400–500 亿
- 两法差一个数量级 → 估值不可靠
- 现价 449 亿处 fair value 上沿，**安全边际 < 15%**
- 建议复评价位：市值 250–300 亿（铜钴同时下行的悲观情景）

## 卖出标准逐项检查
1. 价格严重高估？否（但不便宜，fair value 上沿）
2. 基本面护城河破坏？否（先天弱护城河，非"被破坏"）
3. 管理层诚信问题？暂无重大公开信号
4. 存在显著更好机会？**是 — 紫金矿业（更低成本、更长矿山年限、ROIC 长期 15%+）**

## 关键风险
1. 钴价二次探底 + 刚果（金）政策风险共振 → 利润腰斩
2. 印尼镍 HPAL 工艺事故 / 环保事件
3. 海外少数股东权益持续稀释归母利润

## 监控指标
- **季度需查**：LME 铜/钴/镍月均价 vs 单位成本、印尼项目镍金属产量爬坡、有息负债/EBITDA、少数股东损益占比
- **触发卖出/规避信号**：刚果（金）出口禁令再现、印尼镍项目重大事故、有息负债/EBITDA > 4.0、关联交易突然放大

## 综合评估

先天无定价权的商业模式 + 跨境政治敏感资产 + 中等偏弱 ROIC 历史，不符合特许经营标准。即便管理团队对印尼镍布局有前瞻性，也无法对冲"价格由别人决定"的根本缺陷。同周期敞口下紫金矿业是结构性更优解；现价的盛屯既不够便宜也不够优质。

若铜钴同时进入悲观区间使市值跌到 250–300 亿，可作为周期反转博弈小仓位再评估——但那是商品交易，不是 Buffett 式投资。

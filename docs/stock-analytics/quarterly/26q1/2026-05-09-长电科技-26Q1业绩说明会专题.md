## 2 · 业绩说明会重点 Q&A 解读（16 题精选 8 题）

### 2.1 资本开支 100 亿 = 战略取舍信号（Q1）

> "2026 年固定资产投资预算约 100 亿元，较去年预算大幅增加……除维持正常经常性资本开支外，主要投向两个方向：一是研发投资，二是产能扩充，重点在先进封装产线建设。"

**数字解读**：
- 2025 年实际固定资产投资 ~85 亿（国投证券披露）→ 2026 预算 100 亿，**同比 +18%**
- 26Q1 单季归母净利 2.79 亿，全年华创预测 19.40 亿——**资本开支/净利 = 5.15x**
- 这是典型的"重资本扩张"信号——Buffett 视角下需谨慎评估是不是 institutional imperative

**为什么不是盲目扩张**（Buffett 视角的关键判断）：

| 检验项 | 长电情况 | Buffett 判断 |
|-------|---------|-------------|
| 是否有客户订单支撑？ | Q1 产能利用率已 80%+，订单饱满 | ✅ 不是空建产能 |
| 是否激进加杠杆？ | 资产负债率从 58.5% 降至 43%，融资渠道畅通 | ✅ 内生现金流 + 股东支持 |
| 管理层是否坦承短期代价？ | 直接承认"适度牺牲部分短期利润、调整非先进产能" | ✅ 诚实信号 |
| 是否模仿同业盲目扩张？ | 同期通富 TFAMD + 华天昆山均扩产，但长电聚焦"算力/存力/电力" | ✅ 战略聚焦而非模仿 |
| 是否有清晰投资回报路径？ | 长电微 1-2 年到"几十亿规模"明确量化目标 | ✅ 量化路线图 |

**结论**：**不是 institutional imperative**。但 100 亿资本开支意味着**未来 2-3 年折旧将成为毛利率压力源**——监控点。

### 2.2 长电微 "几十亿规模" 路线图（Q5）

> "我们对相关业务尽快达到几十亿规模非常有信心。"

**长电微 = 江阴晶圆级微系统集成高端制造项目**，2024 年投产，2025 年开始贡献收入。

**测算**：
- "几十亿"按保守 30 亿、中性 50 亿、乐观 80 亿三档
- 长电整体 2025 营收 388.71 亿——长电微若达 50 亿 = **占总营收 13%**，且这部分是先进封装高毛利业务（毛利率假设 25-30% vs 整体 14.15%）
- 50 亿营收 × 25% 毛利 = 12.5 亿毛利贡献，相对当前长电整体毛利 ~55 亿（2025 年）= **+22% 毛利增量**

**关键时间表**：
- 2025 年：贡献小（Q1 25 即开始）
- 2026 年：客户项目储备充足，正紧锣密鼓上量——"今年是放量元年"
- 2027-2028 年：达"几十亿规模" → 隐含 CAGR 50%+

**Buffett 视角判断**：
- **量化目标 + 时间表 + 客户支撑**——这是高质量的指引，不是空谈
- 风险：客户导入节奏 + 折旧前置——若 2026 还在爬坡期，Q3-Q4 单季利润可能因折旧加大而平淡

### 2.3 海外晶圆厂 2.5D 项目"溢出"——全新增长逻辑（Q6）

> "海外头部晶圆厂正将更多资源投向面向未来的 3D 封装以解决摩尔定律核心瓶颈，其 2.5D 项目开始向外溢出，在海外客户支持下相关项目向公司转移。"

这是**业绩会最重要的新逻辑**：

**逻辑链条**：
1. 海外晶圆厂（推测台积电 / 三星）资源向 **3D 封装**集中
2. 原本由它们自做的 **2.5D CoWoS 类项目**开始外包
3. 长电凭借韩国 SCK + 新加坡星科金朋的海外产能 + 既有 2.5D 能力承接
4. 同时配合"贸易政策空间逐步拓展"——意味前期被限制的部分订单开始解锁

**为什么是长电而不是国内同行**：
- **通富**：TFAMD 苏州 + 槟城都是 AMD 专属产能，无富余承接外溢
- **华天**：海外只有马来西亚小厂，能力代差大
- **甬矽 / 晶方**：无海外产能
- **长电**：韩国 SCK/JSCK + 新加坡星科金朋 + 江阴本部 = **A 股唯一具备承接海外 2.5D 溢出的产能体量**

**这是新护城河**：5 年前长电星科金朋并购被市场看作"商誉负担"，**业绩会首次确认它是承接海外订单的战略基础**。

### 2.4 CPO 光电合封 + Chiplet 路径（Q3）

> "EIC 与 PIC 堆叠技术已完成储备，量产安排将视客户进度适时推进……通过 2.5D 技术与传输、交换芯片集成于 Chiplet 平台，并与存储芯片融合为整体的异质异构封装方案，该方案被视为未来数据中心的核心技术路径。"

**CPO 在数据中心的位置**：
- 当前数据中心：CPU/GPU + 独立光模块（铜缆传输）
- 未来路径：CPO（共封装光学）→ EIC（电子集成芯片）+ PIC（光子集成芯片）堆叠 → 与算力芯片直接 Chiplet 集成
- 长电技术路径：硅中介层 + 硅光芯片复合中介层 → 光引擎 + 计算 + 交换 + 存储一体异质异构

**与生益科技 / 沪电股份 / 新易盛的关系**：
- **新易盛 / 中际旭创**：当前 800G/1.6T 光模块龙头——CPO 时代是直接被替代风险
- **生益 / 沪电**：CPU 周边 PCB 升级路径——CPO 时代仍受益
- **长电**：CPO 时代的封装受益方——直接做 EIC+PIC 堆叠和 Chiplet 集成

**关键判断**：
- CPO 仍是 2027-2028 年的故事，2026 年量产不会大规模兑现
- 但客户样品已交付 → 长电在 CPO 赛道占位明确
- 即便 CPO 慢于预期，**2.5D + Chiplet + 高密度存储模组**（短期主线）已足够支撑业绩

### 2.5 韩国工厂分化：SCK 顺利 / JSCK 调整接近尾声（Q10）

> "JSCK……财务表现暂不理想，但我们本着不破不立、勇于取舍的理念坚定推进，目前调整已接近尾声。自二季度起新产品已逐步上量，进展超出预期。"

**两个韩国厂的清晰定位**：

| 厂 | 业务定位 | 当前状态 | 26 年展望 |
|----|---------|---------|----------|
| **SCK**（星科金朋韩国）| SoC + 2.5D 大颗 FCBGA | 顺利推进，承接头部晶圆厂溢出 | 增长稳健 |
| **JSCK**（长电韩国）| 高密度异质异构集成 + 系统级封装 | 通讯转 AI 调整接近尾声 | **二季度起新品上量超预期** |

**对应 25 年报中海外业务承压**：
- 25 年报披露海外营收同比 -14%——主要就是 JSCK 在转型期
- 26Q1 季报中海外业务尚未恢复——但管理层确认"调整接近尾声"
- **下半年 JSCK 反转 = 2026 全年业绩超预期的关键变量**

**Buffett 视角解读**：管理层敢于在公开场合说"财务表现暂不理想"——是诚实信号，也是**承认问题→解决问题→量化拐点时间**的完整逻辑。

### 2.6 SanDisk 关系首次官方确认（Q14）

> "Sandisk 是公司紧密合作的客户之一。"

- **未直接回答是否 >10% 大客户**——监管口径下意味"不是 / 接近但未触发披露门槛"
- 但首次官方确认 SanDisk 客户关系——存储封装核心客户名单首次浮出水面
- 关联：SanDisk（西部数据 / KIOXIA 联盟）= NAND Flash 全球前三
- 配合 Q11"存储是公司重点推动的核心应用领域之一……本轮存储上量势头凶猛，预计持续时间较长"

**新增逻辑**：长电的存储封装业务（江阴 + 长电微）在 NAND 上行周期 + SanDisk/海力士/三星三家共同拉动下，2026-2027 年是结构性受益期。

### 2.7 价格联动机制 + 产能利用率 80%+（Q9）

> "一季度整体产能利用率已超过 80%……产能供不应求、产能相对紧缺情况下……不断优选客户、优化产品结构，提升单位价格。"

**26Q1 数据点串联**：
- 产能利用率 80%+ → 主营业务利润率 14.22%（4 季度新高）
- 营收 -1.76% / 净利 +37% → **单价提升而非数量提升驱动盈利**
- 价格联动机制（金 / 铜原材料）从去年起逐步成熟——**毛利率压力部分转嫁到客户**

**关键解读**：
- 长电传统上是 OEM 制造业（议价权弱）——但 26Q1 数据显示，**先进封装结构占比上升后，定价权开始显现**
- 这正是 Buffett 关注的"毛利率结构性改善"的早期信号

### 2.8 毛利率展望"长期有利因素更多"（Q12）

> "整体毛利率从今年短期和未来看均相对乐观。"

管理层列出 4 个驱动：
1. **规模效应**（高产能利用率维持）
2. **结构优化**（先进封装高附加值占比提升）
3. **价格联动**（金/铜涨价转嫁机制成熟）
4. **降本**（金转铜替代 + 智能制造 + 自动化）

**反向风险**：
- 新工厂爬坡期短期毛利承压（已坦承）
- 100 亿资本开支 → 折旧增加 → 中期毛利率压力

**测算**：
- 2025 毛利率 14.15% → 26Q1 14.55%（+0.4pct）
- 若全年维持 14.5-15.5% → **首次稳定在 15% 上方**，是 4 年来的关键突破

| 重仓触发 | PB < 2.0x（~32 元）+ 全年 EPS > 1.20 元 | **PB < 2.0x（~32 元）+ 长电微季度营收 > 5 亿** | 增加量化指标 |
| 减仓触发 | 26Q2 营收同比仍负 + 净利率回落 < 2.5% | **保留** + 新增"100 亿资本开支兑现度 < 70%" | 收紧 |
| 卖出触发 | PE-TTM > 90x | **PE-TTM > 80x（按 26 年 EPS 折算 PE > 75x）** | 收紧 |

## 4 · 标准 Buffett 输出格式

### 结论
**买入** —— 业绩会兑现并扩展了"高端化 + 海外溢出 + 长电微规模化"三轨道增长逻辑，叠加 26Q1 净利 +42.7% 已落地，3-4% 标准建仓，加仓等 PB < 2.5x（~40 元）。

### 圈内圈外评估
**边界区域**。封测商业模式（规模 + 折旧 + 客户绑定 + 议价）我能理解；但 2.5D / CPO / EIC+PIC 堆叠 / Chiplet 异质异构等技术细节超出严格圈内——所以仓位限制在 3-5%，不重仓。判断主要依赖管理层指引可信度 + 客户结构 + 财务健康度，而非技术路径细判。

### 关键假设（4 个）

1. **2026 全年净利 19-22 亿**（华创 19.4 / 季报外推 22）—— 26Q1 已兑现 2.79 亿（占下限 14.4%），季节性比例 13-15% → 全年 19-21 亿落在合理区间
2. **JSCK 韩国二季度起反转**——管理层称"超预期"，半年报披露海外营收若同比转正即验证
3. **长电微 2026-2027 年达"几十亿规模"**——按 30 / 50 / 80 亿三档情景测算，对应不同毛利率拉动
4. **2026 资本开支 100 亿不引发资产负债率突破 50%**——内生现金流 OCF 26Q1 17.8 亿（+55%），全年 70-80 亿覆盖大部分资本开支

### 商业质量
- **护城河**：**强 + 走宽**——规模（全球第三）、海外产能（韩国 + 新加坡）、多客户分散、CPO/2.5D/Chiplet 全谱系能力。**走宽信号**：海外晶圆厂 2.5D 项目溢出 + 端侧 AI 新需求 + 长电微规模化 = 三条新增曲线
- **管理层**：**诚实 / 资本配置稳健 / 业主心态强**——CEO 郑力主动披露 JSCK 财务承压、坦承新工厂爬坡牺牲短期利润；5 年降杠杆 15pct 是同业最优秀的资本配置案例
- **商业模式**：**OEM 制造业 + 部分 franchise 特征**——主流封装是 commodity，但 2.5D / FCBGA 大颗 / CPO 是 franchise 型业务（客户黏性强 + 技术差异化）；混合型
- **Institutional imperative 警示**：**部分存在但有理性约束**——100 亿资本开支看似激进，但满足"客户订单支撑 + 不激进加杠杆 + 管理层坦承代价 + 战略聚焦不模仿同业 + 量化路线图"五项检验，**不是盲目扩张**

### 财务快照
- **ROIC（10 年均值）**：**~7-8%**（同业第一但低于 Buffett 15% 标准）—— 结构性问题，封测重资产折旧拖累
- **现金转换率（OCF/净利）**：**26Q1 = 6.38x**（极佳）；2025 全年 1.7x（健康）
- **债务安全（最差情景测试）**：营收 -30% 测试 → 资产负债率会升至 ~52%（仍可控）+ OCF 仍可覆盖利息 + 大股东国资支持 = **能挺过**
- **业主收益（Owner Earnings）估算**：净利 ~16 亿 + 折旧 ~50 亿 - 维护性资本开支 ~40 亿 - 营运资金增加 ~5 亿 = **~21 亿**（vs 净利 16 亿，含金量好；但近 2 年因新工厂爬坡，业主收益被 100 亿资本开支稀释）

### 估值
- **内在价值区间**：基于 2026-2028 EPS 1.08-1.65 元（华创预测）、合理 PE 30-40x → **内在价值 32-66 元**，中位数 ~50 元
- **当前安全边际**：当前价 ~45 元 → 距中位数 50 元仅 11%，**安全边际偏薄**（对应"较高确定性"项类别下需 30-40% 折扣，仍未到位）
- **建议买入价**：
  - 基础建仓（3-4%）：当前价 45 元附近
  - 加仓（→6%）：< 40 元（PB < 2.5x）
  - 重仓（→8-10%）：< 32 元（PB < 2.0x）

### 卖出标准逐项检查
1. **价格严重高估？** 否——当前 PB 5y 50% 中位 + 隐含 26 年 PE 41x 合理。**继续持有**
2. **基本面护城河破坏？** 否——业绩会显示三条新增长曲线，护城河走宽。**继续持有**
3. **管理层诚信问题？** 否——主动披露 JSCK 承压 + 量化路线图。**继续持有**
4. **是否有显著更好机会？** **存在比较**——通富 PB 5y 91% 偏贵 + 晶方 PE 5y 28% 历史低位但体量小。**长电仍是封测组合的最佳底仓**

### 关键风险（3 个）

1. **2026 资本开支 100 亿 → 折旧前置 → 中期毛利率承压**（最关键变量）——若 2027 年毛利率回落至 12% 以下，估值杀伤大
2. **运算电子下半年爆发式增长不及预期 + 通讯业务 Q2 反转失败**——管理层指引相对乐观，若实际 Q2-Q3 营收仍负则故事破裂
3. **JSCK 韩国二季度反转不达预期**——管理层公开承诺"超预期"，若证伪则管理层信誉受损（但仅占整体营收 ~10-15%，影响有限）

### 监控指标

**每季度检查**：
- 主营业务利润率（守住 14% 是分水岭，14.5%+ 是结构性改善信号）
- 资产负债率（守住 45% 以下；若突破 50% = 100 亿资本开支引发风险）
- OCF / 净利比（维持 1.5x+ 是健康线）
- 海外营收同比（半年报披露——JSCK 反转的直接证据）
- 长电微季度营收（是否走向"几十亿规模"）
- 运算电子 + 汽车电子收入占比（管理层目标"突破 50%"）

**月度跟踪**：
- 高通 / 海思 / 博通 / Marvell / SanDisk 季度财报
- 台积电 CoWoS 产能预订（推断海外溢出节奏）
- 全球 NAND Flash 价格（影响 SanDisk 出货 → 长电存储业务）

**触发卖出信号**：
- 单季毛利率 < 12% / 资产负债率突破 50%
- 单季营收同比 -30%+
- PE-TTM > 80x 或 PB > 4.0x（5y 80%+ 分位）
- 100 亿资本开支兑现度 < 70%（半年报核查在建工程 + 固定资产增量）
- 大客户出走（高通 / 海思 / SanDisk 任一 official 减少订单）

### 整体评价（Buffett 视角口吻）

> "这家公司让我想起 1980 年代我们持有的吉列——表面是制造业，但里头藏着护城河。它不是消费品式的定价权，但当全球第三的规模、海外产能（韩国 + 新加坡）和多客户分散度叠加在一起时，竞争对手 10 年内难以复制。
>
> 我喜欢的三件事：第一，CEO 郑力在公开场合直接说'JSCK 财务表现暂不理想'——这是我希望从所有 CEO 那里听到但很少听到的话。第二，2025 年杠杆从 58.5% 降到 43%，5 年减 15 个百分点——这是 Tom Murphy 在 ABC 时代干过的事。第三，海外晶圆厂的 2.5D 项目溢出——这意味着 5 年前花大价钱并购的星科金朋，现在终于派上用场了。
>
> 但我也有保留：100 亿的资本开支让我想起当年伯克希尔投美航的失误。重资本生意永远是一场跟折旧的赛跑。所以我会买，但不重仓——3-4% 是合适的位置，留点空间等市场情绪悲观时（PB 跌到 2.5x 以下）加到 6%。
>
> 卖出条件很简单：要么价格涨到 PE 80x（PB 4x）以上脱离基本面，要么资产负债率重回 50% 以上（说明 100 亿资本开支引爆了），要么 CEO 开始模糊回答 JSCK 或长电微的进展。在那之前，10 年持有。"

# 华虹半导体（1347.HK / 688347.SH）— 2026Q1 季报点评

> 报告日期：2026-05-14（公司当日交易时段后披露 Q1 业绩）
> 数据来源：公司业绩公告 + wallstreetcn / 新浪财经业绩前瞻与点评
> 视角：**Buffett 框架快速过滤 + 折旧曲线 / 产能利用率 / 价格战三变量监控**

---

## 一句话结论

**Keep Watching**（边界圈内观察）— 营收 +21% / 毛利率拐点确立，但折旧吞噬利润 + 代工本质偏 commodity，当前估值已 price-in 复苏，安全边际不足。

## 圈内判断

**边界区域**。晶圆代工属"资本密集型类大宗商品"——巴菲特历史上明确远离（航空 / 纺织教训）。华虹的差异点是**特色工艺**（功率分立 / 嵌入式存储 / RF / CIS / 模拟），在 55nm 以上节点对台积电构成错位竞争，且深度受益于国产替代政策。但本质仍是**周期 × 政策 × 资本开支**三驱动，不是富兰克林型复利机器。

**关键观察**：营收涨 21% / 毛利率涨 3.5pp / 净利 QoQ -49%——**折旧这台跑步机的速度正在追上业务的增长**。

## 关键假设

1. 12 英寸无锡 Fab9 折旧高峰已在 2025-2026 释放，2026H2 起摊销曲线趋平
2. 国产替代订单（功率半导体 / IGBT / MCU）维持高产能利用率（Q4 为 103.8%）
3. ASP 缓慢回升，价格战不再恶化（成熟节点 IDM / SMIC / 晶合 / Nexchip 是主要变量）
4. 消费电子（占营收 63.7%）复苏可持续，不是库存回补一次性反弹
5. 美国出口管制不进一步加码到 28nm/22nm 设备进口

## 业务质量

- **护城河**：弱-中。特色工艺有客户黏性 + 转换成本（PDK 重做周期 1-2 年），但**趋势稳定 / 不在扩张**——SMIC、合肥晶合、Nexchip 都在加码同节点产能
- **管理层**：国资背景（华虹集团 / 大基金 / 上海联和），整合能力强，无诚信瑕疵；资本配置受政策驱动多于股东回报视角
- **商业模式**：commodity（晶圆代工产能租赁），靠规模 + 工艺 know-how + 客户认证三件套形成弱壁垒
- **机构强制力警示**：**存在**——12 英寸新厂扩产是政策诉求，非纯 ROIC 驱动

## 财务快照

- **ROIC**：估算 < 5%（低于资本成本，典型资本密集型代工特征）
- **现金转化率**：经营现金流尚可（折旧 add-back 大），但**自由现金流长期为负**（Capex > OCF）
- **债务安全**：央企背景 + A+H 双融资通道，债务安全无忧
- **Owner Earnings**：Fab9 折旧爬坡完成前，账面利润高估真实股东收益——维护性 Capex 难以分离扩张性 Capex

## 估值

- **PB ≈ 1.3-1.5x**（H 股），A 股 PB 显著更贵（A/H 长期 3x+ 溢价）
- **内在价值锚**：按 mid-cycle ROE 6-8% 给 1.0-1.2x PB → H 股合理区间 35-42 HKD
- **当前安全边际**：负到零（已 price-in 周期复苏 + 政策溢价）
- **建议买入价**：H 股 35 HKD 以下；A 股折价非常困难

## 卖出标准逐项

1. 价格严重高估？— **A 股是，H 股临界**
2. 护城河毁灭？— 否，但**没有扩张**
3. 管理层诚信？— 否
4. 更优机会？— **是**（SMIC H 股折价更深 / 中微公司设备端护城河更硬 / 海光信息算力主线）

## 关键风险（Top 3）

1. **折旧吞噬利润**：Q1 毛利率 +1.2pp 但净利 QoQ 腰斩，说明无锡新厂折旧 / 财务费用还在快速增长；H2 若产能利用率回落，利润释放被吃光
2. **价格战重燃**：SMIC、合肥晶合、Nexchip 都在 12 英寸成熟节点扩产，2026H2-2027 供过于求风险
3. **A-H 折价收窄已被透支**：688347 相对 1347 长期 3x+ 溢价，回归触发点是 H 股上涨而非 A 股下跌的概率有限

## 监控指标

**季度跟踪**：
- 毛利率（看是否守住 15%+）
- 产能利用率（>100% 是健康线，<90% 警戒）
- 12 英寸 ASP 同比
- 折旧绝对额环比

**触发卖出**：
- 产能利用率跌破 90%
- 毛利率连续两季环比下行
- Fab9 二期 Capex 公告超预期
- 美国把 28nm 设备纳入出口管制

## 综合评判（Buffett 口吻）

"这是一门**赌产能 + 赌政策 + 赌周期**的生意，不是赌商业模式。Q1 数字漂亮，但你仔细看：营收涨 21%，毛利率涨 3.5 个点，净利润却比上一季少了一半——折旧这台跑步机的速度正在追上业务的增长。当我看到一家公司利润表的'分子'被折旧'分母'追着跑，我就知道这不是 See's Candies、不是可口可乐，它更像 1960 年代的伯克希尔纺织厂。

不是说不能赚钱。代工行业每隔 4-5 年有一波周期红利，现在正处在拐点向上的位置。但你要赚的是 **β 而不是 α**——指数化暴露半导体国产替代板块比单押华虹更聪明。如果你已经持有，且成本足够低（H 股 35 港币以下），可以拿一拿；如果是新开仓，请先回答：为什么是华虹，而不是中芯国际（更深 H/A 折价）或者中微公司（设备端护城河 + 高 ROIC）？"

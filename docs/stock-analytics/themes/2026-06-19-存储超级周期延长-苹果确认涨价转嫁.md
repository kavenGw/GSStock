## 2. 逐标的传导分析（仅中量级以上展开）

### 兆易创新（603986）— 利好 / 中
- **旧 thesis（buffett 2026-05-31, watch）**：三巨头退出低端存储 + AI 挤出让利基 DRAM/NOR 量价齐升（26Q1 净利 +523% 已兑现），GD32 MCU / AMOLED NOR 是被低估护城河；但 wafer 回流可逆 + 4x 溢价让"好生意"难成"好价格"。
- **新闻改变的变量**：分子端的**涨价持续性**——Q3 再涨 30-40% + 周期延长到 2027，意味着 26 全年利基存储量价齐升的可持续性比旧档判断更强，bear 论据"wafer 回流终结涨价"被推后。
- **双面**：利好侧＝景气周期拉长、26 业绩弹性上修；风险侧＝今日股价已 +7.3%、PE 153x/PB 17.8x，估值在周期顶部用顶部利润定价，新闻利好被即时吃掉，安全边际进一步恶化。
- **结论**：维持 watch，**不因涨价升级评级**。旧档矛盾（好生意/坏价格）反而加深。等周期证伪或估值回落，而非追涨。

### 江波龙（301308）— 利好 / 中
- **旧 thesis（buffett 2026-05-31, exclude）**：模组商是颗粒涨价纯顺价受益方（周期 β 非产品 α），26Q1 毛利 55% 是不可持续 windfall；穿越周期归一化期望内在价值仅约 231 亿 vs 市值 2332 亿，无安全边际。
- **新闻改变的变量**：颗粒涨价延长 → windfall 持续时间拉长，短期顺价利润弹性最大（模组对颗粒涨价杠杆最高）。
- **双面**：利好侧＝Q3/Q4 颗粒续涨直接抬模组售价与毛利；风险侧＝windfall 越大、归一化偏离越远，越坐实"用周期顶利润定价"的陷阱；今日仅 +1.07%，前期已大涨。
- **结论**：维持 exclude。新闻强化的是"周期弹性"而非"内在价值"，旧档逻辑不翻案。

### 太极实业（600667）— 利好 / 中
- **旧 thesis（buffett 2026-04-25, watch）**：SK 海力士 DRAM 封测合资（海太半导体），等涨价传导。
- **新闻改变的变量**：海力士把产能挪向 HBM/AI、DRAM 全面涨价 → 合资封测厂稼动率与价同步抬升，"等传导"的前提正在兑现。
- **双面**：利好侧＝封测放量直接受益，今日涨停 +9.99% 是市场确认；风险侧＝太极实业利润含工程/IDC 等非封测板块，存储弹性被稀释，PE 已 95x。
- **结论**：旧档"等涨价传导"的条件部分满足，但涨停后估值不便宜，维持 watch，观察封测分部利润占比兑现，不追涨停板。

### 深科技（000021）— 利好 / 中
- **旧 thesis（buffett 2026-06-08, watch）**：HBM 量产 + 英伟达认证 + 长鑫扩产，从"周期型封测代工"升级为"AI 存储供应链核心节点"，但估值已部分定价、安全边际不足。
- **新闻改变的变量**：苹果确认转嫁＝**需求端验证**，叠加 HBM 2027 翻倍 → 深科技 HBM/DRAM 封测的中长期需求曲线更陡，旧档"核心节点"叙事获增量支撑。
- **双面**：利好侧＝长鑫扩产 + HBM 需求双轮，封测订单可见度提升；风险侧＝净利率薄的封测代工本质未变、PE 58x 已部分定价。
- **结论**：维持 watch，新闻为旧档加分但不足以触发升级；回落给安全边际可重点观察（池内存储链里少有的"真供应链节点"而非纯价格弹性）。

### 澜起科技（688008）— 利好 / 中
- **旧 thesis（buffett 2026-05-30, watch）**：DDR5 内存接口芯片全球寡头 + 堡垒资产负债表，真护城河；但 120x 周期峰值盈利定价，安全边际为负。
- **新闻改变的变量**：DRAM 涨价 + 服务器 DDR5 渗透加速 → 接口芯片量（DDR5 配套）价（行业景气）双击，护城河变现窗口拉长。
- **双面**：利好侧＝寡头格局下涨价周期最受益的"卖铲人"，质地优于纯颗粒厂；风险侧＝PE 129x 已把多年成长贴现，今日 +2.8% 继续推高估值。
- **结论**：池内存储链质地最优标的之一，但维持 watch——好公司贵价格，等周期回落而非追顶。

### 小米集团（01810）— 利空 / 中
- **旧 thesis（buffett 2026-06-02, watch）**：手机+IoT+互联网现金牛叠加 EV 第二曲线，但 EV 烧钱爬坡、**存储涨价压制毛利**、估值仍 price-in 汽车成功——好公司当前价安全边际为负。
- **新闻改变的变量**：库克表态等于行业巨头公开承认存储成本无法自我消化——小米手机毛利率本就薄，存储（DRAM+NAND）是手机 BOM 第二大成本项，旧 thesis 里"存储压毛利"的风险被**坐实并放大**。
- **双面**：利空侧＝26 手机毛利承压、或被迫涨价伤量；缓释侧＝苹果带头涨价为全行业打开终端提价空间，小米若跟涨可部分转嫁（但中低端机型提价弹性弱于苹果）。
- **结论**：维持 watch，旧档对存储成本的警惕被验证，安全边际为负的判断不变；下游里小米是存储成本敞口被点名验证的标的。

### 联想集团（00992）— 利空 / 中
- **旧 thesis（buffett 2026-06-08, watch）**：全球 PC 龙头叠加 AI 服务器高增，但**净利率仅 2%**、股价已翻倍，安全边际不足。
- **新闻改变的变量**：存储（DRAM+SSD）占 PC 物料成本 15-20%，在 2% 净利率的极薄盈利结构下，DRAM Q3 +30-40% 的成本冲击对净利的杠杆效应极大——同等转嫁不足时，单位涨价对 EPS 的侵蚀远高于高毛利公司。
- **双面**：利空侧＝PC 需求疲软期难足额转嫁，毛利/净利双压；缓释侧＝若行业性涨价（戴尔/惠普同步）则可被动跟涨、AI 服务器板块成本可向客户传导。
- **结论**：维持 watch，新闻是旧档"薄利+高估值"的负向催化，下游里盈利结构对存储成本最脆弱的标的，需求转嫁能力是关键观察点。

### 无影响 / 弱传导归并（留总览表，不展开）
- **北京君正/普冉/东芯/佰维/聚辰**：均为利基存储设计/模组，已在 5/31 buffett 重估中明确判为"周期顶 commodity、估值透支（PE 145-25000x / PB 7-33x）"，本新闻只是同向情绪强化，**不改 exclude/watch 评级**，且今日多数已大涨兑现。
- **雅克/中巨芯/南大光电/赛腾（上游材料设备）**：doc 已明确"传导滞后、商品属性、估值透支/叙事先于业绩"，新闻为弱利好但二阶传导慢，不翻案。
- **立讯/工业富联/歌尔/影石（下游代工/消费电子）**：存储成本可部分向客户（苹果/CSP）转嫁或存储占自身 BOM 比例有限，中性偏空，量级低，不改 watch。

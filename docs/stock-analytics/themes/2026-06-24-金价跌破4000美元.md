## 0. 事件核实

**确认属实。** 截至 2026-06-24，现货黄金跌破 $4000，报约 $3968–3978/oz（Kitco 报 $3968.62，down 3.19%），较 1 月高点 ~$5600 累计回撤约 25%（中金黄金档参照点 $4152 已被进一步跌穿）。

**驱动三因素**（多源一致）：
1. **美联储鹰派预期升温**——市场计入更紧货币政策，实际利率上行压制无息资产
2. **美元走强**——美元指数逼近 52 周高位 ~101.7，以美元计价的金价承压
3. **美伊达成临时和平协议**——地缘避险溢价消退

注意三因素中"强美元 + 鹰派 Fed"对**所有以美元计价的大宗**（铜/铝/钨）都是广义压制，但本专题只判**金价自身传导链**（贵金属敞口），广义商品宏观压力仅在末节一句带过、不展开。

## 2. 逐标的传导分析

### 中金黄金（600489）— 利空 / 高

- **旧 thesis（2026-06-21 exclude）**：矿产金毛利率第一梯队 + 纱岭真增量，但当前价用周期顶利润 + PB 3.24 定价，金价 ATH 后回调期不构成买点；exclude_reason 明写"金价 β 仍是当下盈利主驱动"，三情景 Bear(35%) = "金价深跌回中枢"。
- **这条新闻改变了哪个变量**：盈利 ≈ (金价 − 成本) × 产量，金价是最大且不可估的输入。6/21 档以 $4152 为参照，今日跌破 $4000 = **Bear 场景正在路径上兑现**，直接压缩矿产金 53.90% 高毛利分部的单位利润。
- **双面看**：利空侧——纯金矿无对冲，金价 β 单边暴露，破位削弱"用 TTM 周期顶利润支撑现价"的逻辑，向 bear 估值（2.70 元）收敛；潜在缓冲侧——纱岭 10 吨/年 2026H2 投产带来的**量增**可部分对冲价跌，且央行购金(2025 全年 863 吨)是结构性需求底，金价未必直线下行。但量增兑现在下半年、远水难救近渴。
- **结论**：**维持 exclude，确信度提高**。这不是翻案信号而是旧 thesis 的实证强化——旧档的估值纪律（不用周期顶 TTM 出价）被市场验证。无操作必要，继续回避；若后续金价企稳 + 纱岭满产降本兑现至 base(7.15 元)以下再重审。

### 紫金矿业（601899）— 利空 / 中

- **旧 thesis（2026-04-24 core）**：黄金 + 铜双轮驱动，`commodity: copper`（主轴是铜），黄金承担 `gold_defense` 防御主线角色；分析口径"不买/安全边际严重为负"，但作防御配置持有。关键假设#2 早已假定"金价回 $1,600–1,800 中枢的概率高于维持 >$2,500"，**估值已用正常化低金价口径**（正常化所有者盈利 ¥150–180 亿）。
- **这条新闻改变了哪个变量**：① 内在价值——**基本不变**，因旧档的正常化估值早已假设金价远低于现价（$4000 仍显著高于其 $1,600–1,800 中枢假设），金价回落只是向旧档假设收敛，不构成新利空；② **叙事——动摇**：黄金被定位为"防御主线"，而黄金本身在 25% 回撤中，"持金矿避险"的防御逻辑被证伪。
- **双面看**：利空侧——金价回落直击次要的黄金利润腿 + 瓦解防御叙事；对冲侧——主轴是铜（`commodity_impact: positive`），铜价周期与金价驱动不完全同向，铜腿仍是利润压舱石，整体利润弹性远小于纯金矿；且金价向正常化中枢回归本就是旧档预设路径。
- **结论**：**core 维持不变**（铜主线 + 低成本资源位未破坏），但**需重新认知其"防御"定位**——它是优质铜矿股，不是金价对冲工具。持有逻辑应锚定铜的长周期供需，而非"持金避险"。这是认知校正，非操作信号。

### 盛达资源（000603）— 利空 / 中

- **旧 thesis（2026-05-13 exclude）**：国内白银主业，典型大宗无定价权，2024–25 利润爆发由银价周期驱动；PE-TTM ~50x / PB ~7.7x 是按周期顶盈利的高估，10 年正常化净利约 3.3 亿 vs 当前 270 亿市值差 5–7 倍，安全边际严重为负。
- **这条新闻改变了哪个变量**：白银与黄金高度同向（同为贵金属避险/货币属性），金价破位通常伴随银价回落，直接压缩盛达"按周期顶盈利计"的当下利润，使 50x PE 的分母面临下修。
- **双面看**：利空侧——纯银 price-taker，价跌即利润跌，周期顶估值无缓冲；风险对称侧——银的工业属性(光伏/电子)强于金，若工业需求托底，银价回落幅度可能小于金，但当前驱动是货币端(强美元/鹰派)而非工业端，金银同跌概率高。
- **结论**：**维持 exclude，强化**。金价破位是其"周期顶估值"风险的实时催化，旧档"安全边际严重为负"判断被进一步验证，继续回避。

### 铜陵有色（000630）— 利空 / 中

- **旧 thesis（2026-06-02 watch）**：千亿营收、净利率 2% 的纯冶炼 price-taker，冶炼主业被 $0/负 TC/RC 吞噬，**真正赚钱的是金银副产**；当前高 PE/PB 主由控股铜冠铜箔 714 亿 AI 市价重估撑起（本身 140x PE 泡沫）。
- **这条新闻改变了哪个变量**：贵金属下行直击其**唯一真实盈利来源（金银副产）**——冶炼主业本就不赚钱，金银副产是利润垫，金价/银价回落侵蚀这块真实利润。
- **双面看**：利空侧——金银副产利润缩水，暴露冶炼 stub 的平庸本质；但估值锚的特殊性——其市值主驱动是铜冠铜箔 AI 期权(714 亿)而非贵金属副产，故**股价对金价的敏感度被 AI 叙事稀释**，金价利空对基本面真实、对短期股价传导被泡沫盖过。
- **结论**：**watch 维持**。金价破位强化旧档"拆开看是平庸冶炼 stub + 金银副产 + 含泡沫 AI 期权"的判断——真实盈利来源(金银)正在承压，而支撑现价的(AI 期权)与本新闻无关。不构成买点，继续观察 AI 泡沫与贵金属双向消化。

### 弱传导（低量级）：北方铜业、江西铜业 — 弱利空 / 低（表内带过，合并简述）

两只都是「铜矿/冶炼 + 金银副产缓冲」结构，金价回落只削弱其副产缓冲的一角，铜主业才是估值主体，故弱利空、均不改 watch：

- **北方铜业（000737，watch）**：自有矿(自给率 14–33%) + **高毛利贵金属副产**，PB 3.58 板块最贵的溢价"应归因于矿 + 金"。金价回落边际削弱"贵金属副产撑溢价"的一角依据，但铜矿端(自给率)才是其差异化主体，贵金属副产占比有限 → 弱利空、不改 watch。
- **江西铜业（00358，watch）**：德兴铜矿一体化为主业，**金银副产是利润缓冲**（旧档明列为结构性优势之一，"金价高位尤其明显"），且旧档已点明 2026Q1 +44% 利润含「铜金价格高位」的周期 β。金价回落使这部分缓冲收窄、周期 β 部分回吐，但德兴铜矿与 AH 折价/FQM 敞口才是其核心叙事，金价非主驱动 → 弱利空、不改 watch（H 股口径安全边际仍约 -56%）。

## 3. 操作含义

- **无评级翻转**。金价破位对池内贵金属标的是**清一色利空 + 强化既有偏空/回避立场**，没有任何标的因此从利空转利好（池内无黄金珠宝零售等"金价下行受益"的下游标的）。
- **认知校正 > 操作信号**：核心收获是**"持金矿避险"的防御逻辑已被证伪**——黄金自身在 25% 回撤中。紫金(core)需从"gold_defense 防御工具"重新定位为"优质铜矿股"，持有逻辑锚定铜而非金。
- **中金黄金(exclude)** 是金价 β 最纯的载体，破 4000 = 其 Bear 场景兑现，旧档估值纪律被验证，确信回避；后续重审触发点是金价企稳 + 纱岭满产降本。
- **盛达(exclude)/铜陵(watch)** 真实盈利来源(银/金银副产)承压，维持回避/观察。**北方/江铜(watch)** 金银副产缓冲收窄，弱利空，铜主业与各自核心叙事(矿端/AH折价)未变，维持观察。
- **广义宏观**：强美元 + 鹰派 Fed 对池内铜/铝/钨等全大宗形成广义美元计价压力，但那是金价下跌的同因、非金价自身传导，本专题不展开；若美元单边趋势延续，需在后续铜/铝标的复审中单独计入。

## 1. 定律本质：一句话讲清

> 摩尔定律靠 **"晶体管变小→更密→更快"**；韬定律换思路 — **不缩晶体管，缩信号跑过去的时间**（垂直堆叠物理变短 + 总线协议变快），用 **先进封装 + 系统协同** 跳过先进制程封锁。

### 1.1 四层协同体系（来自何庭波演讲）

| 层级 | 优化对象 | 关键动作 |
|------|---------|---------|
| 器件层 | 晶体管电阻 / 寄生电容 | 材料/结构层面降阻容 |
| 电路层 | 物理布线距离 | **逻辑折叠** — 从单层折成双层乃至多层 |
| 芯片层 | 无效开销 | 全栈协同设计 |
| 系统层 | 通信时延 | 自研 **灵衢总线**，重构超节点统一内存编址 |

### 1.2 技术不新，但叙事新

逻辑折叠 ≈ 3D 堆叠 + 混合键合（HBI）+ Chiplet 互联；灵衢 ≈ CXL/NVLink 类协议。**真正的新意是华为把这些分散技术统一为"中国半导体指导原则"，给国产产业链一个统一旗号。**

## 4. 风险点

1. **远期愿景 vs 短期业绩**：1.4nm 等效到 2031 是 **5 年远期**，短期对封装厂订单弹性有限，纯主题炒作易冲高回落
2. **估值已不便宜**：长电、通富已在 CoWoS/AMD 链上跑了一年（参见 `comps/长电通富-封测赛道横向对比`），追高需控仓
3. **数据未披露**：华为未公布逻辑折叠的良率/单位面积成本/功耗，"等效 1.4nm" 的桥接需第三方验证
4. **政策风险**：若美方进一步收紧封装/键合设备出口（盛美、拓荆已在 SDN list 边缘），国产化进度仍可能受扰

## 2. 逐标的传导分析（仅中量级以上展开）

### 2.1 亿纬锂能（300014｜watch）— 强化盈利反转，毛利率止跌命题获增量证据，但估值仍贵不翻案

**旧 thesis（2026-06-16 buffett 档）**：储能放量与盈利反转都真实，但**增收不增利是结构性特征**（毛利率 21.6%→16.2%、储能毛利率 17%→11%），"量增能否持续转化为利润"被全行业 +94.59% 扩产 + 价格战**高度对冲**；期望内在价值约 656 亿 vs 市值 1453.5 亿，安全边际约 -55%。doc 把"**储能毛利率能否止跌于 11%**"列为 2026 全年最大待验证命题，需 2-3 季度扣非验证。

**这条新闻改变了哪个变量**：直击 doc 的**核心悬念——储能电芯价格/毛利率**。旧 thesis 的最强反驳是"全行业疯狂扩产→价格战→毛利难修复"；本新闻给出**反向价格证据**：314Ah 涨 25-35%、满产拒单、排产到 2027，2026 H1 紧平衡。这是"毛利率止跌→反转可持续"的边际增量，向 doc 的 **bull 情景（22%，毛利率止跌于 11% 后修复）倾斜**。

**双面看**：
- **利好侧**：电芯涨价直接改善价差；亿纬满产 87.8% 利用率下涨价几乎纯利润弹性；与 H1 预告净利 +95~110%、扣非 +110~125% 自洽印证。
- **冷静侧（不翻案）**：① 旧 thesis 的"贵"是**估值问题**（安全边际 -55%，连 bull 都 -32%），一波涨价不改"市场已把 H1 年化 60-70 亿当常态定价"的数学；② 涨价可持续性未证——InfoLink 判 **H2 适度宽松**（新增产能释放），这波紧平衡可能是 H1 现象；③ 亿纬自身 260GWh 大铁锂规划本身是后续供给增量；④ 碳酸锂同步涨到 17-18 万是**成本端反向压力**（亿纬是买方），价差改善幅度被部分抵消。

**接旧 thesis / 操作含义**：**评级 watch 维持**。本新闻提高了"反转可持续"的概率（doc 绿灯里"储能毛利率明确止跌回升"的验证信号正在积累），但**不构成买入信号**——买点仍锚 doc 既定阈值（市值回落至 480-680 亿区间 + 2-3 季度扣非验证毛利率止跌）。本新闻是"验证信号之一"，不是买点。跟踪：**储能业务单季毛利率**能否从 11% 回升（doc §11 风险 1 监控指标），这才是涨价证据兑现到财务的试金石。

### 2.2 瑞浦兰钧（00666.HK｜watch）— 薄利涨价弹性最大，反向证据最强，但治理/高负债/估值不变

**旧 thesis（2026-06-20 buffett 档）**：2025 首次年度盈利 6.81 亿（连亏六年后）、储能/户储放量真实，但**全年利润 = H2 单边盈利约 7.4 亿的剧烈拐头、净利率仅 2.8% 极薄、2024 末资产负债率 73.2% 高杠杆 + 青山系治理隐患**；最强反驳是行业 2026 末规划产能 6730GWh/利用率降至 38.8% + **储能集采跌破 0.45 元/Wh 价格战**；期望内在价值约 222 亿 HKD vs 市值 369.69 亿，安全边际约 -40%。

**这条新闻改变了哪个变量**：瑞浦是本主题**价格弹性最大的标的**——净利率仅 2.8%，处盈亏平衡线附近，电芯涨价 25-35% 对其利润是**高杠杆放大**（薄利公司涨价对净利的弹性远大于厚利公司）。同时涨价/满产是旧 thesis"集采跌破 0.45 元/Wh + 利用率降至 38.8%"这个最强反驳的**直接反向证据**——价格在反弹、产能在满产拒单，与"价格战+过剩"的空头叙事相左。

**双面看**：
- **利好侧**：薄利下涨价的利润弹性最大；户储 2025H1 全球第一卡位 + 海外占比目标 90%，海外储能景气共振；这波涨价若延续到 H2，是"H2 单边盈利可持续"的最有力佐证。
- **冷静侧（不翻案）**：① 旧 thesis 的"贵"（PE 53x、安全边际 -40%）+ **青山系治理隐患 + 73.2% 高负债**是**非价格因素**，涨价不解决；② "H2 单边盈利能否成为常态"仍**未被任何后续季报验证**，一波电芯涨价 ≠ 季报兑现；③ 注意口径——新闻的 0.38-0.42 元/Wh 是**电芯出厂价**，旧 thesis 的 0.45 元/Wh 是**储能系统集采价**，两者不同层，但方向（价格底部反弹）一致；④ 碳酸锂涨价同样是成本端压力。

**接旧 thesis / 操作含义**：**评级 watch 维持**。本新闻强化 bull 侧（doc bull 18%：首盈可持续 + 毛利率继续修复），但**估值 + 治理 + 杠杆三重约束不变**，买点仍锚 doc 阈值（市值回落至安全区间 + 2-3 季度季报验证 H2 单边盈利可持续）。瑞浦是"高弹性反转标的"——涨价让它**短期股价弹性更大**，但 buffett 纪律下仍是"等验证、等价格"，不是买点。

### 2.3 阳光电源（300274｜watch）— 下游系统集成，电芯成本 vs 储能需求双面对冲

**旧 thesis（2026-06-07 buffett 档）**：储能系统集成全球 #1 + 海外 60% 收入，穿越周期能力同业第一，但 55% 关税 + FEOC 限制是结构性风险；AIDC 储能是"真期权但不进估值"。

**这条新闻改变了哪个变量**：阳光是**电芯买方**（储能系统集成商外采电芯），电芯涨价 25-35% 是其**系统 BOM 成本上升**——与上游电芯厂方向**相反**。但阳光不是单纯的成本承受者：储能需求爆发（"一芯难求"的需求侧根源）直接利好其系统出货量，且系统集成 + 自研 PCS 有溢价与议价能力，能向下游传导成本。

**双面看**：
- **利好侧**：储能需求旺 → 系统集成销量增；行业景气抬升 + 龙头议价权，成本可部分传导；电芯紧张时绑定头部电芯供应反成壁垒（中小集成商缺货）。
- **风险侧**：固定价订单下电芯涨价直接压系统毛利（短期）；55% 关税 + FEOC 仍是 doc 标注的风险 #1，与电芯成本叠加。

**接旧 thesis / 操作含义**：**评级 watch 维持**，方向净中性。电芯涨价对阳光是"成本压力 vs 需求旺"的双面对冲，不触及 doc 的核心估值/关税逻辑。买入触发条件不变（PE 15x≈131 元 / 储能毛利率守 28%）。跟踪：**储能系统毛利率**能否在电芯涨价中守住（议价权/传导能力的试金石）。

### 2.4 嘉元科技（688388｜watch）— 锂电铜箔需求拉动，强化加工费反转但仍回升早段

**旧 thesis（2026-06-19 buffett 档）**：锂电铜箔加工费触底反转 + AI/高端电子铜箔国产替代双驱，2026 Q1 净利 YoY+393% 确认盈利拐点，但**利润核心（加工费）仍处周期回升早段**、复合铜箔是双刃替代威胁，估值用周期顶想象定价、安全边际深度为负。

**这条新闻改变了哪个变量**：储能电芯放量（出货 +70~94%）**直接拉动锂电铜箔需求**（铜箔是电芯负极集流体），需求旺 → 铜箔加工费回升的需求侧佐证。强化 doc 的"加工费触底反转"假设。

**双面看**：
- **利好侧**：电芯需求爆发是铜箔加工费回升的上游拉力；嘉元 86% 传统电解铜箔主业直接受益于储能/动力电芯放量。
- **冷静侧（不翻案）**：① doc 已认加工费"仍处周期回升早段"，且 Q1+393% 已 price-in 周期顶想象，安全边际深度为负——需求佐证不改"估值贵"；② 铜箔加工费由供需双侧定，电芯需求旺但铜箔自身也在扩产；③ 复合铜箔对传统电解铜箔的替代威胁与本新闻无关、独立存在。

**接旧 thesis / 操作含义**：**评级 watch 维持**。本新闻是嘉元"加工费反转"逻辑的需求侧背书（中性偏多），但估值约束不变，不构成买点。

### 2.5 赣锋锂业（002460｜watch）— 锂盐需求驱动，但 thesis 不赌锂价单边

**旧 thesis（2026-06-15 buffett 档）**：真护城河是**资源自给率从 32% 升至 47% 的穿越周期成本优势，而非赌锂价单边上行**；当前估值已充分反映 2026 盈利修复且无安全边际。

**这条新闻改变了哪个变量**：储能电芯需求爆发是**碳酸锂从 5.9 万回到 17-18 万/吨的核心需求驱动**（万华档已核实该背景）。锂价上行直接利好赣锋盈利。但——doc 的核心论点恰恰是"**护城河是成本优势而非赌锂价**"，故锂价上行对其是**中性偏多但不触及核心 thesis 变量**。

**双面看**：
- **利好侧**：储能需求驱动锂价上行 → 赣锋作为锂盐卖方盈利改善；资源自给率护城河在锂价上行期放大盈利弹性。
- **冷静侧（不翻案）**：① doc 明确"不赌锂价单边"，对边际需求不敏感；② 估值已充分反映 2026 修复、无安全边际；③ 与亿纬/瑞浦（锂买方）方向相反——同一条储能景气，对赣锋是收入端利好、对电芯厂是成本端压力，这正是 related_docs 把赣锋与亿纬并置的上下游对偶价值。

**接旧 thesis / 操作含义**：**评级 watch 维持**。锂价上行是赣锋盈利顺风，但 doc 的买点逻辑锚"成本护城河 + 估值折价"而非锂价，本新闻不改买点（等估值回落至安全区间）。

## 3. 操作含义（组合层）

1. **评级零变动**：本新闻不触发任何池内标的的评级调整。这是诚实结论——电芯量价齐升强化了储能链景气叙事，但亿纬/瑞浦的"贵"（安全边际 -55%/-40%）、治理/杠杆约束、可持续性未证，均非一波涨价能改变。
2. **真正的增量价值在"价格证据"**：本新闻给亿纬/瑞浦两档悬而未决的核心命题（储能毛利率能否止跌 / H2 单边盈利能否持续）提供了**向 bull 侧倾斜的边际证据**，但需 2026 H2 财报兑现到**储能业务单季毛利率回升**才算验证——这是跟踪信号，不是买点信号。
3. **上下游对偶提醒**：同一条储能景气，对**电芯厂（亿纬/瑞浦）是量价齐升利好**、对**下游系统集成（阳光）是外采成本压力**、对**上游材料（赣锋锂盐/嘉元铜箔）是需求拉动**——方向不一，组合里勿把"储能利好"一刀切。
4. **跟踪信号**（验证涨价可持续性 + 兑现到财务）：
   - 储能电芯价格（314Ah/587Ah 月度报价）能否守住涨幅、是否随 H2 新增产能释放回落（InfoLink 判 H2 适度宽松）
   - 亿纬/瑞浦**储能业务单季毛利率**环比（涨价兑现到利润的试金石）
   - 碳酸锂价格（成本端，亿纬/瑞浦为买方，赣锋为卖方，反向）
   - 储能系统集采均价（阳光/上能/科华 PCS 毛利相关）

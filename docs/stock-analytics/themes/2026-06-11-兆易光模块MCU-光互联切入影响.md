## 2. 逐标的传导分析

### 兆易创新（603986）— 中性偏多 / 中（叙事）· 低（业绩）

- **旧 thesis（2026-05-31 buffett，rating=watch）**：好生意被供给侧重估 + AI 挤出确认（GD32 MCU + AMOLED NOR 是真护城河，26Q1 净利 +523% 已兑现），但 3,274 亿市值 / PE 114 / PB 13.2 把所有好消息都 price-in，安全边际 −78%。AI 敞口里**最实的是供给侧挤出（已兑现）；GD32H7 边缘 AI MCU、AI server boot NOR 等是"真实但未量化的产品层期权"**；自身不做 HBM/高端 DRAM（维度④缺位）。
- **这条新闻改变了哪个变量**：本质是给"产品层 AI 期权"再添一个数据点 —— 光模块专用 MCU 锚定 CPO/硅光，与 GD32H7、AI server NOR 同属"AI 算力 capability 真实、对营收贡献未量化"的期权篮子。它**边际拓宽 GD32 MCU 分部的成长叙事与管理层执行力背书**（从基站/传输延伸到 CPO 这一最热的 AI 互联前沿），是收入端的"可选项"而非"确定项"。
- **双面看**：
  - 利好侧：① 验证 GD32 MCU franchise 向高景气 AI 赛道延展的能力，护城河叙事更宽；② 光模块 MCU 此前出货已千万级、全球前列，高速段补强对应 800G/1.6T/CPO 放量节奏，理论 TAM 随 AI 光互联扩张。
  - 风险侧 / 证伪：① 光模块 MCU 是兆易体内**极小的收入线**，相对存储（60%+）+ 通用 GD32 MCU 的盘子，对正常化 owner earnings 的边际贡献接近噪音，**不改 §9 估值的任何情景基数**；② 兆易自 2018 已在此赛道，本次是迭代而非新增长极，"切入"叙事易被市场放大为概念溢价；③ 核心矛盾仍是"贵"——市值 3,274 亿已把 bull 情景都算进去，再加一个未量化期权不构成买入理由。
- **结论**：**强化"好生意"一侧（GD32 护城河 + AI 期权篮子又厚一分），但完全不动摇"贵 / 安全边际 −78%"的评级算术 —— 维持 watch，买点不变（市值回落 <900 亿 + PB <4x）。** 若后续市场把这条新闻 re-rate 成"光互联概念股"溢价，反而进一步压低安全边际，是看空理由而非看多理由（与 5-31 档对大摩 585 目标价的处理同逻辑）。归入 buffett 档维度②③"产品层期权"篮子跟踪，无需单独升评级。

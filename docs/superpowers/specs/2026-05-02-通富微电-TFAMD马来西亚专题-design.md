## 0. 背景

26Q1 季报点评（`2026-04-29-通富微电-26Q1季报点评.md`）已识别出三项警示：
- OCF/股同比 -35%（同业唯一显著下滑，本期最大警示）
- 资产负债率 5 年单边上行至 60-64%
- 毛利率 13.32% 持平未提升

季报点评对前两项的归因首位是「TFAMD 苏州 + 槟城两厂大额资本开支持续吃现金」「TFAMD 槟城/苏州扩产是封测同业里最激进的资本开支节奏」——但这只是一句话推测。本专题的任务是把这个推测**升级为可量化证据链**，并给出 26Q2 / 26 中报的拐点判定标准。

格式参考：`2026-05-02-沪电股份-泰国工厂专题.md`（同期"专题为季报点评的红灯部分破解"对应物）。

## 2. 范围口径（决策结果）

**口径选择**：方案 B — TFAMD 整体（苏州 + 槟城），但**马来西亚槟城为重点章节**。

理由：
- 26Q1 OCF / 资产负债率 / 应收账款的财务因子在 TFAMD 整体口径才能闭环
- 用户问"马来西亚"= 重点是槟城，但财报披露口径是 TFAMD 合并
- 标题串 `TFAMD马来西亚` 同时体现两个语义

**范围内**：
- 2016 年通富 3.71 亿美元收购 AMD 苏州 + 槟城两厂的资本绑定逻辑
- TFAMD 50/50 合资框架下利润分配 / 关联交易 / 通富的财报并表口径
- 槟城三厂当下扩产细节（产能 / 投产时点 / 客户认证）
- 苏州二厂作为对照（同一资本叙事，但篇幅不超过槟城）
- 26Q1 OCF -35% 红灯的可量化破解
- 25-27 年 capex 节奏 + MI400 量产时利润弹性测算

**范围外**：
- 通富全公司基本面分析（属季报点评范畴）
- 长电 / 华天同业的海外布局（仅在第 6 章对照，不展开）
- AMD 自身基本面（仅作为外部变量）

## 5. 核心论证结构

```
                  TFAMD 苏州+槟城 capex 节奏
                            │
            ┌───────────────┴───────────────┐
            ▼                                ▼
   25-26H1 吃现金阶段                26H2-27 利润弹性阶段
   ────────────────                  ──────────────────
   • 槟城三厂在建工程                 • MI400 26H2 量产
   • 苏州二厂折旧爬坡                 • 槟城三厂转固定资产
   • AMD 应收账款占用                 • 折旧/营收比下降
   • OCF 同比 -35% ⚠️                • OCF/EPS 比率回升 ✅
            │                                │
            └───────────────┬────────────────┘
                            ▼
              26Q2 / 26 中报 / AMD 26Q1 财报
              三个时点的拐点验证标准
                            │
                            ▼
              季报点评评级从「观察」是否升级为「偏积极」
```

**论证链 4 步**：
1. **资本投入证据**（第 3 章）：槟城 + 苏州两厂总投资规模 / 投产时点 / 折旧节奏 → 25-27 年 capex 曲线
2. **现金流吃法测算**（第 4 章）：把 26Q1 OCF -35% 拆成 ① 资本支出（CFI 不在 OCF）② 应收账款扩张（OCF）③ 存货爬坡（OCF）→ 后两项可量化
3. **利润弹性测算**（第 3 章后半）：27 满产时槟城三厂年化营收 / 净利 / 折旧贡献 → 集团 EPS 边际贡献
4. **拐点判定标准**（第 7 章）：26Q2 OCF/股同比 / 26 中报应收账款增速 / AMD 26Q1 数据中心营收 → 三选二达标即可触发评级升级

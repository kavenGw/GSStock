## 4. 正文章节结构（12 节）

沿用仓库 buffett 范式（对标 `胜宏科技-buffett分析.md` / `沃尔核材-buffett分析.md` / `罗博特科-buffett分析.md`）：

1. **结论** — 评级 + 一句话立场，含「05-29 历史对照框」（与旧 watch 的异同；结论若变化须给依据）
2. **能力圈评估** — 采选冶一体化 / 金银副产 / 铜加工 / 金属贸易 四块各一句讲透，点明核心盈利变量（铜价/TC·RC/金银价）全在企业之外
3. **关键假设（决策依赖，待验证）** — 铜价中枢、TC/RC 加工费、自有矿品位与产量、金银副产景气、国资治理与分红、不在高点摊薄式融资/激进并购
4. **业务质量** — 护城河（德兴矿端 + 冶炼规模 ⟷ price-taker 无定价权的辩证）/ 管理层资本配置（国企体制、配置中规中矩）/ 商业模式（教科书 commodity 型）/ 机构强制症预警（冶炼产能扩张与铜精矿争夺）
5. **财务快照** — 沿用并更新 2017–2025 年度时序表（营收/净利/扣非/净利率/毛利率/ROE/每股 OCF/资产负债率）+ 2025 OCF 转负的现金流质量审视（owner earnings 显著低于报表净利）
6. **六维深度（核心）**：
   - **① 市场规模** — 全球精炼铜年需求量级（~2600 万吨级口径，多源交叉）+ 江铜阴极铜 ~230 万吨全球占比、矿产铜 / 金 / 银 / 硫酸副产 / 贸易体量；2026–2030 需求增速与江铜产能规划
   - **② 盈利能力** — ROE 8–9%（从未触 15% 卓越线）/ 净利率 1.4% / 2025 OCF 转负 / 毛利结构拆解（阴极铜 4.1% vs 黄金 4.5% vs 白银 8.3% vs 硫酸 63% vs 贸易 1.1% vs 铜杆 0.8%）；与全球矿企净利率/ROIC 对标，点明"巨额营收稀释出薄净利"的 price-taker + 贸易摊薄本质
   - **③ 全球竞争力** — **内联对标表** vs Codelco / Freeport(FCX) / 嘉能可(Glencore) / 必和必拓(BHP) / Antofagasta / 紫金矿业：营收 · 矿产铜产量 · 自有矿占比 · C1 现金成本分位 · 金银副产 · AI/绿电敞口 · 全球排名；点明江铜"冶炼产能全球第一梯队、矿端自给 ~10%"的真实卡位 —— 是"全球最大铜冶炼+贸易平台之一"而非纯资源巨头
   - **④ 核心优势** — 德兴亚洲最大斑岩铜矿（储量大、品位低 ~0.4%）+ 城门山等一体化、规模成本曲线靠左、金银副产利润缓冲、国资（江西省国资委）稳定、**无海外主权黑天鹅**（对照洛钼刚果金 TFM/KFM 的政策/汇兑敞口）；坦诚自给率有限、定价权为零、现金流薄
   - **⑤ AI 潜力** — **需求 β**：数据中心铜缆 / 母线 / 变压器 / 电网扩容的铜密度增量，量化 AIDC 用铜占全球铜需求比重与 2026–2030 增量斜率，标"量级参考"；**直接敞口**：江铜铜箔（→AI 服务器 PCB / 锂电）、高纯阴极铜 + 铜杆（→AIDC 电力/铜连接）的实际业务卡位与收入占比（公司公告/调研纪要交叉），诚实区分"β 受益"与"主业直接受益"的权重
   - **⑥ 铜价潜力** — 供需平衡推演：供给侧（全球矿端 capex 不足、品位下降、新矿投产稀缺、TC/RC 加工费坍塌信号）vs 需求侧（电动化 / 电网 / AIDC / 光伏风电）；结构性缺口牛/基准/熊三情景 → 铜价中枢区间 → 江铜净利/ROE 弹性敏感度表
7. **估值（重开 · 两套交叉）** — 今日实时价（5-30 `qt.gtimg.cn/q=sh600362`）锚定市值；**主**：周期调整盈利法（7–10 年平均 owner earnings × 周期折扣）+ PB 历史分位 floor（2020 周期底 0.72x 参照）+ NAV 储量重置价值；**辅**：简化 SOTP 四分部（铜冶炼 / 金银副产 / 铜加工 / 贸易）分别估再加总，凸显金银副产 + 高毛利硫酸的隐藏价值；**牛/基准/熊** 三情景反推隐含 PE/PB 交叉验证；给内在价值区间、当前安全边际、建议买入价
8. **卖出标准逐项核对** — 4 项（价格高估 / 护城河破坏 / 管理层诚信 / 更优机会，含与紫金的相对优劣）
9. **关键风险（≤3）** — 铜价周期回落、TC/RC 加工费坍塌、金价回调叠加现金流恶化
10. **监控指标** — 季度跟踪（自给率/TC·RC/单位冶炼成本/金银量价/OCF/负债率）/ 触发升级（铜价与 PB 同步回落至 5 年 20 分位以下、OCF 连两季转正）/ 触发回避（高点摊薄融资/激进扩产、加工费长期亏损线、负债率破 60%）
11. **总体评估** — Buffett 口吻收尾
12. **免责声明** — TAM / 份额 / C1 成本 / 铜价情景 / AIDC 用铜强度等非披露口径统一标注

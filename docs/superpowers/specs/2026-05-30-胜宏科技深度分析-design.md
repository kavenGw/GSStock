## 4. 正文章节结构（12 节）

沿用仓库 buffett 范式（对标 `沃尔核材-buffett分析.md`），把标准「四维深度」扩成「六维深度」：

1. **结论** — 评级 + 一句话立场
2. **能力圈评估** — PCB 制造 / 6 阶 24 层 HDI / 78 层正交背板商业模式一句话讲透
3. **关键假设（决策依赖，待验证）** — 4–5 条：Rubin GPU 板 ~50% 份额守稳、92% 良率接力期可持续、NVIDIA >50% 集中度、惠州四期/泰国一期产能爬坡、非 NVIDIA AI 敞口能否打开
4. **业务质量** — 护城河（92% 良率 + GPU 算力板 ~50% 份额 + 78 层正交背板独家）/ 管理层（诚信尽调声明）/ 商业模式（单客户 franchise 的「借来的护城河」脆弱性）/ 机构强制症预警
5. **财务快照** — ROE / 毛利率 / 净利率 / OCF/EPS 时序表 + worst-case 测试（营收 -30% 偿债）
6. **六维深度（核心）**：
   - **① 市场规模** — AI/数据中心服务器 PCB TAM + 高多层/HDI 细分（多源交叉，量级口径，2026–2029 增速）
   - **② 盈利能力** — 毛利率序 vs 净利率序、OCF/EPS 1.64x、5y 净利 CAGR +52.7%、与全球同业净利率对标
   - **③ 全球竞争力** — **内联对标表** vs 台系（臻鼎/鹏鼎、欣兴 Unimicron、健鼎 Tripod、华通 Compeq）、日系（Ibiden、Shinko）：营收 / 毛利率 / AI 敞口 / 技术层数 / 全球 PCB 营收排名；点明胜宏在 AI-PCB 子赛道的相对位置
   - **④ 核心优势** — 92% 良率工艺壁垒（行业 50–60%）+ 78 层正交背板独家 + 产能扩张卡位（惠州四期 6 阶 HDI、泰国一期）
   - **⑤ Rubin 潜力** — VR200 / Rubin Ultra 时间表（CY Q3 2026 量产 / Q4 爬坡 / Q1 2027 大规模出货）、GPU 算力板单板 ASP、份额可持续性、接力期良率风险、立讯/景旺/鹏鼎切入威胁
   - **⑥ AI 潜力** — 非 NVIDIA 敞口：Google TPU V7/V8（Sokol 大客户）、AMD MI 系列、博通/Marvell ASIC、北美云厂自研芯片；客户多元化进展（NVIDIA 客户分散 ≠ 胜宏客户分散的辩证）
7. **估值（重开 · 多情景）** — 今日实时价锚定（5-30 腾讯 qt.gtimg.cn）；牛 / 基准 / 熊三情景 + Rubin 27/28 续航敏感性表；给内在价值区间、当前安全边际、建议买入价
8. **卖出标准逐项核对** — 4 项（价格高估 / 护城河 / 管理层诚信 / 更优机会）
9. **关键风险（≤3）** — 单一客户、接力期良率、估值透支
10. **监控指标** — 季度跟踪 / 触发买入 / 触发卖出
11. **总体评估** — Buffett 口吻收尾
12. **免责声明** — 份额/良率/ASP 等非披露口径标注

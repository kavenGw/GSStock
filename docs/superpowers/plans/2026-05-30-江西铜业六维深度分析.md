## Task 2: 采集硬数据 → JSON

**Files:**
- Create（临时）: `scripts/_jxcu_fetch.py`
- Output: `.omc/artifacts/jxcu_data.json`

- [ ] **Step 1: 写采集脚本**

脚本顶部加 `sys.path.insert(0, ...)`；实时价直连腾讯 HTTP（不走 create_app）；财务/估值/主营走 akshare。

```python
import sys, json, urllib.request
from pathlib import Path
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))
import akshare as ak

out = {}

# 1) 实时价/PE/PB/市值 —— 腾讯 HTTP 直连（GBK，~分隔）
raw = urllib.request.urlopen("http://qt.gtimg.cn/q=sh600362", timeout=10).read().decode("gbk")
f = raw.split("~")
out["realtime"] = {"name": f[1], "price": f[3], "prev_close": f[4], "change_pct": f[32],
                   "PE_TTM": f[39], "year_high": f[41], "year_low": f[42], "mktcap_yi": f[45], "PB": f[46]}

# 2) 多年财务时序（全市场稳定接口）
fin = ak.stock_financial_abstract_ths(symbol="600362", indicator="按年度")
out["financial_abstract"] = fin.to_dict(orient="records")

# 3) PB/PE 5 年历史分位
out["pb_hist"] = ak.stock_zh_valuation_baidu(symbol="600362", indicator="市净率", period="近5年").to_dict(orient="records")
out["pe_hist"] = ak.stock_zh_valuation_baidu(symbol="600362", indicator="市盈率(TTM)", period="近5年").to_dict(orient="records")

# 4) 主营构成（验毛利结构 + 境内外）
zygc = ak.stock_zygc_em(symbol="SH600362")
out["zygc"] = zygc.to_dict(orient="records")

Path(".omc/artifacts").mkdir(parents=True, exist_ok=True)
Path(".omc/artifacts/jxcu_data.json").write_text(json.dumps(out, ensure_ascii=False, default=str, indent=2), encoding="utf-8")
print("OK rows:", {k: len(v) if isinstance(v, list) else 1 for k, v in out.items()})
```

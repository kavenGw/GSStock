# 估值汇总页 — 市场 Tab 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 给现有估值汇总页加按市场（全部/A股/港股/美股）分组的 Tab，纯前端过滤，并补一列币种。

**Architecture:** 服务端 `index()` 额外算各市场计数传入模板；模板渲染 4 个 Bootstrap `nav-tabs`（带计数）、每行加 `data-market` 属性、加「币种」列；前端 `switchTab(market)` 按 `data-market` 显隐行。刷新逻辑不变（批量全量更新所有行）。

**Tech Stack:** Flask + Jinja2 + Bootstrap5 nav-tabs + 原生 JS；pytest。

@valuations_bp.route('/')
def index():
    rows = load_valuations()
    codes = [r['stock_code'] for r in rows]
    prices = {}
    if codes:
        try:
            prices = unified_stock_data_service.get_realtime_prices(codes)
        except Exception as e:
            logger.warning(f'[估值页] 取实时价失败，降级渲染: {type(e).__name__}: {e}', exc_info=True)
    enriched = _enrich(rows, prices)
    market_counts = Counter(r.get('market') for r in enriched)
    return render_template(
        'valuations.html',
        rows=enriched,
        market_counts=market_counts,
        total=len(enriched),
    )
```

(a) 在 `{% block content %}` 内、`<div class="d-flex ...">` 标题行之后、`{% if not rows %}` 之前，插入 nav-tabs：

function switchTab(ev, market) {
  ev.preventDefault();
  document.querySelectorAll('#market-tabs .nav-link').forEach(a => {
    a.classList.toggle('active', a.dataset.market === market);
  });
  document.querySelectorAll('tr[data-code]').forEach(tr => {
    tr.style.display = (market === 'all' || tr.dataset.market === market) ? '' : 'none';
  });
}
```

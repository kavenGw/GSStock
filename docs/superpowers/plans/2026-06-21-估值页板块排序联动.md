# 估值页板块（组）随列排序联动 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 估值页分组模式下，点 Bear/Base/Bull 列时板块（组）之间也按「组内首行」代表值联动重排，最被低估的板块浮到顶部；评级/日期列仅排组内行。

**Architecture:** 纯前端改动，全部落在 `app/templates/valuations.html` 的内联 JS。init 时捕获服务端渲染的「数量降序」组顺序作回退；改造 `applySort()` 在 margin 列排序态下用每组首个市场匹配行的边际值重排组；`switchMarket` 末尾触发 `applySort()` 让组顺序随市场可见行重算。`app/routes/valuations.py` 与 `valuations.yaml` 不动。

**Tech Stack:** Flask + Jinja2 模板 + 原生 JS（无构建、无 JS 测试框架）；测试用 pytest 对渲染 HTML 做钩子断言，行为正确性靠手动浏览器验证。

function rowMatchesMarket(tr) {
  return currentMarket === 'all' || tr.dataset.market === currentMarket;
}

function groupRepresentative(sortedRows) {
  for (const tr of sortedRows) {
    if (rowMatchesMarket(tr)) return marginOf(tr, sortKey);
  }
  return null;
}

- [ ] **Step 5: 改造 `applySort()` 在分组态下重排组顺序**

把现有整段：
```js
function applySort() {
  updateArrows();
  const tbody = document.querySelector('#val-table tbody');
  if (mode === 'flat') {
    sortRows([...tbody.querySelectorAll('tr[data-code]')]).forEach(r => tbody.appendChild(r));
  } else {
    document.querySelectorAll('#val-table .group-header').forEach(header => {
      const sector = header.dataset.sector;
      const rows = sortRows([...tbody.querySelectorAll('tr.grp-' + CSS.escape(sector) + '[data-code]')]);
      let ref = header;
      rows.forEach(r => { ref.after(r); ref = r; });
    });
  }
}
```
整体替换为：
```js
function applySort() {
  updateArrows();
  const tbody = document.querySelector('#val-table tbody');
  if (mode === 'flat') {
    sortRows([...tbody.querySelectorAll('tr[data-code]')]).forEach(r => tbody.appendChild(r));
    return;
  }
  const headers = [...tbody.querySelectorAll('.group-header')];
  const groupRows = new Map();
  headers.forEach(h => {
    const sector = h.dataset.sector;
    groupRows.set(sector, sortRows([...tbody.querySelectorAll('tr.grp-' + CSS.escape(sector) + '[data-code]')]));
  });
  let order = defaultGroupOrder;
  if (MARGIN_SORT_KEYS.includes(sortKey)) {
    const dir = sortDir === 'asc' ? 1 : -1;
    order = [...groupRows.keys()].sort((sa, sb) => {
      const va = groupRepresentative(groupRows.get(sa));
      const vb = groupRepresentative(groupRows.get(sb));
      if (va === null && vb === null) return 0;
      if (va === null) return 1;
      if (vb === null) return -1;
      return (va - vb) * dir;
    });
  }
  const headerBySector = new Map(headers.map(h => [h.dataset.sector, h]));
  order.forEach(sector => {
    const header = headerBySector.get(sector);
    if (!header) return;
    tbody.appendChild(header);
    (groupRows.get(sector) || []).forEach(r => tbody.appendChild(r));
  });
}
```

5. 含 null 边际的组在 margin 列态下沉到底部。
6. 平铺模式：行按列全局排，无组头（行为不变）。

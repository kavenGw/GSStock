**Goal:** 产出 `docs/stock-analytics/comps/2026-06-25-SiC衬底-GaN器件-晶圆工艺对比-comps.md`，做天岳(02631 SiC衬底) vs 英诺赛科(02577 GaN器件IDM)的晶圆/工艺层横向对比。

**Architecture:** 先 fan-out 联网双源交叉验证两侧最易过期的硬数据（落 .omc/artifacts JSON），再据此 + 三篇现有档已沉淀数据写文档（环节差异前置 → 主参数表 → 4 维度展开 → 工艺裁定收尾），最后 lint + 补反向链 + commit。

## Global Constraints（每个 task 隐含遵守）

- 响应中文；不写多余注释；不写 backup 文件；所有 git/python 命令前加 `rtk`，链式 `&&` 中也要（参考项目 CLAUDE.md）。

- 数据硬/软标注：硬数据标【硬，双源 X+Y】，软数据标【软，单源待证】；联网每条 ≥2 源交叉。

- Windows 编码：写含中文文件用 `encoding='utf-8'`；跑 lint 加 `PYTHONIOENCODING=utf-8`。
- 产物 JSON 落 `.omc/artifacts/`（已 gitignore）；一次性取数脚本任务结束 `rm`，不入库。
- frontmatter comps 必填字段集：`{doc_type, stock_codes, stock_names, themes, period, date}`。
- git：`git add <精确路径> && git commit -F <msgfile>` 同一条命令链；中文多行 message 走文件；提交后 `git show --stat` 确认只含本任务文件。

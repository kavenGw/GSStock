## 二、亮点（4 个超预期）

### 2.1 净利同比 +224% — 封测同业池绝对第一

历史单季净利演变：
```
24Q1 0.98 亿 → 24Q2 2.24 亿 → 24Q3 2.30 亿 → 24Q4 1.25 亿
25Q1 1.01 亿 → 25Q2 3.11 亿 → 25Q3 4.48 亿 → 25Q4 3.58 亿
26Q1 3.29 亿 ← 同比 +224.55% / 环比 -8.14%
```

- **26Q1 单季净利 3.29 亿已超过 24 全年的 6.78 亿的近一半**
- 同业横向对照（26Q1 累计净利）：
  - **通富 3.29 亿** > 长电 2.79 亿 > 华天 0.75 亿
  - 通富净利绝对额 = 长电的 **1.18 倍** + 华天的 **4.39 倍**
- 同比 +224.55% 在三家封测中**遥遥领先**：长电 +37% / 华天 扭亏（基数为负）

**驱动力**：AMD MI350 GPU 出货爬坡 + Zen5 EPYC 服务器 CPU 持续放量 + TFAMD 苏州 / 槟城两厂利用率维持高位。**这是 25Q4 +74% 增速的延续而非衰减**——之前 comps 报告中"AMD 红利可能边际递减"的担忧被反证。

### 2.2 推算销售净利率 4.40% — 同业池第一

- 26Q1 净利 3.29 亿 / 营收 74.82 亿 = **4.40%**
- 同业对照：通富 4.40% > 长电 3.04% > 华天 1.57%
- 25Q1 同比 +2.31pct 是质的飞跃
- 与 25 年累计销售净利率 4.93% 已只差 0.53pct——**单季已逼近年度水平**

### 2.3 ROE 2.10% — 同业池最高

- 26Q1 加权 ROE 2.10% > 长电 1.01% > 华天 0.48%
- vs 25Q1 通富 0.69%——**同比 +1.41pct，提升幅度同业最大**
- 25 全年 ROE 8.08%——按 Q1 节奏年化推算 26 全年 ROE 有望突破 **8.5%**
- **首次接近 Buffett "持续 ROE > 10%" 的及格线**（comps 报告显示通富 6y ROE 均仅 5.5%，2025 才首次突破 8%）

### 2.4 营收 +22.80% — 同业第二（仅次华天 +32%）

历史单季营收（亿元）+ 同比：
```
24Q1 52.82  → 24Q2 110.80(累计) → 24Q3 170.81(累计) → 24Q4 238.82(全年)
25Q1 60.92 (+15.34%) → 25Q2 130.38 (+17.67% 累计) → 25Q3 201.16 (+17.77%) → 25Q4 全年 279.21 (+16.92%)
26Q1 74.82 (+22.80%) ← 单季同比加速
```

- **从 25Q3 同比 +17.77% 持续加速至 26Q1 +22.80%**
- 与长电 26Q1 -1.76% 形成镜像分化——**通富 AMD 红利依旧 / 长电高基数下首次单季营收转负**
- 比华天 +32% 略低，但**通富的营收增长是高质量增长**（净利率 4.40% vs 华天 1.57%），不是"以价换量"

## 三、隐忧（3 个核心警示）

### 3.1 OCF/股同比 -35% — 现金流大幅恶化

- 26Q1 OCF/股 0.6205 元 vs 25Q1 0.9609 元——**同比 -35.4%**
- 同业对照（26Q1 OCF/股 + 同比）：
  - **长电 0.99 元 / +55.4%** ← 同业最强
  - **通富 0.62 元 / -35.4%** ← 同业最差（华天 0.149 元绝对值更低，但同比 -28%）
  - **华天 0.149 元 / -27.9%**
- OCF/EPS 比率从 25Q1 的 14.4x 骤降至 26Q1 的 2.86x——**利润含金量同比折半**
- **可能原因**：
  - TFAMD 苏州 + 槟城两厂大额资本开支持续吃现金（5 年加杠杆 +11pct 趋势延续）
  - AMD 应收账款占用模式延续，营收高增长 → 应收账款同步膨胀
  - 存货 / 在产品周转放缓，封装产能未饱和

**关键观察点**：26Q2 半年报披露 OCF 累计值能否反弹。若上半年累计 OCF 仍同比 -20%+ → AMD 红利"账面利润 / 实际现金流"明显背离 = 重大风险信号。

### 3.2 毛利率 13.32% — 持平未提升

- 26Q1 销售毛利率 13.32% vs 25Q1 13.20% **基本持平**
- vs 25 年累计毛利率 14.59% / 25Q4 估算 ~15.7%——**26Q1 实际下行 1.27pct**
- 同业对照（26Q1）：
  - **长电主营业务利润率 14.22%（环比 +0.34pct，连续 4 季单调上行）**
  - **通富毛利率 13.32%（持平）**
  - **华天主营利润率 10.75%（环比 -1.92pct，骤降）**
- 通富的"成长兑现"主要靠**销量增长 + 三费率下降**（费用率从 25Q1 14.6% 降至 26Q1 ~10%），**而非产品结构升级**
- AMD 大客户的议价权压力仍存——单价持续不能提升 = 长期 ROE 天花板

**关键观察点**：26Q2 / 半年报毛利率能否突破 14.5%。若仍卡在 13-14% 区间 → AMD Zen6 / MI400 谈判中**通富依然是"价格接受者"**。

### 3.3 资产负债率 5 年单边上行 + 26Q1 数据未直接披露

- 25 年累计资产负债率 63.73%（5 年累计 +10.90pct）—— 同业池次高
- 26Q1 业绩报表接口未给出资产负债率数据，需等 4 月 30 日季报全文核实
- 历史趋势：2020 末 52.83% → 2025 末 63.73%——**单边上行从未中断**
- 同期对照：
  - 长电 26Q1 43.01%（环比 -0.63pct，5 年降 -15.5pct）—— 同业池最健康
  - 华天 26Q1 52.21%（环比 +2.03pct）—— 同业池唯一仍在加杠杆
  - 通富 25 末 63.73% / 26Q1 估计仍在 60-64% 区间
- **TFAMD 苏州二厂 + 槟城三厂同时扩产**——封测同业里最激进的资本开支节奏

**关键观察点**：26Q1 全文披露的资产负债率若仍超 63% → 加杠杆趋势未止；若环比降至 60-62% → 首次出现内生现金流偿债迹象（与 OCF -35% 矛盾，故概率低）。

### 5.2 商业模式定性

- **类型**：**强绑定特许经营 + 商品业务混合体**
- **关键特征**：
  - TFAMD 合资公司股权（通富 50% / AMD 50%）= 关系深度绑定，不是普通客户
  - 但合资公司财务并表 + AMD 给技术 / 通富承担 capex = **通富承担"代工厂"的所有重资产风险**
  - 利润分配机制不透明——AMD 议价权强（毛利率长期卡在 14% 附近 = 反证）
- **Buffett 视角**：
  - 比纯商品代工（如华天）好——有客户绑定优势
  - 比真正特许经营（如台积电）差——没有不可替代的工艺壁垒
  - "AMD 代理标的" 这个标签 = **既是优势（共享 AMD 数据中心红利）又是风险（AMD 任意一个产品周期下行都会强烈传导）**

### 5.3 institutional imperative 警示

5 年观察到的征兆（vs 长电 / 华天）：

- **资产负债率 5 年单边上行 +10.9pct**——TFAMD 槟城 / 苏州两厂同时扩产，封测同业最激进 capex
- **5y 营收 CAGR +21% / 净利 CAGR +22.9%**—— 营收 vs 净利同步（同业唯一健康背离 -1.9pct）
- **2023 净利曾从 5.62 亿暴跌至 1.67 亿（-70%）**——AMD 周期传导的强证据
- 26Q1 数据**部分缓解**这一警示：
  - ✅ 净利 +224% > 营收 +22.8% = 效益跟得上扩张
  - ⚠️ 但 OCF/股 -35% = 现金维度的扩张成本未计入利润口径
  - **"利润扩张 / 现金流缩减" 的剪刀差** = 典型 AMD 商业模式特征（账面应收账款 = 实际未收回的现金）

### 6.2 投资逻辑

**做多逻辑（强化）**：
1. **AMD 兑现强度超预期**：26Q1 净利 +224% 明显优于 25Q4 +74% 节奏，没有"边际递减"迹象

4. **AMD MI400 + Zen6 路线图**：26H2 + 27 年仍是数据中心 GPU/CPU 出货爆发期

## 八、Buffett 视角整体评估

> 用 Buffett 的口吻：

"通富微电这家公司，本质上是 AMD 在中国大陆的封测分身。26Q1 的数据非常漂亮——净利同比 +224% 是封测行业里最猛的，我必须承认这是 AMD Zen5 + MI350 兑现期的真实证据，没有水分。但是当我把净利的 3.29 亿 和现金流的 -35% 放在一起看时，问题就出现了：账面上多赚的 2.28 亿，并没有变成真正的现金。

这种'账面利润高 / 现金流低' 的模式，在历史上 Berkshire 处理过很多次——它通常意味着两件事：第一，应收账款在膨胀（客户给你订单但回款慢）；第二，在产品 / 存货占用资本（产能没饱和但要支付折旧 / 工资 / 物料）。两种情况都不是问题本身，问题是**它们能持续多久**。

我不喜欢的是通富的客户结构——AMD 一家占 60%+，TFAMD 合资公司一半的股权在通富手上但 AMD 给的是技术不是钱。这意味着每一笔扩产投资（苏州二厂、槟城三厂）都是通富自己的负债——而 AMD 享受'轻资产' 模式 + 拿走利润的 50% 分配权。这就是为什么 5 年来通富资产负债率从 53% 升到 64%，而 AMD 自己资产负债率才 32%——**是通富在替 AMD 持有重资产**。

这不是说通富不能买。AMD 上行周期还有 1-2 年，通富的 EPS 大概率能从 0.80 涨到 1.55-1.80，PE 从 60 倍降到 30 倍出头是合理的估值修复。但是这种持有，需要每个季度都盯紧两个指标：① OCF 是否同比恢复（至少不是 -35% 的级别）② AMD 自己的数据中心业务指引是否还在加速。任何一个出问题，通富的'PB 4.80 倍 / 5 年 88% 分位' 估值会很快杀回 50% 分位以下，对应 PB 杀至 2.74x = -43% 下跌空间。

对比之下，长电同业期同样的报表里，营收虽然 -2% 但利润 +37%、现金流 +55%、负债率降到 43%——**这才是真正符合 Munger 标准的'高质量生意'**。我宁愿付一点估值 premium 买长电（PB 2.81 vs 通富 4.80），也不愿因为'通富成长性更强'就主动加仓。

**结论：观察。仓位建议 2-3% 试探，不主动加。等 26Q2 半年报 OCF 拐点验证 + AMD 26H2 指引出来后再决定是否升级到买入。如果 OCF 26Q2 仍 -25%+，则永久回避——再高的成长性也救不了一家'账面利润漂亮但现金不进来'的公司。**"

## 九、标准输出（Buffett 框架）

### Conclusion
**观察 → 偏积极（仓位 2-3% 试探，不加仓）**。26Q1 净利 +224% / ROE 2.10% / 净利率 4.40% 同业第一兑现 AMD 红利，但 OCF/股同比 -35% / 毛利率持平 / AMD 单点依赖未变 → 不到买入信号。

### Circle of Competence Assessment
**圈内**——AMD 服务器 CPU + 数据中心 GPU 封测代工业务模式清晰，财务可分析，AMD 季报作为先行指标可观察。

### Key Assumptions
1. AMD 26 全年数据中心业务收入同比 +25-40%（MI300X 持续 + MI350 26H1 出货 + MI400 26H2 爬坡），通富作为大陆唯一规模量产 OSAT 享受全部红利
2. TFAMD 苏州二厂 / 槟城三厂 26 年利用率从 25 年 ~75% 提升至 ~85%，但 capex 维持高位
3. 26Q1 OCF -35% 主要来自 AMD 应收账款季节性 + 存货爬坡，而非结构性恶化（半年报应回升至 -10% 以内）
4. 通富对 AMD 议价权未实质提升 → 26 全年毛利率 14-15% 区间，未能突破 16%
5. 26 全年 EPS 落在 1.45-1.67 元（中性 1.55），ROE 14-16%，对应 PE 31-34x、PB 4.7x

### Business Quality
- **护城河**：中等 + 稳定（TFAMD 合资股权 + 大陆唯一服务器 CPU 大尺寸 FCBGA 量产基地，但全球可被日月光 / 安靠替代）
- **管理层**：诚信中性 / **资本配置激进**（5 年加杠杆 +11pct 投 TFAMD 两厂）/ owner mentality 中性
- **商业模式**：**AMD 强绑定特许经营 + 重资产代工** 混合体（不是真正特许经营，但有客户深度绑定优势）
- **institutional imperative 警示**：⚠️ **部分存在**——资本开支激进 + AMD 单点依赖 + OCF 同比 -35%，但**净利兑现度强（+224%）暂时缓解警示**

### Financial Snapshot
- **ROE 26Q1 加权**：2.10%（同业最高）/ **6y 均值仍仅 5.5%**（2025 年起拐头突破 8%）
- **现金转化率**：26Q1 OCF/EPS = 2.86x（vs 25Q1 14.4x），**单季腰斩**——账面利润含金量同比恶化
- **债务安全**：26Q1 资产负债率 60-64%（待确认），中等抗压
- **Owner earnings 估算**：26Q1 净利 3.29 亿 + D&A（按 25 年节奏 ~5 亿/季）- 维持性 capex（按 25 年节奏 ~6 亿/季）= **~2.3 亿/季 owner earnings**（年化 9-10 亿，约为净利的 70%——合理水平）

### Valuation
- **内在价值区间**：按"AMD 周期中位 EPS 1.55 元 × PE 28-35x" = **43-54 元/股**（中位 48 元）
- **当前价格**：49.50 元（4-29 收盘）
- **当前安全边际**：**约 0% 至 -3%**（接近内在价值中位）→ 对应**中等**确定性
- **建议买入价**：< 41 元（对应 PB 4.0x、市值 ~625 亿）+ **同时**26Q2 OCF/股同比 -10% 以内
- **重仓买入价**：< 31 元（对应 PB 3.0x、市值 ~470 亿）+ **同时**半年报 OCF 累计同比转正 + 毛利率突破 14.5%

### Sell Criteria — Item-by-Item Check
1. **价格严重高估？** **部分**——当前 PB 5y 88-90% 偏贵 + PE-TTM 52x 仍偏高，但 26 隐含 PE 仅 32x → 综合**中性偏微高估**
2. **护城河被破坏？** **否**——AMD 关系稳固 + 大陆唯一规模量产 OSAT
3. **管理层诚信问题？** **否**——无重大瑕疵
4. **有显著更优机会？** **是**——长电同业中财务安全性 / 现金流 / 估值合理性全面更优；晶方 CIS 反转 + 净现金 + 估值历史低位

### Key Risks (max 3)
1. **AMD 周期风险**：任意季度数据中心业务指引下修都会强烈传导，2023 净利曾跌 70% 是历史先例
2. **OCF / 净利剪刀差扩大**：26Q1 OCF/股 -35% 的可持续性 + AMD 应收账款占用模式
3. **重资产负债率 + capex 持续**：60-64% 杠杆 + TFAMD 槟城 / 苏州扩产持续吃现金

### Monitoring Indicators
- **季度跟踪**：① OCF/股同比拐点（最关键）② 毛利率能否突破 14.5% ③ 资产负债率能否首次环比下降 ④ AMD 数据中心业务指引（5 月初 AMD 26Q1 财报）⑤ TFAMD 利用率披露
- **触发"加仓"信号**：PB < 4.0x **AND** 26Q2 OCF/股同比 -10% 以内 **AND** 毛利率守住 14%+
- **触发"减仓 / 卖出"信号**：AMD 任意季度数据中心业务指引下修 / 26Q2 OCF/股同比仍 -25%+ / 单季毛利率跌破 12% / 资产负债率突破 67% / PB > 6.5x

### Overall Assessment
**观察（偏积极但不买入）**。26Q1 的"净利 +224% / ROE 2.10% / 净利率 4.40%" 三项同业第一确认 AMD 兑现期延续而非衰减，估值随 EPS 摊薄从 PE 62.6x 降至 52x、隐含 26 PE 仅 32x —— 业绩兑现"消化"了 PB 5y 91% 偏贵的溢价。但 OCF/股同比 -35% 同业唯一显著下滑、毛利率持平未提升、AMD 单点依赖结构未变 —— 这些是不可忽视的警示。**仓位建议 2-3% 试探建仓，不主动加；等 26Q2 OCF 拐点 + AMD 26H2 指引验证后，再决定升级至 "买入" 或维持 "观察"**。同业偏好顺序仍是 **长电 ≥ 通富 > 晶方 >>> 华天 ≈ 甬矽**，长电 + 通富 5:5 或 6:4 双配置依然是封测板块最优解。

# 共创草坪（605099.SH）—— 巴菲特框架分析

> 分析日期：2026-04-24
> 框架：8 问快筛 + 护城河 / 管理层 / 财务 / 估值 / 风险 / 行业
> 说明：财务数据为基于公开已知信息的定性判断，具体数字请以最新年报校准。

---

## Conclusion
**Keep Watching（不买）** —— 人造草坪是"品牌认证 + 成本+规模"的准商品生意，全球龙头地位真实，但 80%+ 外销让公司同时背负地缘、关税、海运、汇率四重外生变量，护城河等级达不到巴菲特"特许经营"的门槛，除非跌到深度折价，否则机会成本太高。

## Circle of Competence Assessment
**边界区**。业务本身极其简单（把塑料纤维织成草、卖给球场 / 景观工程），人人都能看懂；但决定长期回报的关键变量——**欧美反倾销 / 对等关税 / 国别贸易政策**——不在普通投资者的能力圈内，也不在管理层的掌控内。看得懂产品，看不准政策。

## Key Assumptions（3–5）
1. 全球运动草坪需求随 FIFA / FIH 认证球场更新周期稳定增长（~5%）。
2. 中国制造成本优势在 5 年内不会被东南亚 / 土耳其产能系统性取代。
3. 美国对华 PET 人造草丝的反倾销 / 反补贴税不进一步加码，否则对美出口基本归零。
4. 主要原材料（PE / PP / PET 切片）价格围绕油价温和波动，不出现 2021 年级别的单边上行。
5. 公司维持"产能扩张 → 出口为主"的资本开支节奏，不发生跨界并购。

## Business Quality
- **Moat：** 品牌/认证（FIFA Preferred Producer）+ 成本规模 —— **中等偏弱**，**稳定**。认证是准入门槛但非唯一门槛（全球有 10+ 家获认证）；成本优势真实但可被关税/汇率抹平。**不属于巴菲特定义的"特许经营"**（客户有替代品、且受政府关税"类价格监管"影响）。
- **Management：** 公开信息看整体诚信评级**合格**（未见财务造假、重大关联交易争议）；资本配置**中规中矩**（上市后持续产能扩张、现金分红稳定），但缺乏"国家队 + 品牌延伸"的想象力；owner mentality **中等**（实控人持股集中，但未见明显 owner-operator 的资本纪律亮点）。
- **Business model：** **Hybrid**（介于 franchise 与 commodity 之间），更偏 commodity-plus。
- **Institutional imperative warning：** 暂不明显——公司没有频繁并购或激进扩张的迹象，但要警惕行业下行时"继续扩产保市占"的路径依赖。

## Financial Snapshot（需以最新年报校准）
- **ROIC（10 年均值）：** 上市时间较短，可得年度有限；近年口径约 10–15%，峰值年（2020、2023）可达 18–20%+，呈周期性波动。**未稳定达到 >15% 的标准**。
- **Cash conversion：** 经营现金流波动较大（受应收账款和海运周期影响），部分年份 OCF/净利 < 80%，非优秀级。
- **Debt safety：** 资产负债率低（长期 <30%），现金充裕，**-30% 营收压力测试可过**。这是最大的安全项。
- **Owner earnings 估算：** 资本开支持续高于折旧（扩产期），owner earnings 明显低于报表净利，需在估值时下调 15–25%。

## Valuation
- **Intrinsic value range：** 以 "normalized owner earnings ≈ 报表净利 × 0.8" × 10–12x（商品属性明显 + 增长 3–5%）估算。具体数字随当期净利变动，请以最新 TTM 净利自行代入。
- **Current margin of safety：** 如果当前 P/E 在 15x 以上，几乎没有安全边际；低于 10x 才进入讨论区间。对应**中等确定性**，至少需要 35–40% 折价。
- **Recommended buy price：** 约等于 "normalized owner earnings × 8–10x"。宁愿等行业出口压力峰值时买，也不要在出口红利期追。

## Sell Criteria —— 持有者自查
1. **价格严重高估？** 若 P/E > 20x 且净利并非周期底部，**是**。
2. **护城河基本面破坏？** 美国关税大幅加码、FIFA 认证级别下降或大客户长期流失——目前**否**，但需每季监测。
3. **管理层诚信问题？** 目前**否**。出现即立即卖。
4. **存在明显更好的机会？** 在 A 股出口链里，确实有 ROIC 更稳定（如消费类龙头）或政策风险更低的标的；**视组合而定**。

## Key Risks（仅列最关键 3 条）
1. **美国 / 欧盟贸易壁垒加码**：这是唯一可以一夜之间把北美业务打掉 30%+ 的风险，且公司无法控制。
2. **原材料 + 海运双杀重演**：2021–2022 是前车之鉴，毛利率可以从 30%+ 压到 15% 以下。
3. **人造草坪替代需求周期**：球场更新周期 8–10 年，若全球体育基建投资放缓（欧洲财政 + 中国校园草坪饱和），增长假设会被打穿。

## Monitoring Indicators
- **每季看：** 出口占比结构（欧美/其他）、美国反倾销 / 关税政策公告、毛利率、经营现金流 / 净利比、海运 BDI 指数。
- **Sell 触发信号：**
  1. 美国加征反倾销税 > 30% 且无迁厂对冲；
  2. 连续两年 ROIC < 8%；
  3. 主要客户（大型体育工程承包商）合同流失；
  4. 管理层大比例减持。

## Overall Assessment
> "这是一家**好公司、但不是伟大公司**。它能在一个细分市场做到全球第一，管理层诚实、财务干净、负债低——这些都很难得。但它卖的产品太接近商品，它的利润被太平洋那头的关税部长一句话就能改写。**我会把它放进'看着'文件夹**，等哪天市场因为一条关税新闻把它砸到 8 倍市盈率以下、而基本面并未真实恶化时，再认真看一眼。现在这个价位？我宁可拿现金、等下一个 See's Candies 式的机会，也不会在一家必须持续祈祷海运费不涨、美国总统不皱眉的生意上押注十年。"

—— 以巴菲特口吻给出的结论。落到 A 股出口制造业，几乎都适用这个"好但不够伟大"的判词。

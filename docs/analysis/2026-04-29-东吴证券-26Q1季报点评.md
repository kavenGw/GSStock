## 二、Buffett 视角拆解

### 2.1 这是什么类型的生意？

**核心定性**：东吴证券是**典型的中型地方综合券商**，业务结构 = 经纪（受益于市场成交量）+ 投行（IPO/再融资节奏）+ 自营（市场 beta）+ 资管（规模效应）+ 信用（两融）。

四个业务**全部高度顺周期、全部缺乏定价权**——

- **经纪**：佣金率行业性长期下行（万 2.5 → 万 1 以下），客户基本无忠诚度
- **投行**：IPO 节奏被监管节流（2024 全年 IPO 大幅收缩、2025 边际恢复但未回到 2021 水平），中型券商受冲击大于头部
- **自营**：是 25 年净利暴增的**最主要原因**——大牛市自营盘浮盈兑现；但 26Q1 市场转入震荡，自营贡献立刻塌缩
- **资管**：东吴资管规模在行业第二梯队，缺乏头部规模效应

> **Buffett 关键判断**：「**若你不能在熊市中维持盈利，那你的牛市利润只是借用的、不是赚来的。**」东吴 25 全年 35.52 亿净利中，**自营/投资业务贡献比例估计在 50% 以上**——这部分利润随市场情绪而生灭，不构成「owner earnings」。

### 2.2 护城河评估（参考 references/03-business-moat.md）

| 维度 | 评价 |
|------|------|
| 无形资产（品牌） | **弱**——苏南区域品牌，无全国性认知 |
| 成本优势 | **无**——人均 cost 显著高于头部（员工费用占比） |
| 转换成本 | **极弱**——客户切换券商成本接近于零 |
| 网络效应 | **无**——非平台型业务 |
| 高效规模 | **无**——头部 5 家占行业 50%+ 利润，地方券商无规模 moat |

**护城河趋势**：**结构性收窄**——

- 行业并购整合加速：国君 + 海通已合并完成（2024Q4），头部券商规模 / 牌照 / 资本金优势进一步拉大
- 互联网券商（华泰张乐 / 东方财富）继续吞蚀经纪份额，地方券商首当其冲
- 投行业务被「三中一华一国」（中信 / 中金 / 中信建投 / 华泰 / 国君）虹吸 70%+ 项目

> **结论**：东吴证券处于**护城河持续被侵蚀的中游位置**——既没有头部的规模 / 牌照 / 客户质量护城河，也没有互联网券商的成本结构优势。**这正是 Buffett 在《1990 年致股东信》中描绘的「不上不下的 commodity 生意」**。

### 2.4 现金流"漂亮"的含金量分析

| 指标 | 26Q1 | 评估 |
|------|------|------|
| 每股 EPS | 0.16 | — |
| 每股经营现金流 | 1.90 | OCF/EPS ≈ 11.9x |
| 表面解读 | "11.9 倍现金流验真" | ❌ 错误 |
| 实际含义 | 券商 OCF 主要由客户保证金、自营资金流动构成，与 EPS 没有实业意义上的"质量"对应 | — |

> **关键警示**：**不要把券商的"经营现金流"按实业逻辑解读**——
>
> - 客户保证金存款增加 → 经营现金流入（但这不是公司赚到的钱）
> - 自营投资资金腾挪 → 体现在经营现金流（但不构成 owner earnings）
> - 真正衡量券商盈利质量的是**净利润 vs 净资产收益率**——东吴单季 ROE 1.83% 折年化 7.3%，**低于 10% 这个金融业最低门槛**
>
> 26Q1 现金流 EPS 1.90 元的"漂亮"是会计性的、不是商业性的。

### 2.5 估值再测算

**情景 A（中性）—— 26 全年线性外推**：
- 假设 Q2-Q4 与 25 同期持平（不再衰退）：26 全年净利 = 8.01 + (35.52-9.80) = **33.73 亿**
- 给中型券商合理 PE 10-12x：合理市值 = **337-405 亿**
- 当前市值 ~403 亿 → **几乎处于公允价上沿**

**情景 B（乐观）—— 26H2 市场再启动**：
- 假设 26 全年净利 38 亿（Q2-Q4 比 25 同期还小幅增长）
- 给 PE 12-13x：合理市值 = **456-494 亿**
- 当前 403 亿 → 上行空间 13-22%

**情景 C（悲观）—— 26 全年继续负增长 10%**：
- 26 全年净利 31.97 亿
- 给 PE 9-10x（熊市估值）：合理市值 = **288-320 亿**
- 当前 403 亿 → **下行空间 20-29%**

**情景 D（PB 视角，券商更适用）**：
- 净资产规模约 437 亿（每股 8.80 × 50 亿股）
- 中型券商 PB 合理区间 0.7-1.2x → 合理市值 = **306-524 亿**（中位 415 亿）
- 当前 403 亿 ≈ PB 0.92x → **几乎卡在中位**

**新的内在价值区间：300-500 亿**（中位约 380-420 亿）

**当前市值 ~403 亿 —— 安全边际接近 0、向下不对称风险显著大于向上空间**

> **关键结论**：表面「PE 12x、PB 破净」很便宜，但**对中型券商在头部并购加速 + 业务下行期，这就是合理估值，不是低估**。Graham 烟蒂式投资也要求「公允价至少 50%+ 的折扣」——东吴目前**没有任何折扣**。

## 五、关键风险

### 风险一：26 全年净利可能继续下滑
- 25 全年的高增长（+50.12%）核心来自 25Q2-Q4 的市场大涨
- 26Q1 已确认「失去自营市场 beta 后真实经营能力薄弱」
- 若 26Q2 市场继续震荡，营收 / 净利可能持续 YoY 负增长

### 风险二：行业并购整合加速
- 国君 + 海通合并完成后，第二梯队压力倍增
- 监管层鼓励头部券商并购整合趋势明确
- 中型券商面临「被并购」或「被边缘化」的二元选择，无论哪种都对小股东不利

### 风险三：资产负债率突破历史高位
- 26Q1 资产负债率 81.65%，是过去 5 年高位区
- 券商业务高度依赖回购融资 / 短融 / 同业拆借——资金成本若上行将进一步压缩利差

### 风险四：投行业务持续收缩
- 东吴投行项目储备质量在中型券商中居中
- 监管对中小券商投行风控持续加严
- 历史教训：参考 25Q1 同业另一家中型券商（光大）IPO 失败案例后净利波动

## 七、整体评价

> "东吴证券 26Q1 给我两个清晰的判决：
>
> **第一**，所谓的「便宜」是个错觉——PE 12x、PB 0.92x 听起来像 Graham 的烟蒂，但对一个**护城河持续被侵蚀的中型金融商品业务**，这只是公允价、不是折扣价。烟蒂投资的关键是「**用 50 美分买 1 美元的资产**」——东吴现在是「用 92 美分买 1 美元的账面资产，且这 1 美元资产的内在价值正在缓慢蒸发」。
>
> **第二**，**与其支付中型券商的便宜价格，不如支付头部券商的合理价格**——这正是 Munger 那句『**支付较高价格买入伟大企业，远胜于以便宜价格买入平庸企业**』。中信、广发 26Q1 同比 +54%、+70%，估值同样在 PE 10-12x；东吴 -18% 估值也是 PE 12x——**机会成本极不对称**。
>
> 我会同样的话再说一遍：**24Q1 净利 4.56 亿，24 全年 23.66 亿；25Q1 净利 9.80 亿，25 全年 35.52 亿**——线性外推 26 全年净利的话，按 Q1 占比 22-25%（历史值）反推，26 全年合理预期是 32-36 亿，**几乎不增长**。这意味着 25 年的暴增是一次性 beta，不是趋势。
>
> 更重要的是，**当中信、广发作为同行业更优秀的标的，估值与东吴几乎没有 gap 时，理性的 capital allocator 不会持有东吴**。我对券商这个行业的整体观点没有改变——**这是一个 commodity-cyclical、监管加严、头部集中度持续提升的行业**——但即便要在这个行业里分配资本，**也只配头部、不配中游**。
>
> 当日股价 -4.05% 是市场即时给出的判决。如果今天这个公司给的不是 26Q1 季报而是私下询价收购 80% 股权，我开出的价格是 **PB 0.6-0.7x，对应市值 260-310 亿**——比当前还要打 25-35% 的折扣。这就是 Buffett 视角下"私人买家测试"的真实答案。"

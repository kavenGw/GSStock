## 结论

**维持「观察」评级，向上微调为「观察（同业第一兑现）」** —— 26Q1 兑现度在 PCB 板块同期已发布的 AI 链 4 家中**质量最高**：营收同比 +53.91%、净利同比 +62.90% 的增速**比 2025 全年 +42% / +48% 还在加速**（与胜宏 +28% / +40% "加速度结束"形成镜像）；毛利率 35.63% 紧贴 2025 全年 35.48% 的 5y 历史高位、与 25Q4 35.66% 几乎持平 ⇒ **AI 高多层 PCB 议价能力结构性确立**。但有一个必须警示的红灯：**26Q1 每股经营现金流仅 0.27 元、OCF/EPS = 0.42x**——24Q1 / 25Q1 同期均为 1.7-1.9x，本季现金流断崖式偏离了历史季节性。这是利润已兑现但**应收账款 / 存货扩张吃掉了现金**的信号，必须在中报披露后重点交叉验证。**估值维度：PE-TTM 45.88x（5y 88% 分位）相对胜宏 71x、深南 60x、鹏鼎 48x 已是 PCB AI 龙头里"最不贵"的一家；但 PB 11.76x 仍是 5y 100% 极值**——典型的"利润兑现速度跑赢估值，但 PB 已透支"双重定价。仍不构成买入安全边际，但与胜宏比较是 PCB 板块**质量最优 + 估值偏合理**的一只。

### 1.2 OCF/EPS 异常拆解（最大警示信号）

| 报告期 | 每股 OCF | 每股 EPS | OCF/EPS | 健康度 |
|--------|---------|---------|---------|--------|

| **26Q1** | **0.27** | **0.6455** | **0.42x** ❌ | **同期断崖** |

> **Buffett 视角**：「**利润不等于现金**——这是最古老但最常被新投资者忽视的一条戒律。当一个公司单季度利润同比 +63% 但经营现金流同比 -63%，要么是会计在为利润让路，要么是应收账款 / 存货在替客户当银行。沪电此前 3 年同期 OCF/EPS 都在 1.7-3.1x 之间——26Q1 的 0.42x **不是季节性**，是结构性扩张吃掉了现金。」
>
> 三个可能解释，必须在 26 中报通过应收 / 存货 / 预付明细交叉验证：
> 1. **AI 大客户回款放缓**（NVIDIA / 博通账期被动延长）—— 不太可能，胜宏同期 OCF/EPS 1.64x 充分证明 NVIDIA 信用极好
> 2. **沪电主动备货 GB300 / Rubin 接力期**—— 存货大幅扩张，是合理但有风险的扩张行为
> 3. **应收账款扩张 + 政府补贴 / 应付款季节性集中支付**—— 短期波动，下季可恢复
>
> **结论**：在没有看到中报应收 / 存货拆分前，**不能将"OCF/EPS 突然降至 0.42x" 解释为正常**——这是本季最重要的"待证伪"问号。

## 二、Buffett 视角拆解

### 2.1 这是商业本质改善还是周期顶部维持？

**关键判断**：**70% 是 AI 服务器高多层 PCB 结构性需求兑现**（NVIDIA Hopper → Blackwell GB200 → Rubin 三代接力 + 博通 / Marvell ASIC 路线 + 国内昇腾 / 海光配套），**20% 是云数通设备升级（800G 交换机渗透）**，**10% 是行业景气共振**。

**支持"结构性而非脉冲"的证据**（重要 + 可验证）：
- **客户结构远比胜宏分散**：NVIDIA / 博通 / Marvell / Cisco / Arista / 国内昇腾产业链——任何单一客户占比预计 <30%（4-27 comps 推断），抗周期性显著强于胜宏（NVIDIA 占比 >50%）
- **毛利率 35.63% 持续 5 个季度站稳 35%+ 平台**（25Q2 37.31% / 25Q3 35.84% / 25Q4 35.66% / 26Q1 35.63%）—— 不是单季异常，是**新平台**
- **5y ROE 6 年均值 21.2%（同业最高）**—— 4-27 comps 已确认，26Q1 单季 7.81% 年化对应 ~31%，与 25 全年 28.57% 完全可比，**ROE 加速通道未结束**
- 26Q1 营收占全年 32.8%（25Q1 仅 21.3%）—— 季节性比例的跃升 = **下游订单从 Q2 / Q3 高点切换回 Q1 大量备货**，符合 GB200 持续放量 + GB300 / Rubin 提前备货的逻辑

**支持"加速度可能仍有顶部"的证据**（同样重要）：
- 营收占全年 32.8% 这个比例本身意味着**26 全年增速可能不能完全简单线性外推**（如果 Q2-Q4 各占 22-23% 即可，那全年净利 ~50 亿，YoY +30% 而非 +63%）
- 资产负债率 48.64%（22 年末 33.87% 加 +14.77pct）= 扩产融资压力同时上升
- **OCF/EPS 0.42x ⚠️**（本季最大警示，详见 1.2）—— 利润含金量出现单季断层
- PCB 高多层赛道竞争对手深南、生益（CCL 上游）也在扩产，**ASP 维持的"相对竞争结构"在 27-28 年存在被冲击的可能**

**结论**：**这是"AI PCB 结构性红利继续兑现 + 季节性现金流偏离"的复合状态**——业绩兑现质量本身极高，但现金流质量存疑。**不是简单"周期顶部维持"**（与胜宏不同），但也不是无瑕疵兑现。

### 2.3 估值再测算

**情景 A（中性）—— 26Q1 节奏外推 + 季节性正常化**：
- 沪电过往季节性：Q1 通常占全年 19-21%（25Q1 21.3% / 24Q1 19.4%）
- 26Q1 占比飙升至 32.8% 是**异常季节性**——若解释为"Q2-Q4 仍按历史节奏增长 / 不再线性放大"：
  - 按 26Q1 占比 25%（保守，承认 Q1 异常拉升）：26 全年营收 ~248 亿（YoY +31%），净利 ~50 亿（YoY +30%）
  - 按 26Q1 占比 28%（中性）：26 全年营收 ~222 亿，净利 ~44 亿（YoY +15%）
- 给沪电（多客户、ROE 21% 同业最优）合理 PE = 25-30x
- **公允市值区间：1,250-1,500 亿**

**情景 B（乐观）—— Q2-Q4 维持单季 12 亿+ 净利节奏**：
- 假设全年净利 55-60 亿（YoY +44-57%），27 年维持 60 亿，28 回归 50 亿
- 10% 折现 5 年 + 终值（50 × 14x 折现）= **1,800-2,200 亿**

**情景 C（悲观）—— OCF/EPS 0.42x 暴露应收 / 存货问题，下半年消化拖累毛利率**：
- 假设全年净利 40 亿（毛利率从 35.6% 退到 31%，营收 230 亿）
- 给 PE 18x = **720 亿**

**新的内在价值区间：720-2,200 亿**（中位约 1,300-1,500 亿）

**当前市值 1,972 亿（按 102.57 元）—— 安全边际**：
- 对中性情景中位（1,400 亿）：**负 41%**
- 对乐观情景上限（2,200 亿）：**正 12%**（仍存在但极薄）
- 对悲观情景（720 亿）：**负 174%**

> **关键结论**：与胜宏"全场景下都是负安全边际"不同，沪电 1,972 亿市值在 Buffett 视角下**不是绝对的"严重高估"**——而是处于"乐观假设下尚有安全边际、中性假设下需折扣 30%、悲观假设下需折扣 60%"的中间地带。这正是为什么"维持观察"而非"明确不买"——但**也绝不是买点**。

## 四、关键风险（更新）

### 风险一：OCF/EPS 0.42x 的"应收账款黑洞"风险（**新增本季最大风险**）
- 24Q1 / 25Q1 同期 OCF/EPS 都在 1.7-1.9x，26Q1 直接降至 0.42x 是**断崖式偏离**
- 三种解释中**最危险的是"客户被动延长账期 + 沪电主动备货过激"叠加**
- 触发情景：26 中报披露应收账款 > 80 亿（25Q1 / 25Q2 合计应收 ~50 亿）+ 存货 > 35 亿
- 风险量级：若验证为质量问题，**估值杀伤可能 25-30%**（PE 从 45 → 32，市值 ~1,400 亿）

### 风险二：26Q1 营收占全年 32.8% 的"季节性塌方"风险
- 历史 Q1 通常占年 19-21%，本季占比异常拉升
- 解释 1：客户加速备货（GB300 / Rubin 接力前置）—— 利好但意味着 Q2-Q3 可能空窗
- 解释 2：客户提前提货 + Q3 / Q4 增速放缓 —— 全年增速远低于线性外推
- 触发情景：Q2 单季营收同比 < +20%（vs 25Q2 44.56 亿）
- 风险量级：估值杀伤 15-20%

### 风险三：杠杆持续抬升（48.64%）+ AI PCB 资本开支大年
- 5y 资产负债率 +14.77pct
- 26 年继续扩产 → 折旧 / 利息费用进入压力测试期
- 任何 ROE 下行（如从 28% 降至 22%）都会触发估值切换

### 风险四：NVIDIA 26H2 需求 / Rubin 量产时点
- 与胜宏同样暴露
- 沪电相对优势：客户分散使其抗冲击能力较强（但仍非免疫）
- 触发情景：NVIDIA 26Q2 / Q3 财报 capex 指引环比下调 >10%

## 六、整体评价

> 「26Q1 数据放在桌上，沪电是给我惊喜更多的一家——比胜宏惊喜，比 4-27 我自己写的 comps 估计更惊喜。
>
> **它做对了三件事**：第一，营收 +54% / 净利 +63% 同比比 25 全年 +42% / +48% 还在加速——这种'高基数后再加速'在任何行业都是稀有的'多客户结构性红利' 的标志，不是单一客户红利能解释的。第二，毛利率连续 5 个季度站稳 35%+——这意味着 AI 高多层 PCB 已经从'周期红利'切换到'结构性议价'。第三，单季 ROE 年化 ~31%——与 25 全年 28.57% 完全无缝衔接，没有任何降速迹象。
>
> **但它做错了一件事，而且这件事本身可能就是'季节性'，但绝不可掉以轻心**：单季每股经营现金流 0.27 元，对应 OCF/EPS 0.42x。沪电 23-25 年同期都在 1.7-3.1x 之间，本季直接降到 0.42x 是结构性偏离。这可能是好事（积极备货 GB300 / Rubin），也可能是坏事（应收 / 存货吃掉利润）——**我不知道**。但 Buffett 的戒律是清楚的——'当我看不懂现金流为何偏离时，我宁可在中报披露后再决定，也不要在不确定中加仓'。
>
> 与胜宏的比较给了一个有趣的镜像：**胜宏赚的是 NVIDIA 一家的真金白银（OCF/EPS 1.64x），沪电赚的是分散客户的纸面利润（OCF/EPS 0.42x）**。在 PE-TTM 一个 71x 一个 45x 的对比下，市场已经在为这种差异定价——胜宏的估值溢价部分来自现金流验真，沪电的估值折扣部分来自 PB 100% 极值 + 现金流暂时不及胜宏。
>
> 我维持观察。**目标买点是 PE < 30x（市值 ~1,300 亿，对应价格 ~67 元）**——这个价位在任何 26 / 27 / 28 全年净利情景下都给到 30%+ 安全边际。在那之前，'好生意 + 好价格' 中，好生意已经验证，**好价格还差最后 30-40%**。」

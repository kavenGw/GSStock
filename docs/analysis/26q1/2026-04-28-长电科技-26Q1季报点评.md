## 二、核心亮点（4 个超预期）

### 2.1 "营收 -2% / 利润 +37%" 的剪刀差 = 高端化兑现

- **销售净利率从 2.18% 跃升至 3.04%（相对涨幅 +39%）**——典型"以量换质"
- **主营业务利润率 14.22% 创近 4 个季度新高**（24Q1: 12.0% → 25Q1: 12.4% → 25Q4: 13.9% → **26Q1: 14.2%**）单调上行
- 与年报"先进封装产能利用率提升 + 高端工艺占比上升"的叙事**首次在季度数据上得到兑现**
- 历史对比：长电 5 年毛利率从 18.4% 跌至 14.2%——**26Q1 主营利润率 14.2% 是过去 4 年首次出现"季度环比连续上行"**

### 2.2 经营现金流同比 +55% = 利润含金量极高

- OCF/股 0.99 元 vs 净利/股 0.156 元 = **OCF/EPS = 6.38x**（极健康）
- 25Q1 该比率为 5.62x，**26Q1 进一步改善 +14%**
- 现金流大幅领先于利润 = **应收账款管理改善 + 客户回款健康**
- 应收账款周转率 1.61 次（年化 6.4 次）vs 通富 1.17 次（年化 4.7 次）——**长电运营效率优势进一步拉大**

## 三、隐忧（2 个需要持续观察）

### 3.1 营收 -1.76% 是否结构性失速？

历史季度营收同比演变：
```
24Q1 +16.7% → 24Q2 +27.2% → 24Q3 +22.3% → 24Q4 +21.2%
25Q1 +36.4% → 25Q2 +20.1% → 25Q3 +14.8% → 25Q4  +8.1%
26Q1  -1.76% ← 转负
```

- **增速已连续 4 个季度下降，并在 26Q1 首次转负**——不能仅用"高基数"解释
- 如果纯基数效应：25Q1 同比 +36% / 26Q1 同比 -2% → 两年 CAGR ≈ +15.7%，仍处合理水平
- **但增速曲线明显走平**：可能由 ① 中低端封装价格压力持续，② 高端封装单价高但订单数量未跟上，③ 全球封测 Q1 季节性 + 大客户库存调整 共同导致

**关键观察点**：
- 若 26Q2 营收同比仍为负 = 行业进入新一轮下行
- 若反弹至 +5%+ = **高端化对冲低端萎缩成功**——重大利好信号

### 3.2 单季环比 -54.9% 的季节性因素

历史 Q1 vs 上年 Q4 环比：
```
24Q1 vs 23Q4 = -47%
25Q1 vs 24Q4 = -62%
26Q1 vs 25Q4 = -55%
```

- **季节性极强（封测 Q4 是出货高峰）**——本期环比 -55% 在历史正常范围
- 但意味着**全年 EPS 不能简单用 Q1×4 推算**
- 按历史季节性比例（Q1 占全年 ~13-15%）：26 全年 EPS 约 **0.156 / 0.14 ≈ 1.11 元**（vs 25 年 0.88 元，+26%）
- 保守推算：26 全年 EPS **0.95-1.10 元区间**

**原 comps 报告建议**：长电 + 通富 6:4 或 5:5 配置（业务结构对冲）

3. **估值差距未变**：长电 PB 2.81x（5y 50%）/ 通富 PB 4.92x（5y 91%）——**通富仍偏贵**
4. **财务安全性差距拉大**：长电资产负债率 43%（环比 -0.63pct）/ 通富 60.6%（虽环比 -3pct 但绝对值仍高 17.6pct）

| 情景 | 长电 26 全年 EPS | 长电 PE | 长电股价空间 | 通富 26 全年 EPS | 通富 PE | 通富股价空间 |

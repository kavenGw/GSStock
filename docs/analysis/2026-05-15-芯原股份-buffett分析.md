# 芯原股份（688521）—— Buffett 视角分析

> **结论先行**：不买。**技术稀缺性极强 + 财务从未证明 + 估值极端高估**——这正是 Buffett 一辈子都在远离的组合（"故事+不能算账+不便宜"）。

> **历史档案**：本仓未存芯原股份 / 688521 / VeriSilicon 任何先前分析（已 Glob `docs/**/*芯原*`、`*688521*`、`*VeriSilicon*`）。本文为首篇。

## 0 · 执行摘要

| 维度 | 判断 | 关键证据 |
|---|---|---|
| 能力圈 | **边界外** | IP 授权 + Chiplet + 端侧 AI 三重技术变量，10 年盈亏路径无法可靠预测 |
| 技术地位 | **极强且稀缺** | VPU 全球第一、IP Top 10 国内独苗、谷歌 Coral/Tensor 合作方 |
| 护城河 | **存在但未变现** | IP 复利模型理论上漂亮（版税滚雪球）；实际 10 年没赚到钱 |
| 商业模式 | **混合：IP（好生意）+ 一站式 ASIC（重资产苦活）** | 一站式收入占比 60-70%，拉低整体盈利质量 |
| 财务质量 | **从未证明** | 2016-2025 十年扣非累计亏损 ~18 亿；2025 前三季再亏 6.14 亿 |
| 估值 | **极端高估** | PB 44x（5 年中位 11.88x、5 年最低 5.13x）、PS 44x |
| 杠杆安全 | **警示** | 资产负债率 55.7%（上升中），现金流持续为负 |

**口径**：实时价 265.91 元（2026-05-15 收），总股本 5.26 亿，**总市值 1398 亿**。

## 1 · 能力圈评估

**结论：边界外**。

芯原的业务有三条腿，每一条都需要独立的预测能力：

1. **半导体 IP 授权**（GPU/NPU/VPU/ISP/DSP 五类自主 IP）—— 这是经典好生意：IP 授权费 + 长期版税。但全球玩家是 ARM、Synopsys、Cadence、Imagination；芯原能否在端侧 AI NPU IP 上守住"国内独苗"地位 10 年？需要预测 RISC-V 生态、Chiplet 协议（UCIe）、各家 NPU 软件栈竞争结果。
2. **一站式芯片定制**（Turnkey ASIC）—— 给客户做后端设计、流片、封测。这门生意接近"芯片代工服务"，毛利率受单子结构影响大，重周期。
3. **Chiplet / 端侧 AI 大客户绑定**—— 与谷歌（Coral/Tensor）、Meta、Mobileye 等大厂的项目能否持续放量？这是 case-by-case 的"重大客户事件驱动"，预测难度极高。

>「风险来自于你不知道你在做什么」—— 单一变量已超出能力圈；三重叠加进一步降低判断置信度。

**与全志科技对比**：全志是"看得懂但不值得"；芯原是"看不懂的好故事"。两者都不买，但理由不同。

## 2 · 关键假设

> 列出后续验证用，但当前估值已剥夺验证机会。

1. **IP 业务理论上是好生意**（参考 ARM 模型：高毛利 + 版税复利 + 轻资产），但**芯原至今 IP 收入占比仅 30-40%**，主体仍是一站式 ASIC 服务（占 60-70%），生意综合气质偏服务而非授权。
2. **谷歌 + RISC-V + 端侧 AI 三重叙事**在 2025-2026 推升估值；任一兑现不及预期都会触发 50%+ 的估值回归。
3. **正常化 Owner Earnings 不存在**——过去 10 年仅 2021、2022 微利（合计不到 1 亿），扣非全部为负或勉强为正，无法做 normalization。
4. **盈亏平衡点估计**：以毛利率 40% 算，收入需达到约 45-50 亿才能覆盖目前研发（~14 亿/年）+ 销售管理费用。当前 2025 全年指引营收 ~40+ 亿，仍未到。
5. **估值锚是叙事而非财务**：1398 亿市值对应 PB 44x、PS 44x，这种估值水平在全球 IP 行业（含 ARM 控股、Cadence、Synopsys）都属顶部，但那些公司是真有 30%+ ROE 的，芯原还在亏损。

## 3 · 业务质量

### 3.1 五类护城河逐项体检（参考 03-business-moat.md）

| 类型 | 是否成立 | 说明 |
|---|---|---|
| 无形资产（IP/专利） | △ **部分** | 五类自主 IP 是真正壁垒，VPU 全球第一是事实；但 IP 客户集中度高、单家流失可见 |
| 低成本优势 | ✗ | 一站式 ASIC 没有规模优势（小于联电、世芯-KY） |
| 转换成本 | ✓ **真实** | IP 一旦嵌入客户芯片版图，迁移需重新流片，成本巨大——这是真正的护城河 |
| 网络效应 | ✗ | 不适用 |
| 高效规模 | ✗ | IP 市场是全球开放市场 |

**护城河结论**：存在（转换成本 + 自主 IP），但**强度未经规模检验**——客户每年新流片项目数十个，看似在用，但综合 IP 授权 + 版税收入只有 8-10 亿/年，证明客户付的价不高（议价权弱于 ARM）。

### 3.2 业务模式：franchise vs commodity

Buffett 1991 三条件：

| 条件 | 芯原符合？ |
|---|---|
| ① 被需要或被渴望 | ✓ 端侧 AI 真实需求 |
| ② 客户认为没有近似替代 | △ IP 部分成立，定制服务部分不成立 |
| ③ 不受价格管制 | ✓ |

**结论：混合型业务**，IP 部分接近 franchise，但一站式服务部分是 commodity。整体气质介于 ARM（纯 IP，franchise）和世芯-KY（纯定制，commodity）之间，且偏向后者。

### 3.3 经济商誉（参考 03 文件 "Economic Goodwill"）

按 Buffett 1983 信定义：
```
经济商誉 = 在净有形资产上获得显著超额回报的能力之资本化
```

- 芯原账面净资产约 34 亿（每股 6.50 × 5.26 亿股）
- 假设它能赚到行业平均 IP 公司的 20% ROE → 年净利 6.8 亿
- 实际：仍然亏损 ~6 亿

→ **经济商誉为负**（capitalized adrenaline）。账面无大额收购商誉是好事，但更深层问题是"有形净资产+研发投入"目前还在毁损价值，不是创造价值。

## 4 · 财务快照（按 05-financial-metrics.md）

### 4.1 长期亏损史

```
扣非净利润（亿）:
2016: -0.23    2021: -0.47
2017: +0.44    2022: +0.13   ← 唯一两个微利年
2018: -0.61    2023: -3.18
2019: -1.01    2024: -6.43
2020: -1.07    2025前三季: -6.14
```

**10 年累计扣非净利约 -18.5 亿**。这是一家上市以来从未真正赚过钱的公司，但市场给的市值 1398 亿。

### 4.2 ROE/ROIC

10 年绝大多数年份 ROE 为负，无法做 10 年均值；连"行业一般水平 10%"都从未达到，距离 Buffett 标准（15%）极远。**用 ROE 评价芯原本身就是错配——它根本不在"能持续创造价值"的样本里。**

### 4.3 现金流质量

```
每股经营现金流（元）:
2021: 0.31
2022: -0.66
2023: -0.02
2024: -0.69
2025前三季: -0.42
```

**近 5 年累计经营现金流为负**——这是更严重的问题。Buffett 在 05 文件给出的红旗清单中："经营现金流持续低于净利润"，芯原是"经营现金流低于本就为负的净利润"，深度赤字。

### 4.4 债务安全

资产负债率 2020 17.8% → 2025 55.7%，**5 年提升 38pp**。研发投入扩张 + 持续亏损 → 持续融资。
- 短期看：现金 + 理财仍可支撑 1-2 年研发
- 中期看：必须发新股或发可转债（继续稀释 5.26 亿现有股本）

### 4.5 营收高速增长能否拯救？

```
营业收入（亿）:
2020: 15.06    2023: 23.38 (-12.7%)
2021: 21.39    2024: 23.22 (-0.7%)
2022: 26.79    2025前三季: 31.52 (+35.77%)
```

2025 营收暴增是亮点，但毛利率从 2022 的 41.6% 降到 2025 的 34.2%——说明**新增收入是低毛利的一站式 ASIC 大订单**（可能 Chiplet 定制单），不是高毛利的 IP 授权。这与"IP 业务规模复利"的故事相反，是"接低毛利服务单子保住现金流"。

## 5 · 估值

### 5.1 三种方法都无法适用

按 06-valuation-capital.md 三方法：

| 方法 | 是否适用 | 原因 |
|---|---|---|
| Owner Earnings 倍数 | ✗ | Owner Earnings 为负，无法乘倍数 |
| 简化 DCF | ✗ | 未来 10 年现金流方向都不确定 |
| 历史盈利能力法 | ✗ | 10 年均值仍为负 |

→ **巴菲特原话**："如果一家公司的内在价值无法可靠估计，不要投资。"

### 5.2 只能做相对估值（极度高估警示）

```
当前 PB:        44.44x
PB 5 年中位:    11.88x  → 当前是中位 3.7 倍
PB 5 年最低:     5.13x  → 当前是最低 8.7 倍
PB 5 年最高:    78.71x  → 当前在历史 60% 高位

PS（按 2025 营收 ~42 亿估）: ~33x
对比 ARM 控股 PS: ~25x（且 ARM 持续盈利、ROE 30%+）
对比 Cadence PS: ~17x
对比 Synopsys PS: ~12x
```

**1398 亿市值的对标**：
- 国内 EDA/IP 龙头华大九天（市值约 350 亿）
- 美股纯 IP 公司 Cadence（市值约 8000 亿人民币、年利润 25 亿美元）

芯原市值 = 4 个华大九天 = 1/6 个 Cadence，但财务体量不到 Cadence 的 1/30、且持续亏损。

### 5.3 推荐买入价（如果护城河真兑现）

假设 5 年后 IP 业务收入翻倍到 20 亿（带 60% 毛利） + 一站式服务稳定毛利 25% → 综合净利可达 4-5 亿，给 IP 公司 30x PE：
- 合理市值 ~120-150 亿
- 当前 1398 亿距离合理估值有 **88%+ 下跌空间**

要 30% 安全边际：
- 合理市值 × 70% ≈ 100 亿 → 股价 ~19 元
- **当前 265.91 元，距离 19 元有 93% 跌幅**

更现实的"再观察"触发线：
- 市值跌至 **400 亿**（股价 ~76 元），PB 回到 12x 中枢
- **同时**扣非连续 4 季度转正

## 6 · 风险（按 07-risk-behavior.md 三大类排序）

1. **行为类 - 估值泡沫破裂**：PB 44x 是 RISC-V + 谷歌 + 端侧 AI + Chiplet 四重叙事的最大公约数定价。一旦任一兑现不及预期（如 2026 财报扣非未明显改善、谷歌项目放量不及预期、或 ARM 在 RISC-V NPU 上构建反制方案），市场重估幅度可能 50%+。
2. **结构性 - IP 业务相对体量太小**：IP 授权 + 版税收入只有约 8-10 亿/年，占总收入 25-30%，无法独立支撑"轻资产高利润"故事；公司被动接低毛利定制大单维持现金流，business mix 在恶化。
3. **财务 - 持续稀释风险**：连年亏损 + 经营现金流为负 + 资产负债率上升 → 几乎必然需要再融资（定增/可转债），5.26 亿股本会继续被稀释。

## 8 · 监控指标（季度跟踪）

**每季关注**：
- **IP 业务收入占比** 是否从 30-35% 向 50%+ 抬升（结构性盈利能力的核心信号）
- **毛利率** 是否回到 42%+（2022 水平）以上
- **扣非净利** 是否单季转正
- 经营性现金流是否连续 2 季度转正
- 大客户事件：谷歌 Tensor / Coral 项目放量节奏、Meta / Mobileye 续签状态

**触发重新审视的信号**（同时满足）：
- 市值回落至 **400 亿以下**（股价 ~76 元）
- 连续 4 个季度扣非转正
- IP 收入占比超过 45%

**触发立即排除的信号**：
- 公告大额定增 / 可转债（确认稀释）
- 谷歌或重大客户公开宣布替换方案
- 单季毛利率跌破 30%

## 9 · 总体评估（巴菲特口吻）

> 这家公司有一些我喜欢的元素：他们在做 IP 这门生意，这是 ARM 让我重新理解过的好生意——你卖出去的不是芯片，是版税，每一颗芯片都给你一点点钱，年复一年，像 See's Candies 不需要工厂扩张就能涨价一样优雅。VPU 全球第一、IP Top 10 国内独苗，这些是真东西，不是宣传稿。
>
> 但我不会买。
>
> 第一，**他们 10 年没有赚过钱**。我不是说一家好公司不能在早期亏损——亚马逊也亏过很多年。但亚马逊每一年的现金流改善路径都是清晰可计算的；芯原 10 年里我看不到一条"成本结构 vs 收入结构"明显改善的曲线，反而看到毛利率从 41% 滑到 34%，看到一站式服务低毛利单子在挤占 IP 高毛利收入。这是错的方向。
>
> 第二，**1398 亿市值买一家 10 年累计亏损 18 亿的公司**。我用什么估值？Owner earnings 是负的，DCF 没法贴现一个方向都不明的现金流，历史平均利润是负的。1989 年我说过："如果计算出的内在价值对贴现率高度敏感，估值不可靠。"这家公司比那还糟——根本不存在可靠的估值。市场用的不是估值，是叙事——谷歌、RISC-V、端侧 AI、Chiplet，四个故事叠在一起。
>
> 第三，**Charlie 跟我说过 50 多年的话**：以合理价格买入卓越企业，远胜以卓越价格买入合理企业。芯原可能是未来的卓越企业，但**现在的价格已经是未来卓越实现后的价格**了——市场给我留的不是安全边际，是"如果一切都对了你刚好不亏"的零边际。
>
> 我宁可承认这超出了我的能力圈，把钱留给我看得懂、有 10 年以上 15%+ ROE 历史的生意。如果有一天它的股价跌到 75 元（市值 400 亿，PB 回到 12x 中枢），同时它真的开始稳定赚钱、IP 收入占比超过 45%——那时候我才会重新打开这一页。
>
> 在那一天之前，这是一个"我尊重但不参与"的标的。
